## 🔄 后台管理特殊说明

### localStorage数据优先级
后台管理的智能体配置存储在`localStorage.aiAgentsConfig`中，优先级如下：

1. **已有localStorage数据**: 优先使用localStorage中的配置
2. **无localStorage数据**: 使用代码中的`defaultAgents`初始化

### 更新方式

#### 方式1: 清空localStorage（推荐）
```javascript
// 浏览器Console执行
localStorage.removeItem('aiAgentsConfig')
// 刷新页面，会自动使用新的默认配置
```

#### 方式2: 后台手动编辑
1. 访问 `http://localhost:3002/admin/ai-agents`
2. 找到第1行（拧紧工具选型）
3. 点击"编辑"按钮
4. 修改"标题"字段为"拧紧工具选型"
5. 点击"确定"保存

#### 方式3: 直接修改localStorage
```javascript
// 浏览器Console执行
const config = JSON.parse(localStorage.getItem('aiAgentsConfig'))
config[0].title = '拧紧工具选型'
localStorage.setItem('aiAgentsConfig', JSON.stringify(config))
location.reload()
```
