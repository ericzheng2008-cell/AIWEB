# ⚡ Pandoc 安装问题最终解决方案

## 🎯 当前状况

您已经：
- ✅ 下载了正确的 .msi 文件
- ✅ 运行了安装程序
- ❌ 但系统仍无法识别 pandoc 命令

## 🔍 可能的原因

1. **安装未完成** - 安装过程中可能有错误但未显示
2. **需要管理员权限** - 可能没有以管理员身份运行
3. **环境变量未更新** - 需要重启命令行或电脑
4. **安装到非标准位置** - Pandoc可能安装在了用户目录

---

## ✅ 解决方案A：重新安装（推荐）⭐⭐⭐⭐⭐

### 步骤：

1. **卸载现有安装**（如果有）
   - 按 `Win + R`
   - 输入 `appwiz.cpl`
   - 找到 Pandoc，点击卸载

2. **重新下载安装文件**
   - 🔗 https://github.com/jgm/pandoc/releases/download/3.1.11.1/pandoc-3.1.11.1-windows-x86_64.msi

3. **以管理员身份安装**（重要！）
   - 找到下载的 .msi 文件
   - **右键点击** → **"以管理员身份运行"**
   - 等待看到 "User Account Control" 提示
   - 点击 "是"
   - 按照向导完成安装
   - **注意查看是否有任何错误提示**

4. **验证安装**
   - 安装完成后，按 `Win + R`
   - 输入 `cmd`
   - 在新打开的命令行中输入：`pandoc --version`
   - 应该看到版本信息

---

## ✅ 解决方案B：使用便携版（免安装）⭐⭐⭐⭐

如果MSI安装一直有问题，使用免安装的ZIP版本：

### 步骤：

1. **下载便携版**
   - 🔗 https://github.com/jgm/pandoc/releases/download/3.1.11.1/pandoc-3.1.11.1-windows-x86_64.zip

2. **解压文件**
   - 解压到：`C:\Pandoc`
   - 确保 `C:\Pandoc\pandoc.exe` 存在

3. **添加到环境变量**（请仔细操作）

   **方法1：通过图形界面**
   - 按 `Win + Pause/Break`（或右键"此电脑" → 属性）
   - 点击 "高级系统设置"
   - 点击 "环境变量"
   - 在"系统变量"区域找到 `Path`
   - 点击 "编辑"
   - 点击 "新建"
   - 输入：`C:\Pandoc`
   - 点击 "确定" 保存所有窗口

   **方法2：使用命令行**（需要管理员权限）
   ```cmd
   setx PATH "%PATH%;C:\Pandoc" /M
   ```

4. **重启命令行**
   - 关闭所有命令行窗口
   - 重新打开
   - 输入：`pandoc --version`
