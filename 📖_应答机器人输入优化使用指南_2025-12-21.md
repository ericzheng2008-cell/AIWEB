## 🎯 快速开始

### **打开应答机器人**

```
方式1️⃣: 点击右下角AI人形助手图标
方式2️⃣: 使用快捷键 Ctrl+K (Mac: Cmd+K)
```

### **输入区域位置**

应答机器人窗口最底部,包含:
```
┌─────────────────────────────┐
│  🎤  [麦克风]  浏览器提示      │  ← 语音输入工具栏
├─────────────────────────────┤
│  [输入消息,按回车发送...]  [→] │  ← 文字输入+发送
├─────────────────────────────┤
│  ℹ️ 支持文字输入和语音输入    │  ← 功能提示
└─────────────────────────────┘
```

## 💬 文字输入

### **基本使用**

1. **输入文字**
   - 点击输入框
   - 输入您的问题或消息
   - 支持多行输入(自动扩展1-4行)

2. **发送消息**
   - 方式1: 按 `Enter` 键
   - 方式2: 点击右侧紫色圆形发送按钮 `→`

3. **输入提示**
   - 输入框显示: "输入消息,按回车发送..."
   - 输入内容时,发送按钮会有脉冲动画提示

### **输入技巧**

✅ **换行**: 输入框会自动扩展,无需手动调整  
✅ **快速发送**: 输入完直接按Enter,无需鼠标  
✅ **长文本**: 支持最多4行显示,超出会出现滚动条  
✅ **焦点高亮**: 点击输入框时边框会变成紫蓝色

## 🎤 语音输入

### **浏览器要求**

| 浏览器 | 语音支持 | 推荐度 |
|--------|----------|--------|
| Chrome | ✅ 完美支持 | ⭐⭐⭐⭐⭐ |
| Edge | ✅ 完美支持 | ⭐⭐⭐⭐⭐ |
| Safari | ✅ 完美支持 | ⭐⭐⭐⭐ |
| Firefox | ❌ 不支持 | ⭐⭐ |

### **首次使用**

1. **点击麦克风按钮** 🎤
   - 位置: 输入区域左上方
   - 颜色: 蓝紫渐变圆形按钮

2. **允许麦克风权限**
   - 浏览器会弹出权限请求
   - 点击"允许"
   - 仅首次使用需要授权

3. **开始说话**
   - 按钮变为红色脉冲动画
   - 说出您的问题
   - 清晰、不要太快

4. **自动识别**
   - 说完后停顿1-2秒
   - 系统自动识别文字
   - 识别成功后自动发送

### **使用步骤详解**

#### **步骤1: 启动录音**
```
点击: 🎤 蓝色麦克风按钮
状态: 按钮变为红色,并有脉冲动画
提示: "🎤 开始录音,请说话..."
```

#### **步骤2: 说话**
```
动画: 红色波形动画表示正在录音
输入框: 显示 "🎤 正在录音中,说话后会自动识别..."
状态栏: 显示 "🎤 正在识别语音,请说话..."
```

#### **步骤3: 停止录音**
```
自动: 说完后停顿1-2秒,自动结束
手动: 再次点击麦克风按钮
手动: 点击状态栏"停止录音"按钮
```

#### **步骤4: 识别结果**
```
成功: ✅ 识别成功: "您说的内容"
     → 文字自动填入输入框
     → 0.5秒后自动发送

失败: ❌ 显示具体错误原因
     → 可以重新尝试
```

### **语音识别状态**

| 按钮颜色 | 状态 | 说明 |
|---------|------|------|
| 🔵 蓝紫渐变 | 待机 | 可以点击开始录音 |
| 🔴 红色脉冲 | 录音中 | 正在录音,请说话 |
| ⚫ 灰色 | 不可用 | 浏览器不支持 |

### **语音识别技巧**

✅ **环境**: 安静环境,减少噪音  
✅ **距离**: 靠近麦克风(20-30cm)  
✅ **语速**: 正常语速,不要太快  
✅ **发音**: 清晰发音,使用普通话/标准英语  
✅ **长度**: 短句比长句识别更准确  
✅ **停顿**: 说完后停顿1-2秒让系统识别

❌ **避免**: 
- 环境噪音过大
- 距离麦克风太远
- 说话太快或含糊
- 句子过长复杂
- 方言或口音过重

## 🐛 常见问题

### **Q1: 找不到输入框?**

**检查步骤:**
1. 确认已打开聊天窗口(点击右下角AI助手)
2. 滚动到窗口最底部
3. 输入区域有明显的边框和阴影
4. 尝试刷新页面 `Ctrl + F5`

### **Q2: 麦克风按钮是灰色的?**

**原因分析:**
- 浏览器不支持语音识别
- 使用了Firefox或其他不支持的浏览器

**解决方案:**
1. 切换到Chrome、Edge或Safari浏览器
2. 或使用文字输入作为替代方案
3. 查看输入区域的提示信息

### **Q3: 点击麦克风没反应?**

**检查清单:**
- [ ] 是否允许了麦克风权限?
  - 检查浏览器地址栏的权限图标
  - 重新授权麦克风权限
  
- [ ] 麦克风是否被占用?
  - 关闭其他使用麦克风的程序
  - 检查系统麦克风设置
  
- [ ] 网络连接是否正常?
  - 语音识别需要网络连接
  - 检查网络状态
  
- [ ] 浏览器控制台是否有错误?
  - 按 `F12` 打开开发者工具
  - 查看Console选项卡

### **Q4: 语音识别不准确?**

**优化建议:**

1. **环境优化**
   - 选择安静的环境
   - 关闭音乐、电视等噪音源
   - 使用耳机麦克风效果更好

2. **说话技巧**
   - 清晰发音,不要吃字
   - 正常语速,不要太快
   - 使用短句,一次说一个问题
   - 使用普通话(中文)或标准英语

3. **设备检查**
   - 确保麦克风工作正常
   - 调整麦克风音量
   - 清理麦克风孔

4. **重新尝试**
   - 如果识别错误,可以重新录音
   - 或直接使用文字输入

### **Q5: 手机端输入框被键盘遮挡?**

**已优化功能:**
- ✅ 使用sticky定位,输入框始终可见
- ✅ 自动适配虚拟键盘高度
- ✅ 支持iOS安全区域

**如仍有问题:**
1. 旋转屏幕到横屏模式
2. 使用浏览器全屏模式
3. 更新浏览器到最新版本
4. 关闭其他占用屏幕的功能

### **Q6: Enter键没有发送消息?**

**可能原因:**
- 在快捷问题区域按的Enter
- 输入框没有焦点
- 正在等待AI回复中

**解决方法:**
1. 点击输入框,确保有焦点
2. 等待AI回复完成后再发送
3. 或直接点击发送按钮

### **Q7: 发送按钮点不了?**

**检查项:**
- [ ] 输入框是否有内容?(空内容不能发送)
- [ ] 是否正在等待AI回复?(回复中禁用)
- [ ] 是否正在录音?(录音时禁用)

## 📱 移动端使用

### **手机访问**

1. **连接同一WiFi**
   - 确保手机和电脑在同一网络

2. **访问地址**
   - 电脑显示的局域网地址
   - 或扫描二维码

3. **打开应答机器人**
   - 点击右下角AI助手
   - 自动全屏显示

### **移动端特点**

✅ **全屏体验**
- 聊天窗口占满整个屏幕
- 更好的阅读和输入体验

✅ **触摸优化**
- 所有按钮至少44×44px
- 方便手指点击

✅ **键盘适配**
- 输入框固定在底部
- 键盘弹出时自动调整
- 不会遮挡输入区域

✅ **语音输入**
- 移动端更适合语音输入
- 说话更自然方便

### **移动端语音使用**

1. **横屏模式** (可选)
   - 更大的显示区域
   - 更舒适的输入体验

2. **点击麦克风**
   - 按钮尺寸针对触摸优化
   - 更容易点击

3. **靠近麦克风**
   - 手机麦克风通常在底部
   - 说话时保持手机稳定

4. **查看状态**
   - 红色脉冲表示正在录音
   - 波形动画表示识别中

## 💡 使用技巧

### **高效输入**

1. **快捷键**
   - `Ctrl+K`: 打开/关闭应答机器人
   - `Enter`: 发送消息
   - `Esc`: 关闭聊天窗口

2. **文字输入**
   - 短问题用文字更快
   - 需要精确表达时用文字
   - 需要复制粘贴时用文字

3. **语音输入**
   - 长问题用语音更快
   - 双手不方便时用语音
   - 移动场景下用语音

### **组合使用**

```
场景1: PC办公
建议: 文字输入为主,语音为辅

场景2: 移动外出
建议: 语音输入为主,文字为辅

场景3: 嘈杂环境
建议: 仅使用文字输入

场景4: 安静私密
建议: 语音输入最佳
```
