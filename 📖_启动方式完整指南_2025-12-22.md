## 🤔 为什么需要启动服务器？

AIWEB 是一个**前后端分离**的 Web 应用，包含：

| 组件 | 技术栈 | 端口 | 作用 |
|------|--------|------|------|
| 前端服务器 | Vue 3 + Vite | 3002 | 渲染网页界面 |
| 后端服务器 | Node.js + Express | 3000 | 处理数据和业务逻辑 |

**为什么不能像普通网站一样直接打开？**
- 普通网站：静态 HTML 文件，双击即可打开
- AIWEB：需要编译 Vue 代码 + 运行后端 API，必须启动服务器

## 🚀 三种启动方式

### 方式一：一键启动（推荐）⭐

**步骤：**
1. 双击运行 `⚡_一键启动AIWEB_2025-12-22.bat`
2. 自动启动前后端服务器
3. 自动打开浏览器

**优点：**
- ✅ 最简单，一键完成
- ✅ 自动检查端口占用
- ✅ 自动打开浏览器

---

### 方式二：开机自启动（最省事）🌟

**设置步骤：**
1. 双击运行 `🔧_设置开机自启动_2025-12-22.bat`
2. 选择 `[1] 启用开机自启动`
3. 完成！

**效果：**
- ✅ 开机后自动在后台启动 AIWEB
- ✅ 无需手动操作
- ✅ 随时打开浏览器访问 `http://localhost:3002`

**如何禁用？**
- 再次运行设置工具，选择 `[2] 禁用开机自启动`

---

### 方式三：手动启动（适合开发调试）

**步骤：**
1. 打开终端，进入项目目录
2. 启动后端：
   ```bash
   cd server
   node index.js
   ```
3. 新开一个终端，启动前端：
   ```bash
   npm run dev
   ```

## ⚙️ 开机自启动详细说明

### 启动原理
1. 在 Windows 启动文件夹创建 VBS 脚本
2. 开机时自动执行脚本
3. 脚本调用一键启动批处理文件
4. 服务器在后台静默启动

### 启动文件位置
```
C:\Users\你的用户名\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup\AIWEB_Auto_Start.vbs
```

### 如何验证是否启用？
1. 运行 `🔧_设置开机自启动_2025-12-22.bat`
2. 选择 `[3] 查看当前状态`
3. 或者直接查看上述文件夹是否有 `AIWEB_Auto_Start.vbs`

## ❓ 常见问题解答

### Q1: 开机自启动后，如何知道服务器已经启动？
**A:** 打开浏览器访问 `http://localhost:3002`，如果能正常显示页面，说明已启动。

---

### Q2: 开机自启动后，窗口会一直显示吗？
**A:** 不会！使用 VBS 脚本隐藏启动，后台静默运行，不会弹窗打扰。

---

### Q3: 如何关闭正在运行的服务器？
**A:** 
- 方法1：运行 `🔧_强制清理所有Node进程.bat`
- 方法2：任务管理器 → 详细信息 → 找到 `node.exe` → 结束任务

---

### Q4: 服务器占用电脑资源吗？
**A:** 
- 空闲时：占用极少（< 100MB 内存，CPU < 1%）
- 访问时：正常占用（200-300MB 内存，CPU 5-10%）
- **建议：** 如果长时间不用，可以关闭服务器节省资源

---

### Q5: 能否像 cpolar 一样外网访问？
**A:** 
- 默认启动：仅本机访问（`localhost:3002`）
- 外网访问：需要运行 cpolar 工具
  - 双击 `🚀_启动AIWEB并生成手机访问链接.bat`
  - 或参考 `📖_cpolar使用完整指南_2025-12-21.md`

---

### Q6: 公司电脑能设置开机自启动吗？
**A:** 
- ✅ 可以！不需要管理员权限
- ✅ 启动文件夹是用户目录，任何人都可以写入
- ⚠️ 注意：某些企业防火墙可能阻止 Node.js，需要 IT 部门授权

---

### Q7: 忘记启动服务器，打开网页会怎样？
**A:** 
- 浏览器显示 "无法访问此网站" 或 "连接被拒绝"
- 解决：运行 `⚡_一键启动AIWEB_2025-12-22.bat`
