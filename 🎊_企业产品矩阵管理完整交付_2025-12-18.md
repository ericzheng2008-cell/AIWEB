### 2️⃣ AI智能推荐系统

**三大推荐类别**:

**🎯 战略建议** (基于产品组合整体分析)
- 强化核心产品 → 增加市场投入
- 优化现金流 → 改善库存和账期
- 优化产品组合 → 淘汰或升级低效产品

**⚡ 运营优化** (具体可执行建议)
- 库存优化 → 针对高库存周转产品
- 回款加速 → 针对长回款周期产品
- 成本降低 → 针对低毛利产品

**⚠️ 异常预警** (实时风险监测)
- 毛利率过低 (< 15%)
- 现金流贡献低 (< 10%)
- 库存积压 (> 90天)
- 回款超期 (> 60天)

### 核心算法

#### 1. 四象限自动计算
```javascript
determineQuadrant(grossMargin, cashFlowRate) {
  const marginThreshold = this.strategyConfig.grossMarginThreshold  // 默认25%
  const cashThreshold = this.strategyConfig.cashFlowThreshold      // 默认15%
  
  if (grossMargin >= marginThreshold && cashFlowRate >= cashThreshold) {
    return 'core'        // 核心战略产品 (高收益 + 高现金流)
  } else if (grossMargin >= marginThreshold) {
    return 'optimize'    // 优化提升产品 (高收益 + 低现金流)
  } else if (cashFlowRate >= cashThreshold) {
    return 'potential'   // 潜力产品 (低收益 + 高现金流)
  } else {
    return 'maintain'    // 低投入维持 (低收益 + 低现金流)
  }
}
```

#### 2. 矩阵坐标计算
```javascript
// X轴位置 (基于毛利率, 假设最大50%)
x = Math.min(100, Math.max(0, (grossMargin / 50) * 100))

// Y轴位置 (基于现金流贡献率, 假设最大30%)
y = Math.min(100, Math.max(0, (cashFlowRate / 30) * 100))

// 气泡大小 (基于销售额, 范围30-100px)
const maxRevenue = Math.max(...products.map(p => p.revenue))
size = 30 + ((revenue / maxRevenue) * 70)
```

#### 3. AI推荐生成引擎
```javascript
generateStrategicRecommendations() {
  const coreProducts = this.matrixData.filter(p => p.quadrant === 'core')
  const optimizeProducts = this.matrixData.filter(p => p.quadrant === 'optimize')
  const maintainProducts = this.matrixData.filter(p => p.quadrant === 'maintain')
  
  const recommendations = []
  
  // 核心产品建议
  if (coreProducts.length > 0) {
    const topCore = coreProducts.sort((a, b) => b.revenue - a.revenue)[0]
    recommendations.push({
      title: `强化核心产品"${topCore.name}"`,
      content: `该产品贡献销售额¥${(topCore.revenue / 10000).toFixed(1)}万，建议增加市场投入和产能保障`,
      priority: 'success',
      priorityText: '高优先级'
    })
  }
  
  // 优化产品建议
  if (optimizeProducts.length > 0) {
    const highMargin = optimizeProducts.sort((a, b) => b.grossMargin - a.grossMargin)[0]
    recommendations.push({
      title: `优化"${highMargin.name}"现金流`,
      content: `该产品毛利率${highMargin.grossMargin}%，但库存周转${highMargin.inventoryDays}天，建议优化供应链`,
      priority: 'warning',
      priorityText: '中优先级'
    })
  }
  
  // 淘汰产品建议
  if (maintainProducts.length > 2) {
    recommendations.push({
      title: `优化产品组合`,
      content: `当前有${maintainProducts.length}个低收益低现金流产品，建议评估是否淘汰或升级`,
      priority: 'danger',
      priorityText: '需关注'
    })
  }
  
  return recommendations
}
```
