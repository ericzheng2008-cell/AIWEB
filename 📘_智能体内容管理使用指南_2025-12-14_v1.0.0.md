# 智能体内容管理使用指南

## ✅ 功能概述

CMS内容管理系统新增**智能体内容编辑**功能模块，支持对智能体的文字、图片、排版进行可视化编辑管理。

---

## 📂 访问路径

**后台管理 → CMS内容管理 → 🤖 智能体内容**

```
http://localhost:3002/admin/cms
点击 "🤖 智能体内容" 标签页
```

---

## 🎯 核心功能

### 1. 智能体列表管理
- ✅ 左侧标签页显示所有智能体
- ✅ 点击切换编辑不同智能体
- ✅ 支持添加新智能体
- ✅ 预置3个默认智能体

### 2. 基本信息编辑
- **智能体名称**: 显示在前台的标题
- **状态开关**: 启用/停用智能体显示
- **图标设置**: 使用Emoji或符号
- **简短描述**: 一句话介绍（限100字）

### 3. 详细内容编辑
- **详细描述**: 全面介绍智能体功能（限500字）
- **核心功能**: 多条功能列表，可动态添加/删除
- **应用场景**: 多个使用场景，可动态添加/删除

### 4. 图片管理
- **封面图片上传**: 支持本地上传或URL输入
- **建议尺寸**: 800x600px
- **支持格式**: JPG, PNG, GIF
- **预览功能**: 实时查看图片效果

### 5. 操作功能
- **保存单个智能体**: 保存当前编辑的智能体
- **保存所有更改**: 一键保存全部修改
- **预览效果**: 查看前台显示效果
- **删除智能体**: 移除不需要的智能体

---

## 📝 使用步骤

### 步骤1: 进入编辑页面
1. 登录后台管理系统
2. 点击"CMS内容管理"
3. 点击"🤖 智能体内容"标签页

### 步骤2: 选择智能体
- 左侧点击要编辑的智能体名称
- 或点击"+ 添加新智能体"创建新的

### 步骤3: 编辑基本信息
```
智能体名称: 拧紧工具选型智能体
状态: 启用 ✓
图标: 🔧
简短描述: 智能分析工艺需求，推荐最优拧紧工具方案
```

### 步骤4: 编辑详细内容
**详细描述**:
```
基于AI算法，智能分析您的工艺需求，包括扭矩、精度、
动力形式等参数，为您推荐最合适的拧紧工具方案。
支持多维度筛选和对比，提供专业的工艺优化建议。
```

**核心功能** (点击"添加功能"按钮):
- AI智能匹配推荐
- 多维度参数分析
- 工艺优化建议
- 需求报告生成

**应用场景** (点击"添加场景"按钮):
- 新工位建设
- 工具升级换代
- 工艺优化改善
- 成本控制优化

### 步骤5: 上传图片
**方式1: 本地上传**
1. 点击上传区域
2. 选择本地图片文件
3. 自动显示预览

**方式2: URL输入**
1. 在"图片URL"输入框粘贴图片链接
2. 自动加载显示

### 步骤6: 保存
- 点击"保存此智能体"
- 或点击顶部"保存所有更改"

### 步骤7: 预览
- 点击"预览效果"查看前台显示样式

## 💡 最佳实践

### 内容撰写建议
1. **名称**: 简洁明了，10字以内
2. **简短描述**: 突出核心价值，吸引用户
3. **详细描述**: 分段说明，逻辑清晰
4. **功能列表**: 3-5条为宜，简明扼要
5. **应用场景**: 具体实用，贴近实际

### 图片设计建议
- **尺寸**: 800x600px（宽高比4:3）
- **文件大小**: 建议 < 500KB
- **内容**: 体现智能体特色
- **风格**: 统一的视觉风格
- **格式**: 推荐PNG（支持透明背景）

### 操作技巧
1. **批量编辑**: 先编辑所有内容，最后统一保存
2. **定期备份**: 重要修改前先预览确认
3. **分步保存**: 大量修改建议分批保存
4. **图片优化**: 上传前压缩图片以提升加载速度

## 🚨 注意事项

1. **数据保存**: 修改后务必点击保存，否则刷新页面会丢失
2. **图片大小**: 建议图片文件 < 500KB，避免加载缓慢
3. **浏览器兼容**: 建议使用 Chrome、Edge、Firefox 最新版本
4. **字数限制**: 
   - 简短描述: 100字
   - 详细描述: 500字
5. **删除确认**: 删除智能体需二次确认，操作不可恢复

## 🆘 常见问题

### Q1: 图片上传后不显示？
**A**: 检查图片格式和大小，刷新页面重试

### Q2: 修改后前台没有更新？
**A**: 清除浏览器缓存，或使用无痕模式访问

### Q3: 删除按钮不见了？
**A**: 至少保留1个智能体，最后一个无法删除

### Q4: 可以恢复删除的智能体吗？
**A**: 不能，删除前请确认

### Q5: 如何批量管理智能体？
**A**: 使用"保存所有更改"统一保存多个修改
