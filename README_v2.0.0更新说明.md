## 🚀 快速使用

### 访问CMS管理系统
```bash
URL: http://localhost:3000/admin/cms
需要登录: 是（如已配置）
```

### 更新Logo
1. 进入 "CMS管理" → "Logo和基本设置"
2. 输入Logo图片URL
3. 点击"更新Logo"
4. 刷新页面查看效果

### 管理产品系列
1. 进入 "CMS管理" → "产品系列管理"
2. 可以添加、编辑、删除产品系列
3. 支持中英文双语编辑

### 管理事业部
1. 进入 "CMS管理" → "事业部管理"
2. 可以添加、编辑、删除事业部
3. 支持中英文双语编辑
