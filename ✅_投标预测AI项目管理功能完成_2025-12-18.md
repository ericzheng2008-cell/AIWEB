## 📋 详细功能

### 一、项目基本信息编辑

#### 可编辑字段
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 项目名称 | 文本 | ✅ | 投标项目名称 |
| 客户名称 | 文本 | - | 客户/甲方名称 |
| 项目预算 | 数字 | ✅ | 项目总预算金额 |
| 我方成本 | 数字 | ✅ | 我方成本估算 |
| 项目描述 | 文本域 | - | 项目详细描述 |

### 二、我方投标信息

#### 我方报价录入
| 轮次 | 字段 | 说明 |
|------|------|------|
| 第一轮 | 第一轮报价 | 初次投标报价 |
| 第二轮 | 第二轮报价 | 第二次投标报价 |
| 第三轮 | 第三轮报价 | 最终投标报价 |

#### 其他信息
- **方案供货范围**: 我方提供的产品和服务范围

### 三、竞对信息管理

#### 竞对基本信息
| 字段 | 说明 |
|------|------|
| 公司名称 | 竞对公司全称 |
| 品牌名称 | 竞对使用的品牌 |
| 方案供货范围 | 竞对提供的范围 |

#### 竞对三轮报价
| 轮次 | 说明 |
|------|------|
| 第一轮报价 | 竞对初次报价 |
| 第二轮报价 | 竞对第二次报价 |
| 第三轮报价 | 竞对最终报价 |

#### 历史成交信息
| 字段 | 说明 |
|------|------|
| 历史案例描述 | 竞对的历史成交案例 |
| 历史成交金额 | 历史成交项目的金额 |

```

  ↓

  ↓
用户修改信息
  ↓
点击保存
  ↓
验证数据
  ↓
更新项目列表

```
