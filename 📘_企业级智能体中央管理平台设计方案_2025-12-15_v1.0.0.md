### 1.2 核心设计原则

- **模块化:** 每个智能体作为独立模块，可插拔式设计
- **标准化:** 统一的API接口和数据格式规范
- **可扩展:** 支持动态添加新智能体和功能
- **高可用:** 分布式部署，容错机制
- **安全性:** 多层次安全防护，数据加密

## 4. 数据库设计

### 4.1 核心数据表

#### agents (智能体注册表)
```sql
CREATE TABLE agents (
  agent_id VARCHAR(50) PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  type VARCHAR(50) NOT NULL,
  version VARCHAR(20) NOT NULL,
  status VARCHAR(20) DEFAULT 'active',
  capabilities JSON,
  endpoints JSON,
  configuration JSON,
  dependencies JSON,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_type (type),
  INDEX idx_status (status)
);
```

#### knowledge_base (知识库)
```sql
CREATE TABLE knowledge_base (
  knowledge_id VARCHAR(50) PRIMARY KEY,
  title VARCHAR(200) NOT NULL,
  type VARCHAR(50) NOT NULL,
  category VARCHAR(100),
  tags JSON,
  content_format VARCHAR(20),
  content_data JSON,
  metadata JSON,
  relations JSON,
  access_control JSON,
  search_vector VECTOR(1536), -- 用于向量搜索
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  FULLTEXT INDEX idx_fulltext (title, content_data),
  INDEX idx_category (category),
  INDEX idx_type (type)
);
```

#### agent_interactions (交互记录)
```sql
CREATE TABLE agent_interactions (
  interaction_id BIGINT AUTO_INCREMENT PRIMARY KEY,
  agent_id VARCHAR(50) NOT NULL,
  user_id VARCHAR(50),
  session_id VARCHAR(100),
  query TEXT,
  response JSON,
  context JSON,
  feedback JSON,
  duration_ms INT,
  status VARCHAR(20),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  INDEX idx_agent (agent_id, created_at),
  INDEX idx_user (user_id, created_at),
  FOREIGN KEY (agent_id) REFERENCES agents(agent_id)
);
```

#### learning_records (学习记录)
```sql
CREATE TABLE learning_records (
  record_id BIGINT AUTO_INCREMENT PRIMARY KEY,
  agent_id VARCHAR(50) NOT NULL,
  learning_type VARCHAR(50), -- feedback | retrain | optimization
  source_data JSON,
  action_taken VARCHAR(100),
  model_version_before VARCHAR(20),
  model_version_after VARCHAR(20),
  performance_delta JSON,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  INDEX idx_agent (agent_id, created_at),
  FOREIGN KEY (agent_id) REFERENCES agents(agent_id)
);
```

#### agent_metrics (性能指标)
```sql
CREATE TABLE agent_metrics (
  metric_id BIGINT AUTO_INCREMENT PRIMARY KEY,
  agent_id VARCHAR(50) NOT NULL,
  metric_date DATE NOT NULL,
  hour INT, -- 0-23，用于小时级统计
  total_requests INT DEFAULT 0,
  successful_requests INT DEFAULT 0,
  failed_requests INT DEFAULT 0,
  avg_response_time DECIMAL(10,2),
  p95_response_time DECIMAL(10,2),
  thumbs_up INT DEFAULT 0,
  thumbs_down INT DEFAULT 0,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  UNIQUE KEY idx_agent_date_hour (agent_id, metric_date, hour),
  INDEX idx_date (metric_date),
  FOREIGN KEY (agent_id) REFERENCES agents(agent_id)
);
```
