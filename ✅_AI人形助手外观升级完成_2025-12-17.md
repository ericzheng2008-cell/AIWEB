## ✨ 动画效果总览

### **持续动画（自动播放）：**

| 动画名称 | 元素 | 效果 | 周期 |
|---------|------|------|------|
| **悬浮动画** | 整体 | 上下缓慢漂浮 | 4s |
| **眼睛呼吸** | 眼睛 | 发光强弱变化 | 3s |
| **眼睛光晕** | 眼睛外围 | 光晕脉冲扩散 | 2s |
| **额头闪烁** | 额头点 | 明暗交替 | 3s |
| **颈部发光** | 颈部线条 | 发光线条伸缩 | 2s |
| **肩部旋转** | 肩部圆环 | 360°连续旋转 | 4s |
| **阴影脉冲** | 底部阴影 | 大小变化 | 4s |
| **光环扩散** | 外围光环 | 向外扩散消失 | 3s |

### **交互动画（鼠标悬停）：**

| 触发 | 动画 | 效果 |
|------|------|------|
| **悬停** | 整体上浮 + 放大 | 1.1倍放大，上浮12px |
| **悬停** | 眼睛眨眼 | 0.6秒眨眼动画 |
| **悬停** | 头部倾斜 | 左右微微摆动 |
| **悬停** | 颈部增强发光 | 更亮的脉冲效果 |
| **悬停** | 滤镜增强 | 更强的青蓝色光晕 |
| **点击** | 快速缩小 | 1.05倍缩放 |

## 🎨 配色方案

### **主色调：**
```css
银白色系：#e8f4f8 → #cfe9f3 (头部、肩部)
深色系：  #2c3e50 → #34495e (发型)
暗色系：  #1a1a2e → #16213e (颈部)
```

### **发光色系：**
```css
青蓝主色：#00d4ff (眼睛、装饰圈、发光线)
青色辅助：#0099cc (眼睛渐变)
青色光晕：rgba(0, 212, 255, 0.3-0.8) (各种发光效果)
```

### **装饰色系：**
```css
面板线条：rgba(100, 150, 180, 0.3-0.4) (脸部线条)
阴影：    rgba(0, 0, 0, 0.1-0.4) (各种阴影)
高光：    rgba(255, 255, 255, 0.6-0.8) (内发光)
```
