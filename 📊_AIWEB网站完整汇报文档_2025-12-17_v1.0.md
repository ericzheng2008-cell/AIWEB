## 项目概述

### 1.1 项目定位

**AIWEB** 是一个面向工业制造企业（特别是汽车制造行业）的**企业级智能化营销与服务平台**，集成了：

- ✅ **营销获客闭环系统**
- ✅ **AI智能体服务矩阵**
- ✅ **设备全生命周期管理**
- ✅ **知识库与学习引擎**
- ✅ **数据分析与决策支持**

### 1.2 核心价值主张

```
传统模式：分散的工具、低效的流程、被动的服务
AIWEB平台：
  → 一站式智能化解决方案
  → 数据驱动的精准营销
  → AI赋能的自动化服务
  → 可量化的降本增效
```

### 1.3 目标用户

**主要客户群体**：
- 汽车制造企业（整车厂、零部件厂）
- 工业设备制造商
- 设备维护服务商
- 工业品销售企业

**使用角色**：
- 企业决策层（查看数据分析、ROI报告）
- 技术工程师（使用工具选型、曲线分析）
- 设备管理员（设备运营、维护管理）
- 销售团队（客户管理、商机跟进）
- 普通访客（浏览产品、在线咨询）

### 6.3 知识库智能搜索

```javascript
// 基于TF-IDF + 向量相似度的搜索
class IntelligentSearch {
  constructor(knowledgeBase) {
    this.documents = knowledgeBase
    this.buildIndex()
  }
  
  buildIndex() {
    // 构建倒排索引
    this.invertedIndex = {}
    this.documents.forEach(doc => {
      const terms = this.tokenize(doc.content)
      terms.forEach(term => {
        if (!this.invertedIndex[term]) {
          this.invertedIndex[term] = []
        }
        this.invertedIndex[term].push(doc.id)
      })
    })
  }
  
  search(query) {
    const queryTerms = this.tokenize(query)
    
    // 计算TF-IDF分数
    const scores = {}
    queryTerms.forEach(term => {
      const docIds = this.invertedIndex[term] || []
      const idf = Math.log(this.documents.length / (docIds.length + 1))
      
      docIds.forEach(docId => {
        const doc = this.documents.find(d => d.id === docId)
        const tf = this.countTerm(term, doc.content)
        scores[docId] = (scores[docId] || 0) + (tf * idf)
      })
    })
    
    // 排序返回
    return Object.entries(scores)
      .sort((a, b) => b[1] - a[1])
      .slice(0, 10)
      .map(([docId]) => this.documents.find(d => d.id === parseInt(docId)))
  }
  
  tokenize(text) {
    // 中文分词 + 英文分词
    return text.toLowerCase()
      .replace(/[^\u4e00-\u9fa5a-z0-9]/g, ' ')
      .split(/\s+/)
      .filter(t => t.length > 1)
  }
}
```
