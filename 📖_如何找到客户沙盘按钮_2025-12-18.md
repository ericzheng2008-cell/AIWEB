# 📖 如何找到"客户沙盘"按钮

**更新时间**: 2025-12-18

---

## 🎯 快速指南

### 方法 1：滚动菜单（推荐）✅

1. **刷新页面** - 按 `F5` 或 `Ctrl+R`
2. **找到菜单栏** - 页面顶部紫色背景区域
3. **鼠标悬停在菜单上** - 任意菜单项上方
4. **滚动鼠标滚轮** - 向右滚动
5. **看到"客户沙盘"** - 最右边，图标 📦

![示意图]
```
┌───────────────────────────────────────────────────┐
│ Logo  [销售漏斗] [商机管理] ... [客户沙盘📦]     │
│                                    ▲               │
│                            滚动到这里              │
└───────────────────────────────────────────────────┘
```

---

### 方法 2：拖动滚动条

如果看到菜单底部有一条小滚动条：

1. **鼠标点击滚动条**
2. **向右拖动**
3. **看到"客户沙盘"按钮**

---

### 方法 3：放大浏览器窗口

1. **最大化浏览器窗口** - 点击窗口右上角的最大化按钮
2. **降低浏览器缩放** - 按 `Ctrl + 0` 重置缩放
3. **所有菜单项可能全部可见**

## 🐛 常见问题

### Q1: 我看不到"客户沙盘"按钮

**A**: 菜单项太多，被挤出屏幕了。请：
1. 滚动菜单到最右边
2. 或最大化浏览器窗口

### Q2: 我滚动鼠标滚轮但菜单不动

**A**: 请确保：
1. 鼠标光标在菜单栏上方
2. 尝试左右拖动滚动条
3. 或使用触摸板双指滑动

### Q3: 点击后页面没有变化

**A**: 请：
1. 打开控制台（F12）查看日志
2. 确认是否有错误提示
3. 刷新页面重试

### Q4: 我点的是"客户360°"，不是"客户沙盘"

**A**: 这两个名称相似，请注意区分：
- **客户360°**: 左边第3个，图标 👤
- **客户沙盘**: 最右边第10个，图标 📦
