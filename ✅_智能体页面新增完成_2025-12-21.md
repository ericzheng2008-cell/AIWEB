## 📝 技术实现细节

### 卡片结构
每个智能体卡片包含7个部分：

1. **徽章 (badge)** - 右上角标识
2. **发光效果 (card-glow)** - 鼠标悬停
3. **图标 (agent-icon)** - 64px Element Plus图标
4. **帮助按钮 (help-btn)** - 右上角问号
5. **标题+描述** - h2 + p.agent-desc
6. **功能特性** - 3个feature-item
7. **统计数据** - 2个stat-item
8. **操作按钮** - "进入系统"按钮

### 数据属性
```vue
data-aos="fade-up"              <!-- 淡入动画 -->
data-aos-delay="1000"           <!-- 延迟1秒 -->
data-category="marketing"       <!-- 分类标识 -->
class="agent-card highlight"    <!-- 高亮样式 -->
```

### 事件处理
```javascript
@click="goToAgent('/mingsheng-aicrm')"        // 点击跳转
@mouseenter="onCardHover($event)"             // 鼠标进入
@mouseleave="onCardLeave($event)"             // 鼠标离开
@click.stop="showHelp('mingsheng-aicrm')"     // 帮助按钮
```
