## 👤 客户360画像

### 8大维度数据

#### 1. 基本信息
- 公司名称
- 所属行业
- 公司规模
- 地址位置
- 官方网站

#### 2. 商业数据
- 累计总营收
- 活跃商机数量
- 成交订单数
- 平均订单金额
- 年度增长率

#### 3. 健康分数
- **计算维度**:
  - 交易频率 (30%)
  - 营收贡献 (25%)
  - 互动活跃度 (20%)
  - 付款及时性 (15%)
  - 产品采用率 (10%)

- **评分标准**:
  - 90-100分: 🟢 优秀客户
  - 70-89分: 🟡 健康客户
  - 50-69分: 🟠 关注客户
  - 0-49分: 🔴 高风险客户

#### 4. 风险预警
- **流失风险**
  - 互动减少检测
  - 购买频率下降
  - 竞品接触信号
  
- **付款风险**
  - 逾期付款历史
  - 信用评估
  - 财务状况变化

- **竞争风险**
  - 竞品活动监测
  - 价格敏感度
  - 替代方案考虑

#### 5. 互动记录
- 会议次数和质量
- 邮件往来频率
- 电话沟通记录
- 活动参与情况

#### 6. 产品偏好
- 购买历史
- 产品组合分析
- 使用频率
- 满意度评价

#### 7. 行为预测
- 下次购买时间
- 潜在购买金额
- 扩展产品意向
- 增长潜力评估

#### 8. 联系人网络
- 关键决策人
- 技术负责人
- 采购负责人
- 关系强度分析

## 🎮 客户沙盘分析

### 4大分析模块

#### 1. 营收预测 📈

**功能说明**:
- 基于历史数据预测未来6个月营收
- 提供最佳/最差/平均三种预测场景
- 自动识别季节性趋势

**如何使用**:
1. 点击"沙盘分析"标签
2. 选择预测时间范围（1个月/1季度/半年/1年）
3. 查看营收预测曲线图
4. 对比历史数据验证准确性

**图表说明**:
- 蓝色线: 历史实际营收
- 绿色线: 最佳情况预测
- 红色线: 最差情况预测
- 橙色线: 平均情况预测

#### 2. 风险分析 ⚠️

**风险维度**:
- **流失风险**: 0-100分，>70分高风险
- **付款风险**: 0-100分，>60分需关注
- **竞争风险**: 0-100分，>50分需应对
- **扩展风险**: 业务增长阻力评估

**风险雷达图**:
- 直观显示多维度风险水平
- 对比行业平均水平
- 识别最需要关注的领域

**使用建议**:
1. 每周检查一次客户风险
2. 重点关注风险分>60的客户
3. 根据AI建议制定应对策略

#### 3. 机会识别 🎯

**机会类型**:
- **交叉销售**: 推荐相关产品
- **追加销售**: 升级高端产品
- **续约机会**: 合同到期提醒
- **扩展机会**: 新需求挖掘

**AI推荐逻辑**:
- 分析购买历史
- 对比同行业客户
- 识别产品组合空缺
- 评估客户预算能力

**操作流程**:
1. 查看"机会列表"
2. 评估机会优先级
3. 制定销售策略
4. 跟踪转化效果

#### 4. 场景模拟 🔄

**支持场景**:
- **价格上涨10%**: 评估客户流失风险
- **新竞争对手进入**: 模拟竞争压力
- **客户预算削减**: 测试业务影响
- **产品线扩展**: 预测增长机会

**模拟步骤**:
1. 选择要模拟的场景
2. 查看预测影响
3. 查看AI应对建议
4. 制定预案

## ❓ 常见问题

### Q1: 如何提高客户健康分？

**A**: 健康分由5个维度组成，建议：
1. 增加互动频率（会议、邮件、电话）
2. 提高客户满意度和产品采用率
3. 确保按时付款，建立良好信用
4. 增加营收贡献和购买频率
5. 及时响应客户需求

### Q2: 风险预警如何使用？

**A**: 
1. 每周检查一次客户风险雷达图
2. 重点关注红色区域（高风险）
3. 查看AI建议的应对策略
4. 制定具体的跟进计划
5. 记录执行结果并跟踪效果

### Q3: 营收预测准确吗？

**A**: 
- 预测基于历史数据和趋势分析
- 建议结合行业情况和客户反馈
- 定期对比实际结果优化模型
- 最佳/最差/平均三种场景供参考

### Q4: 数据从哪里来？

**A**: 
系统整合多个数据源：
- 商机管理系统
- 报价管理系统
- 产品数据库
- 知识库
- 学习训练数据
- 手动录入数据

### Q5: 如何添加新客户？

**A**: 
- **方法1**: 前台 → 明升AICRM → 商机管理 → 添加客户
- **方法2**: 后台 → 客户360画像 → 添加客户
- **方法3**: 通过商机自动创建客户

### Q6: 联系人网络如何维护？

**A**: 
1. 添加客户后，补充联系人信息
2. 标注每个联系人的角色和重要性
3. 记录与联系人的互动历史
4. 系统自动分析关系强度
5. 定期更新联系人状态
