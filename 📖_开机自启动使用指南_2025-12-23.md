# AIWEB 开机自启动使用指南

## 📋 功能说明

设置AIWEB开机自动启动，无需每次手动启动服务器。

---

## 🚀 快速设置（3步完成）

### 第一步：运行配置工具
1. 找到文件：`🔧_设置开机自启动_管理员运行.bat`
2. **右键点击** → 选择 **"以管理员身份运行"**
3. ⚠️ 必须使用管理员权限，否则无法设置

### 第二步：选择操作
```
选择操作:
[1] 设置开机自启动
[2] 取消开机自启动
[3] 查看自启动状态

请输入选项 (1/2/3):
```

**输入 `1` 并回车** → 设置开机自启动

### 第三步：完成
看到以下提示即表示成功：
```
✅ 开机自启动设置成功！

配置信息:
   启动文件: C:\...\⚡_一键启动AIWEB_2025-12-22.bat
   快捷方式: C:\...\Startup\明升AIWEB.lnk

💡 下次开机时，AIWEB将自动启动
```

## ✅ 验证配置

### 方法1：查看状态
运行配置工具，选择选项 `3` 查看状态

### 方法3：重启电脑测试
重启后观察：
- ✅ 自动弹出2个命令窗口（前端 + 后端）
- ✅ 自动打开浏览器访问 http://localhost:3002
- ✅ AIWEB首页正常显示

```
开机启动流程:
1. Windows启动
   ↓
2. 执行启动文件夹中的快捷方式
   ↓
3. 运行 ⚡_一键启动AIWEB_2025-12-22.bat

   ↓

   ↓
7. AIWEB就绪 ✅
```

## 🔧 管理操作

### 暂时禁用（不删除配置）
1. 按 `Win + R`
2. 输入：`shell:startup`
3. 将 `明升AIWEB.lnk` **重命名**为 `明升AIWEB.lnk.bak`
4. 下次开机不会自动启动
5. 需要时改回原名称即可

### 永久取消
运行配置工具，选择选项 `2` 取消开机自启动

### 修改启动参数
如需自定义启动行为，可以编辑：
`⚡_一键启动AIWEB_2025-12-22.bat`

## ⚠️ 注意事项

### 1. 管理员权限
- ✅ 必须以管理员身份运行配置工具
- ❌ 普通权限无法写入启动文件夹

### 2. 开机启动时间
- 开机后需等待 **10-15秒** 服务器完全启动
- 不要在服务器启动完成前关闭命令窗口

### 3. 命令窗口
- 两个命令窗口（前端 + 后端）必须保持运行
- 关闭任一窗口会导致服务停止
- 可以最小化，不影响使用

### 4. 端口冲突
- 如果端口3000或3002被占用，启动会失败
- 解决方案：关闭占用端口的程序

### 5. 开机延迟
- 自启动会略微增加开机时间（约5-10秒）
- 如需快速启动系统，建议暂时禁用

## 🛠️ 故障排除

### 问题1：设置后没有自动启动
**原因**：可能启动文件夹路径错误
**解决**：
1. 手动检查：`shell:startup` 中是否有快捷方式
2. 右键快捷方式 → 属性 → 检查目标路径

### 问题2：启动失败，弹出错误
**原因**：Node.js未安装或环境变量问题
**解决**：
1. 确认Node.js已正确安装
2. 在命令提示符中输入 `node -v` 验证
3. 重启电脑使环境变量生效

### 问题3：浏览器打开但页面空白
**原因**：服务器启动时间不足
**解决**：
1. 等待30秒后手动刷新浏览器
2. 或编辑启动脚本，增加等待时间：
   ```batch
   timeout /t 15 >nul  (从8改为15)
   ```

### 问题4：每次开机都要手动确认
**原因**：防火墙或安全软件拦截
**解决**：
1. 将 `node.exe` 添加到防火墙白名单
2. 在安全软件中设置信任
