```javascript
const addFeature = (agent) => {

const removeFeature = (agent, index) => {
  agent.features.splice(index, 1)
}
```

#### 2. 图片上传
```javascript
const handleImageChange = (agent, field, file) => {
  const reader = new FileReader()
  reader.onload = (e) => {
    agent[field] = e.target.result
  }
  reader.readAsDataURL(file.raw)
}
```

## 🎓 使用建议

### 内容编辑最佳实践
1. **标题**: 控制在10字以内，简明扼要
2. **描述**: 分段书写，突出重点
3. **功能**: 3-5条为佳，突出核心价值
4. **场景**: 具体化，贴近用户需求
5. **图片**: 统一风格，专业美观

### 图片优化建议
- 尺寸: 800x600px（推荐）
- 格式: PNG或JPG
- 大小: < 500KB
- 内容: 体现智能体特色
- 背景: 简洁干净

### 操作技巧
- 先编辑后保存，避免频繁操作
- 使用预览功能确认效果
- 重要修改前截图备份
- 分批编辑，逐步完善
