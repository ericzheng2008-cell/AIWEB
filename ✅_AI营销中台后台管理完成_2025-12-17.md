## 📖 使用指南

### 配置Phase 2功能

#### 1. AI产品选型系统
```
1. 调整推荐准确率滑块（目标：92%+）
2. 设置问答步骤数（推荐：4步）
3. 上传训练数据集（CSV/JSON格式）
4. 点击"配置AI模型"进行高级设置
```

#### 2. 资源中心
```
1. 添加资源分类标签
2. 开启下载追踪（用于AI分析）
3. 点击"管理资源文件"上传资源
```

#### 3. AI询盘评分
```
1. 调整4个维度的权重滑块
2. 配置A/B/C/D分级标准
3. 设置自动跟进策略
4. 点击"重新训练评分模型"优化算法
```

### 配置Phase 3功能

#### 1. AI邮件营销
```
1. 设置每日发送上限
2. 开启A/B测试，设置测试组占比
3. 编辑邮件模板或使用AI生成
4. 监控发送统计数据
```

#### 2. 线索孵化
```
1. 配置6阶段漏斗
2. 创建自动化工作流
3. 设置触发条件和动作
4. 启用/禁用工作流
```

### 配置Phase 4功能

#### 1. 营销数据中台
```
1. 选择要集成的数据源
2. 开启实时数仓
3. 设置刷新间隔（推荐：5秒）
4. 配置可视化看板
```

#### 2. AI预测分析
```
1. 选择预测模型（推荐：集成学习）
2. 勾选预测场景
3. 点击"开始训练"训练模型
4. 查看预测准确率
```
