# AI智能客服快速使用指南

## 🚀 快速开始

### 1. 打开AI客服
- 访问网站任意页面
- 点击**右下角蓝色圆形按钮**
- AI客服窗口弹出

### 2. 开始对话
在输入框中输入问题,例如:
```
你们主要的产品有哪些?
如何联系销售团队?
支持定制化服务吗?
有哪些应用案例?
```

### 3. 切换语言
- 点击右上角 **"中文 / EN"**
- 整站内容切换为英文
- AI客服也会用英文回答

## 🎓 AI知识库管理

### 访问管理后台
```
http://localhost:3000/ai-knowledge
```

### 添加新知识
1. 点击 **"添加知识"** 按钮
2. 选择分类 (如: 产品介绍)
3. 添加关键词 (如: 价格、报价、多少钱)
4. 填写中文回答
5. 填写英文回答
6. 点击保存

## 🔑 关键词设置技巧

### 技巧1: 多角度覆盖
同一个问题可能有多种问法:
```
✅ 好的关键词组合:
价格、报价、多少钱、cost、price、quote、费用

❌ 不好的关键词:
只设置"价格"一个关键词
```

### 技巧2: 中英文结合
```
✅ 中英文关键词:
产品、工具、设备、product、tool、equipment

❌ 只有中文:
产品、工具、设备
```

### 技巧3: 包含常见口语
```
✅ 包含口语化表达:
怎么联系、如何找到你们、在哪里、contact、where、how to reach

❌ 只有正式用语:
联系方式、contact information
```

## 🔧 故障排查

### 问题1: AI没有回答
**原因**: 没有匹配到关键词  
**解决**: 在知识库中添加相关关键词

### 问题2: 回答不准确
**原因**: 关键词匹配错误分类  
**解决**: 优化关键词,使其更具体

### 问题3: 语言切换无效
**原因**: 浏览器缓存  
**解决**: 清除缓存或使用无痕模式

### 问题4: 知识库导入失败
**原因**: JSON格式错误  
**解决**: 检查JSON格式,确保符合规范
