### 3. 节点测试功能 ⭐ (核心特性)

**使用方式**:
```
1. 选中节点
2. 切换到"执行"标签
3. 点击"执行此节点"
4. 查看执行结果
```

**执行结果包含**:
- ✅ 成功/失败状态
- 📥 输入数据 (JSON)
- 📤 输出数据 (JSON)
- ⏱️ 执行时长 (ms)
- 🎯 Token消耗 (AI节点)
- 🕐 执行时间戳

**示例**:
```json
{
  "success": true,
  "timestamp": "2025-12-20 10:30:45",
  "duration": 342,
  "tokens": 156,
  "input": {
    "deviceId": "DEV001",
    "faultCode": "E103"
  },
  "output": {
    "faultType": "传感器故障",
    "severity": "中等",
    "solution": "更换温度传感器"
  }
}
```
