## 📋 功能概述

产品技术销售小课堂是一个专业的工业自动化技术知识分享平台，为技术人员和销售人员提供系统化的学习资源。支持多级分类体系，涵盖协作机器人、AGV/AMR、PLC、拧紧工艺、视觉检测等多个技术领域。

---

## 🎯 核心功能

### 1. 多级分类体系

#### 一级分类（9个技术领域）
- 🤖 **协作机器人** - 协作机器人技术、应用场景、编程与集成
- 🚗 **AGV/AMR** - AGV自动导引车和AMR自主移动机器人技术
- 💻 **PLC控制系统** - PLC编程、工业控制系统设计与应用
- 🔧 **拧紧工艺与工具** - 拧紧技术、工艺参数、工具选型与维护
- 💧 **自动涂胶SPRFDS** - 自动涂胶系统、流体分配、密封胶应用
- 👁️ **机器视觉** - 视觉检测、图像处理、AI视觉识别
- 📏 **精密测量** - 工业测量技术、尺寸检测、质量控制
- ⚡ **焊装NC柔性系统** - NC焊接系统、柔性制造、焊装工艺
- 🛠️ **设备服务知识** - 设备维护、故障诊断、服务技术

#### 二级分类（每个一级分类包含多个专题）
每个一级分类下包含4-5个二级专题分类，例如：
- 协作机器人 → 基础知识、编程调试、安全规范、应用案例
- 拧紧工艺与工具 → 工艺基础、工具选型、Open Protocol协议、质量管理、问题解决

### 2. 课程管理功能

#### 课程属性
- **基本信息**: 标题、简短描述、详细内容
- **分类归属**: 一级分类、二级分类
- **教学信息**: 难度等级（入门/初级/中级/高级）、课程时长、讲师
- **多媒体**: 封面图片、图文内容
- **互动数据**: 浏览量、点赞数
- **标签系统**: 多标签支持，便于检索
- **发布状态**: 已发布/草稿

### 3. 前台展示功能

#### 技术分类展示
- 3列网格布局展示9个技术领域
- 每个分类卡片显示图标、名称、描述、主题数、课程数
- 悬停动画效果，提升交互体验

#### 课程浏览流程
1. **选择一级分类** → 打开二级分类对话框
2. **选择二级分类** → 显示该分类下的所有课程
3. **选择课程** → 查看课程详情

#### 课程详情页面
- 封面图片展示
- 课程元信息（难度、讲师、时长、浏览量、点赞数）
- 课程简介
- 详细内容（支持长文本）
- 标签展示
- 点赞功能

### 4. 后台管理功能

#### 一级分类管理
- 添加/编辑/删除一级分类
- 设置分类名称、图标（Emoji）、描述、排序
- 查看该分类下的二级分类数量和课程数量

#### 二级分类管理
- 按一级分类分组展示
- 添加/编辑/删除二级分类
- 选择所属一级分类
- 设置分类名称和排序

#### 课程管理
- 完整的CRUD操作
- 筛选功能（按一级分类、二级分类）
- 富表单编辑器
  - 级联选择一级和二级分类
  - 标题、描述、详细内容编辑
  - 封面图片上传（支持本地上传和URL输入）
  - 难度、时长、作者设置
  - 多标签选择
  - 发布状态控制
- 课程数据统计（浏览量、点赞数）

## 🗂️ 技术架构

### Store状态管理（Pinia）
**文件**: `src/store/classroom.js`

#### State
```javascript
{
  categories: [],      // 一级分类列表
  subcategories: [],   // 二级分类列表
  lessons: []          // 课程列表
}
```

#### Getters
- `getAllCategories` - 获取所有一级分类
- `getSubcategoriesByCategory` - 根据一级分类获取二级分类
- `getLessonsBySubcategory` - 根据二级分类获取课程
- `getLessonsByCategory` - 根据一级分类获取所有课程
- `getLessonById` - 获取课程详情

#### Actions
- 分类管理: `addCategory`, `updateCategory`, `deleteCategory`
- 子分类管理: `addSubcategory`, `updateSubcategory`, `deleteSubcategory`
- 课程管理: `addLesson`, `updateLesson`, `deleteLesson`
- 互动功能: `incrementViews`, `incrementLikes`
- 数据持久化: `saveToLocalStorage`, `loadFromLocalStorage`

### 前台视图组件
**文件**: `src/views/TechClassroom.vue`

#### 核心功能
- 技术分类卡片展示（3列网格）
- 二级分类对话框（2列网格）
- 课程列表对话框（3列网格）
- 课程详情对话框（全屏展示）
- AOS动画集成
- 响应式布局

### 后台管理组件
**文件**: `src/views/admin/ClassroomManage.vue`

#### 三个标签页
1. **一级分类管理** - 表格形式管理技术领域
2. **二级分类管理** - 按一级分类分组展示
3. **课程管理** - 带筛选的课程列表和编辑功能

## 💾 数据持久化

使用 **localStorage** 存储数据：

```javascript
// 存储键
localStorage.setItem('classroom_categories', JSON.stringify(categories))
localStorage.setItem('classroom_subcategories', JSON.stringify(subcategories))
localStorage.setItem('classroom_lessons', JSON.stringify(lessons))
```

**注意**: 数据仅在浏览器本地存储，清除浏览器缓存会丢失数据。
