## 网站概述

### 网站定位
AIWEB是一个集成了智能制造、AI营销、设备全生命周期管理等多功能的企业级Web平台。

### 核心技术栈
- **前端**: Vue 3 + Element Plus + ECharts
- **后端**: Node.js + Express + SQLite
- **部署**: 支持云端/内网双模式

### 访问方式
- **开发环境**: http://localhost:3002
- **生产环境**: http://localhost:3000

#### 2.2 手动录入生产数据
![手动录入截图](AIMES → 数据录入)

**功能说明**:
- **三类数据**:
  1. **生产数据**: 生产线、工单号、产品型号、计划/实际产量、合格率
  2. **质量数据**: 检验批次、检验项目、合格数、不良数、不良原因
  3. **设备数据**: 设备编号、运行状态、故障类型、停机时间

**表单验证**:
- 必填项检查
- 数值范围验证
- 日期格式验证
- 逻辑关系验证（如：实际产量≤计划产量）

**操作流程**:
1. 选择数据类型标签页
