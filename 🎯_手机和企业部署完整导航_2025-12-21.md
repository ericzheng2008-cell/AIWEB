## 🔧 常见问题速查

### Q1: 手机无法访问怎么办？

**排查步骤**：

1. **检查WiFi连接**
   ```
   电脑WiFi: [查看WiFi名称]
   手机WiFi: [必须一致]
   ```

2. **检查服务状态**
   ```
   运行：🧪_测试手机访问_快速验证.bat
   ```

3. **检查防火墙**
   ```
   运行：🔧_配置Windows防火墙_管理员运行.bat
   ```

4. **临时关闭防火墙测试**
   ```
   控制面板 → Windows Defender防火墙 → 关闭（测试后记得开启）
   ```

### Q2: 页面加载很慢？

**解决方案**：

1. **使用生产构建**
   ```bash
   npm run build
   npx serve dist -p 3002
   ```

2. **检查电脑性能**
   - 关闭不必要的程序
   - 确保CPU < 80%
   - 确保内存充足

3. **使用5G WiFi**
   - 2.4G频段：速度慢但稳定
   - 5G频段：速度快但距离短

### Q3: 某些功能不可用？

**检查项目**：

1. **浏览器兼容性**
   - Chrome 90+
   - Safari 14+
   - Edge 90+

2. **JavaScript错误**
   - 打开浏览器控制台（F12）
   - 查看Console标签
   - 截图发送给技术支持

3. **后端日志**
   - 查看后端命令行窗口
   - 查找error关键字

### Q4: 如何添加到手机桌面？

**iOS系统**：
1. Safari打开网站
2. 点击底部「分享」按钮
3. 向下滚动，选择「添加到主屏幕」
4. 编辑名称：`明升AIWEB`
5. 点击「添加」

**Android系统**：
1. Chrome打开网站
2. 点击右上角「⋮」菜单
3. 选择「添加到主屏幕」
4. 编辑名称：`明升AIWEB`
5. 点击「添加」

**必备技能**：
- ⭐⭐⭐⭐⭐ Linux基础命令
