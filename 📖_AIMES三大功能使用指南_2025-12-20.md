## 1️⃣ **MES数据接口使用指南**

### **第一步：打开接口配置**
```
AIMES助手页面 → 头部工具栏 → 点击"MES数据接口"绿色按钮
```

### **第二步：配置MES系统**

在"接口配置"标签页中：

| 配置项 | 说明 | 示例 |
|-------|------|------|
| MES系统类型 | 选择您使用的MES系统 | SAP MES / Oracle MES |
| 接口地址 | MES系统的API地址 | `http://mes.company.com:8080/api` |
| 认证方式 | 选择API Key/OAuth/Basic Auth | API Key |
| API Key | 输入API密钥 | `sk-xxxxxxxxxxxxx` |
| 同步频率 | 数据同步间隔时间 | 1分钟（60秒）|
| 数据项 | 勾选需要同步的数据类型 | ☑工单 ☑生产 ☑质量 |

### **第三步：测试连接**
1. 填写完配置后，点击"测试连接"按钮
2. 等待1-2秒
3. 看到"MES接口连接成功！"提示
4. 连接状态变为"已连接"

### **第四步：启动同步**
1. 切换到"接口状态"标签
2. 确认连接状态为"已连接"
3. 点击"启动同步"按钮
4. 同步状态变为"同步中"
5. 实时查看数据同步记录

### **第五步：查看日志**
1. 切换到"同步日志"标签
2. 查看实时同步记录
3. 每次同步都会添加新的日志条目

### **常用操作**
- **保存配置**: 点击"保存配置"按钮，配置会保存到浏览器本地
- **停止同步**: 在"接口状态"标签点击"停止同步"
- **刷新状态**: 点击"刷新状态"查看最新的连接状态

## 2️⃣ **手动录入数据使用指南**

### **第一步：打开录入窗口**
```
AIMES助手页面 → 头部工具栏 → 点击"手动录入数据"黄色按钮
```

### **第二步：录入生产数据**

在"生产数据"标签页中：

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| 产线 | ✅ | 选择生产线 | 机加工产线1号 |
| 工单号 | ✅ | 输入工单编号 | WO-20251220-001 |
| 产品编码 | ✅ | 产品型号代码 | PROD-2025-A01 |
| 批次号 | | 批次追溯编号 | BATCH-20251220-001 |
| 计划产量 | ✅ | 计划生产数量 | 1000 |
| 实际产量 | ✅ | 实际完成数量 | 985 |
| 不良品数 | | 不合格产品数量 | 15 |
| 开始时间 | | 生产开始时间 | 2025-12-20 08:00 |
| 结束时间 | | 生产结束时间 | 2025-12-20 18:00 |

### **第三步：录入质量数据**

切换到"质量数据"标签：

| 字段 | 说明 | 示例 |
|------|------|------|
| 检验人员 | 质检员姓名 | 张三 |
| 检验时间 | 检验完成时间 | 2025-12-20 18:30 |
| 抽检数量 | 抽样检验数量 | 50 |
| 合格数量 | 合格产品数 | 48 |
| 不合格数量 | 不合格产品数 | 2 |
| 不良类型 | 勾选缺陷类型 | ☑尺寸偏差 ☑外观缺陷 |
| 备注 | 质量问题描述 | 部分产品表面有划痕 |

### **第四步：录入设备数据**

切换到"设备数据"标签：

| 字段 | 说明 | 示例 |
|------|------|------|
| 设备编号 | 设备唯一标识 | CNC-B02 |
| 设备状态 | 当前运行状态 | 运行中 |
| 运行时间 | 累计运行小时数 | 8.5 |
| 停机时间 | 累计停机小时数 | 0.5 |
| 能耗 | 能源消耗（kWh）| 125.8 |
| 故障描述 | 故障详细描述 | （如有故障填写）|

### **第五步：提交数据**
1. 检查必填项是否已填写
2. 点击"提交数据"按钮
3. 等待验证和提交
4. 看到"生产数据已成功录入！"提示
5. 数据自动更新到AIMES仪表盘

### **快捷操作**
- **重置表单**: 点击"重置"按钮清空所有输入
- **取消录入**: 点击"取消"按钮关闭窗口
- **切换标签**: 点击标签快速切换数据类型

## 💡 **使用技巧**

### **MES接口技巧**
1. **首次使用**: 先测试连接，确认成功后再配置其他参数
2. **同步频率**: 建议从1分钟开始，根据实际情况调整
3. **数据项选择**: 只勾选需要的数据项，避免不必要的流量
4. **日志查看**: 定期查看同步日志，及时发现问题

### **手动录入技巧**
1. **批量录入**: 可以分标签页填写，最后一次性提交
2. **数据校验**: 注意必填项标记（红色星号）
3. **快速重置**: 填写错误时使用"重置"按钮清空重填
4. **时间选择**: 使用日期选择器快速选择时间

### **图表查看技巧**
1. **趋势判断**: 观察曲线斜率判断变化速度
2. **异常识别**: 关注突然下降或波动较大的点
3. **对比分析**: 同时查看多条趋势线找出关联
4. **数据导出**: 可截图保存用于报告

## ❓ **常见问题**

### **Q1: MES接口连接失败怎么办？**
A: 检查以下几点：
- ✅ 接口地址是否正确
- ✅ 网络是否畅通
- ✅ 认证信息是否正确
- ✅ MES系统是否正常运行

### **Q2: 手动录入数据后在哪里查看？**
A: 数据会自动更新到：
- ✅ 质量合格率（todayYieldRate）
- ✅ MES同步日志
- ✅ 实时生产监控大屏

### **Q3: 图表显示空白怎么办？**
A: 尝试以下方法：
- ✅ 刷新页面
- ✅ 切换到其他标签再切回AIMES
- ✅ 点击"刷新数据"按钮
- ✅ 清除浏览器缓存后重新加载

### **Q4: 如何保存MES配置？**
A: 点击"接口配置"标签中的"保存配置"按钮，配置会自动保存到浏览器LocalStorage，下次打开自动加载。
