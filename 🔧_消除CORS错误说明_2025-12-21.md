# 🔧 CORS错误说明与解决方案

## ❓ 什么是CORS错误?

CORS (Cross-Origin Resource Sharing) 跨域错误是浏览器的安全机制:
- 当直接打开本地HTML文件时 (`file://` 协议)
- 浏览器会阻止对外部网站的API请求
- 这是**正常的安全保护**,不是代码问题

## 📊 您看到的错误

```
Access to fetch at 'https://dsp.lenovo.com.cn/lenovo/bid...' 
from origin 'null' has been blocked by CORS policy
```

**原因**: 这是联想广告SDK的外部请求被浏览器拦截

## ✅ 解决方案

### 方案1: 使用开发服务器(推荐) ⭐

**双击运行**:
```
⚡_快速测试网页_2025-12-21.bat
```

或者:
```
本地启动并测试.bat
```

通过 `http://localhost:5173` 访问,CORS错误自动消失。

### 方案2: 理解这是正常现象

✅ **不影响功能** - 其他所有功能都能正常使用  
✅ **部署后消失** - 网站上线后不会出现这个错误  
✅ **可以忽略** - 这只是外部广告SDK的请求  

### 方案3: 移除广告SDK(可选)

如果不需要联想广告,可以移除相关代码:

1. 找到引用 `dsp.lenovo.com.cn` 的文件
2. 删除或注释掉相关代码
