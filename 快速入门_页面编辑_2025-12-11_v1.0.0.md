# 页面内容编辑快速入门

## 🚀 5分钟快速上手

### 第一步：登录管理后台

1. 访问：`http://localhost:3004/admin/login`
2. 输入默认账号：
   - 用户名：`admin`
   - 密码：`admin123`
3. 点击登录

### 第二步：进入页面管理

登录后，点击左侧菜单 **"页面内容管理"**

## 📝 编辑示例

### 例1：修改首页Banner

```
1. 切换到"首页内容"标签
2. 修改内容：
   
   中文标题：广州市明升伟业机电有限公司
   英文标题：Guangzhou Mingsheng Industry Co., Ltd.
   
   中文副标题：成立于1996年，总部位于珠江三角洲美丽的花城——广州
   英文副标题：Established in 1996, headquartered in Guangzhou
   
   中文描述：是一家集科研开发、生产制造、销售服务为一体的现代化科技企业
   英文描述：A modern technology enterprise integrating R&D, manufacturing and sales

3. 点击背景图片上传区域，上传公司大楼照片
   或粘贴图片URL：
   https://images.unsplash.com/photo-1581091226825-a6a2a5aee158?w=1920

4. 点击"保存首页内容"
```

### 例2：添加公司简介模块

```
1. 在"首页内容"标签下
2. 点击"添加内容模块"按钮
3. 填写内容：

   中文标题：关于明升伟业
   英文标题：About Mingsheng
   
   中文内容：
   广州市明升伟业机电有限公司成立于1996年，是一家专注于工业
   智能装配领域的高新技术企业。公司总部位于广州，业务覆盖全国。
   我们致力于为客户提供专业的工业工具和智能装配解决方案。
   
   英文内容：
   Guangzhou Mingsheng Industry Co., Ltd., established in 1996, 
   is a high-tech enterprise focusing on industrial intelligent 
   assembly. Headquartered in Guangzhou with nationwide business 
   coverage, we are committed to providing professional industrial 
   tools and intelligent assembly solutions.

4. 上传公司照片或粘贴图片URL
5. 点击"保存"
```

### 例3：编辑"关于我们"页面

```
1. 切换到"关于我们"标签
2. 编辑Banner：

   中文标题：关于我们
   英文标题：About Us
   
   中文副标题：致力于成为工业领域的领军企业
   英文副标题：Committed to becoming a leading enterprise
   
3. 点击"保存Banner配置"

4. 添加企业文化模块：
   - 点击"添加内容模块"
   - 标题：企业使命 / Our Mission
   - 内容：描述公司的使命和愿景
   - 上传团队照片
   - 保存

5. 添加发展历程模块：
   - 再次点击"添加内容模块"
   - 标题：发展历程 / Development History
   - 内容：
     1996年：公司成立
     2000年：扩大业务范围
     2010年：引进智能制造技术
     2020年：成为行业领军企业
   - 上传历史照片
   - 保存
```

## 🎨 图片使用技巧

### 免费高质量图片来源

1. **Unsplash**（推荐）
   ```
   工业类：
   https://images.unsplash.com/photo-1581091226825-a6a2a5aee158?w=1920
   
   办公类：
   https://images.unsplash.com/photo-1497366216548-37526070297c?w=1920
   
   团队类：
   https://images.unsplash.com/photo-1522071820081-009f0129c71c?w=1920
   ```

2. **图片尺寸调整**
   在URL后面添加参数：
   - `?w=1920` - 宽度1920px
   - `?w=800&h=600` - 指定宽高

### 图片上传方式

**方式1：本地上传**
- 点击上传区域
- 选择图片文件
- 等待上传完成

**方式2：网络URL**
- 复制图片URL
- 粘贴到输入框
- 按回车确认

**方式3：直接替换**
- 点击已上传的图片
- 选择新图片
- 无需删除再上传

## ✅ 验证修改效果

### 方法1：返回前台查看
1. 点击右上角"返回前台"
2. 浏览修改的页面
3. 检查显示效果

### 方法2：新标签页对比
1. 在新标签页打开前台页面
2. 在管理后台修改内容
3. 刷新前台页面查看效果

## 💡 常用操作速查

| 操作 | 步骤 |
|------|------|
| **修改首页标题** | 首页内容 → 编辑Banner → 保存 |
| **添加内容模块** | 选择页面 → 添加内容模块 → 填写内容 → 保存 |
| **编辑已有模块** | 找到模块 → 点击"编辑" → 修改 → 保存 |
| **删除模块** | 找到模块 → 点击"删除" → 确认 |
| **上传图片** | 点击上传区域 → 选择文件 |
| **替换图片** | 点击图片 → 选择新图片 |
| **使用网络图片** | 粘贴URL → 回车 |
| **预览效果** | 返回前台 → 刷新页面 |

## ⚠️ 注意事项

1. **保存数据**
   - 每次编辑后必须点击"保存"
   - 未保存的修改会丢失

2. **图片大小**
   - Banner图：建议小于2MB
   - 内容配图：建议小于1MB
   - 过大会影响加载速度

3. **内容长度**
   - 标题：建议20字以内
   - 描述：建议200字以内
   - 长文本可分多个模块

4. **数据备份**
   - 重要修改前先备份
   - 定期导出数据
   - 清除缓存会丢失数据
