## 🐛 常见问题

### Q: 动画不显示？
**A**: 
1. 确认aos库已安装: `npm list aos`
2. 清除缓存: `npm run dev`
3. 检查浏览器控制台

### Q: 构建失败？
**A**:
1. 检查Node.js版本 (需要18+)
2. 删除node_modules重新安装
3. 查看错误信息

### Q: 页面空白？
**A**:
1. 检查浏览器控制台错误
2. 确认服务器正常启动
3. 访问正确的端口(5173)

更多问题请查看: `使用指南_UX优化功能_2025-12-13_v1.0.0.md`
