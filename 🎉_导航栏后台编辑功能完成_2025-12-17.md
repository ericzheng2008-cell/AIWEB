## 📖 快速上手

### **1. 访问后台管理**

```bash
访问地址：http://localhost:5173/admin
登录管理员账号
```

### **2. 进入配置页面**

```
侧边栏 → 产品与服务管理 → 导航栏配置标签
```

### **3. 编辑配置**

| 操作 | 说明 |
|------|------|
| **修改名称** | 直接在表格输入框中编辑 |
| **调整顺序** | 点击 ☰ 图标上下拖动 |
| **控制可见** | 切换右侧开关按钮 |
| **保存配置** | 点击右上角"保存配置"按钮 |

### **4. 查看效果**

```
前台页面 F5 刷新 → 顶部导航栏"产品与服务"下拉菜单已更新
```
