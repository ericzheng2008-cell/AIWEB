## ❓ 常见问题排查

### 问题1: 在AI智能体页面看不到新卡片

**原因**: 浏览器缓存或开发服务器未重启

**解决方法**:
1. 硬刷新页面: `Ctrl + Shift + R` (Windows) 或 `Cmd + Shift + R` (Mac)
2. 清除浏览器缓存
3. 确认开发服务器已重启
4. 检查浏览器控制台是否有错误

### 问题2: 点击卡片无反应

**原因**: 路由配置问题或JavaScript错误

**解决方法**:
1. 打开浏览器开发者工具 (F12)
2. 查看Console标签页是否有红色错误信息
3. 检查Network标签页，看ProcessVerification.vue是否成功加载
4. 尝试直接访问: `http://localhost:5173/process-verification`

### 问题3: 直接访问工艺验证页面显示空白

**原因**: 组件加载失败或依赖缺失

**解决方法**:
1. 检查文件是否存在: 
   ```bash
   dir src\views\ProcessVerification.vue
   ```
   应该显示 56,754 字节

2. 检查浏览器控制台错误:
   - 如果是模块导入错误，重启服务器
   - 如果是组件错误，查看具体错误信息

3. 检查依赖是否完整:
   ```bash
   npm install
   ```

### 问题4: 统计数字仍显示"7+智能体"

**原因**: 页面缓存

**解决方法**:
1. 硬刷新浏览器
2. 检查AiAgents.vue第85行是否已更新为"8+"
3. 重新启动开发服务器
