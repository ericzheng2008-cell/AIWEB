## 🎯 技术亮点

### 1. 事件冒泡控制
```vue
<!-- 明升企业智能体 -->
<div class="core-agent-card" @click="scrollToAgents">
  <el-button @click.stop="backToHome">返回主页</el-button>
</div>
```
**作用**：防止点击返回按钮时触发卡片的 `scrollToAgents` 事件

### 2. 毛玻璃效果
```css
.card-back-home .el-button {
  background: rgba(255, 255, 255, 0.95);
  backdrop-filter: blur(8px);
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.15);
}
```
**效果**：半透明白色背景 + 背景模糊 + 柔和阴影

### 3. 渐变背景
```css
.dialog-back-home .el-button {
  background: linear-gradient(135deg, #409EFF 0%, #2196F3 100%);
}
```
**效果**：蓝色渐变，视觉层次更丰富

### 4. 平滑滚动
```javascript
window.scrollTo({ top: 0, behavior: 'smooth' })
```
**效果**：流畅的滚动动画，优于瞬间跳转

### 5. 条件显示
```css
.card-back-home {
  opacity: 0;
  transition: opacity 0.3s ease;
}

.core-agent-card:hover .card-back-home {
  opacity: 1;
}
```
**效果**：悬浮时淡入，离开时淡出
