# 产品与服务多级分类系统功能文档

**版本：** v1.0.0  
**日期：** 2025-12-14  
**状态：** ✅ 已完成

---

## 📋 功能概述

实现了完整的产品与服务多级分类管理系统，支持：
- **一级分类**：6个子系统（与导航栏保持一致）
- **二级分类**：每个一级分类下可添加多个二级分类
- **三级分类**：每个二级分类下可添加多个三级分类
- **产品详情**：每个三级分类下可添加具体产品
- **PANTONE 2736C配色**：除首页外的所有页面使用统一主色调

---

## 🎯 一级分类（6个子系统）

与顶部导航栏保持一致的6个子系统：

1. **电动工具** (Electric Tools)
   - 图标：Lightning
   - 描述：专业电动拧紧工具、电动螺丝刀等

2. **气动工具** (Pneumatic Tools)
   - 图标：Wind
   - 描述：气动拧紧工具、气动扳手、气动螺丝刀等

3. **手动工具** (Manual Tools)
   - 图标：Tools
   - 描述：扭力扳手、手动拧紧工具、套筒等

4. **测量工具** (Measurement Tools)
   - 图标：DataAnalysis
   - 描述：扭力测试仪、角度测量仪、传感器等

5. **自动化系统** (Automated Systems)
   - 图标：Monitor
   - 描述：自动拧紧系统、装配线、工作站等

6. **智能解决方案** (Smart Solutions)
   - 图标：Management
   - 描述：数据采集、MES系统、AI分析等

## 🎨 PANTONE 2736C配色方案

### 主色定义
```css

--pantone-2736c-light: #0066dd;
--pantone-2736c-dark: #003d99;
--pantone-2736c-lightest: #e6f0ff;
```

### 应用场景
- ✅ 页面横幅背景渐变
- ✅ 卡片悬停边框和阴影
- ✅ 按钮主色调
- ✅ 链接文字颜色
- ✅ 对话框标题背景
- ✅ 图标覆盖层

### 渐变效果
```css

```

## 🔄 数据存储

### LocalStorage键值
- `productsLevel1` - 一级分类数据
- `productsLevel2` - 二级分类数据
- `productsLevel3` - 三级分类数据
- `productsData` - 产品数据
- `productsServicesPageSettings` - 页面设置（横幅等）

## 🎯 交互流程

### 前台浏览流程
```
一级分类卡片 → 点击 → 二级分类对话框
                      ↓ 点击
                三级分类对话框
                      ↓ 点击
                产品列表对话框
                      ↓ 点击
                  产品详情页
```

### 后台管理流程
```
1. 创建一级分类（6个子系统）
2. 为每个一级分类添加二级分类
3. 为每个二级分类添加三级分类
4. 为每个三级分类添加具体产品
5. 设置页面横幅样式
```

## 📝 使用建议

1. **分类规划**
   - 提前规划好产品分类层级
   - 保持分类结构清晰简洁
   - 每级分类不宜超过10个

2. **图片规格**
   - 一级分类图片：600x400px
   - 产品图片：800x600px
   - 横幅图片：1920x600px

3. **内容编写**
   - 分类描述简洁明了
   - 产品描述突出特点
   - 中英文内容保持一致

4. **配色使用**
   - 主色仅用于除首页外的页面
   - 保持配色统一性
   - 合理使用渐变和透明度
