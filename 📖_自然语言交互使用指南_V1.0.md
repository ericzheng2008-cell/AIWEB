## 📋 Phase 1: 快速指令使用

### 什么是快速指令?
预设的常用查询指令,一键执行,无需输入文字。

### 可用的快速指令

#### 客户相关 (跳转到客户360°)
1. **🌟 高价值客户**
   - 查询: CLV > 50000
   - 用途: 快速查看VIP客户列表

2. **⚠️ 流失风险客户**
   - 查询: churnRisk > 70%
   - 用途: 及时发现需要挽留的客户

3. **👤 本月新客户**
   - 查询: 本月新增客户
   - 用途: 了解新客户获取情况

4. **💬 客户满意度低**
   - 查询: NPS < 6
   - 用途: 发现需要改进服务的客户

#### 商机相关 (跳转到销售漏斗)
5. **🚩 待跟进商机**
   - 查询: 超过7天未跟进
   - 用途: 防止商机流失

#### 销售分析 (跳转到销售目标)
6. **📈 本季度销售额**
   - 查询: 本季度销售统计
   - 用途: 查看销售目标达成情况

#### 数据分析 (跳转到客户沙盘)
7. **📊 活跃客户分析**
   - 查询: 本月活跃客户
   - 用途: 分析客户活跃度趋势

#### 产品相关 (跳转到产品矩阵)
8. **🏆 产品销售排行**
   - 查询: 按收入排序
   - 用途: 了解畅销产品

### 使用步骤
```
1. 点击搜索框
2. 确保"AI模式"开关为OFF
3. 在"⚡ 快速指令"区域选择指令
4. 系统自动执行并跳转到对应视图
5. 查询记录保存到历史
```

## 🧠 Phase 2: NLP智能解析

### 什么是NLP解析?
输入自然语言,系统自动识别关键词并执行查询。

### 支持的关键词

#### 时间关键词
- **本月** / **上月**
- **本季度** / **本年**
- **今天** / **本周**
- **最近7天** / **最近30天**

#### 客户分类
- **高价值** - CLV > 50000
- **流失风险** - churnRisk > 70%
- **活跃** - 最近30天有互动
- **沉睡** - 超过90天无互动
- **新客户** - 注册时间 < 30天

#### 指标关键词
- **销售额** - revenue
- **转化率** - conversionRate
- **客单价** - avgOrderValue
- **毛利** - grossProfit

#### 动作关键词
- **查询** - 执行查询操作
- **分析** - 进行数据分析
- **预测** - AI预测分析
- **推荐** - 获取AI推荐
- **统计** - 数据统计
- **对比** - 数据对比

### 使用示例

#### 示例1: 简单查询
```
输入: 查询本月高价值客户
解析: 
  - 动作: 查询
  - 时间: 本月
  - 客户类型: 高价值
执行: 跳转到客户360°视图
```

#### 示例2: 组合查询
```
输入: 分析本季度新客户的转化率
解析:
  - 动作: 分析
  - 时间: 本季度
  - 客户类型: 新客户
  - 指标: 转化率
执行: 生成分析报告
```

#### 示例3: 复杂查询
```
输入: 对比本月和上月的高价值客户销售额
解析:
  - 动作: 对比
  - 时间1: 本月
  - 时间2: 上月
  - 客户类型: 高价值
  - 指标: 销售额
执行: 生成对比图表
```

### 使用技巧
1. **组合使用关键词**: 时间 + 客户类型 + 指标
2. **自然表达**: 无需记忆精确语法
3. **模糊匹配**: 支持同义词识别
4. **查看历史**: 学习成功的查询模式

## 🤖 Phase 3: AI对话助手

### 什么是AI对话助手?
类似ChatGPT的智能对话系统,支持自然语言提问,AI理解并回答。

### 启用AI模式
```
1. 点击搜索框
2. 将"AI模式"开关切换到ON
3. 看到欢迎消息: "您好！我是AICRM智能助手..."
4. 开始对话
```

### 支持的查询类型

#### 1. 流失风险分析 ⚠️
**提问示例**:
- "查询本月高流失风险客户"
- "有多少客户有流失风险"
- "哪些客户需要重点关注"

**AI回复内容**:
```
我帮您查询到本月高流失风险客户共28位，主要特征：
1. 近60天无购买记录
2. 最近客服评分下降
3. 客单价降低超过30%

建议：立即安排专属客服跟进，并发送定向优惠券。

[查看详情] ← 点击跳转到客户360°
```

#### 2. 销售额分析 📈
**提问示例**:
- "本月销售额完成情况如何"
- "分析本季度销售额"
- "销售目标达成率"

**AI回复内容**:
```
本月销售额达成情况：
- 当前: ¥1,580万
- 目标: ¥2,000万
- 完成率: 79%
- 预测: 按当前趋势预计达成率92%

建议：重点跟进5个大单客户，有望冲刺目标。

[查看销售目标] ← 点击跳转
```

#### 3. 新客户分析 👤
**提问示例**:
- "本月新客户情况"
- "新增了多少客户"
- "新客户转化率"

**AI回复内容**:
```

- 新客户数: 156位
- 主要来源: 线上推广(62%)、老客推荐(28%)
- 转化率: 18.5%
- 平均客单价: ¥3,200

建议：加强老客推荐奖励计划。

[查看新客户列表] ← 点击跳转
```

#### 4. 待跟进商机 🚩
**提问示例**:
- "有哪些商机需要跟进"
- "待办商机列表"
- "未跟进的商机"

**AI回复内容**:
```

- 超过7天未跟进: 42个
- 超过14天未跟进: 18个

建议：优先跟进高价值商机，本周内完成。

[查看商机列表] ← 点击跳转到销售漏斗
```

#### 5. 预测分析 🔮
**提问示例**:
- "预测本季度目标达成率"
- "下月销售预估"
- "流失客户预测"

**AI回复内容**:
```
AI预测分析：
- 本季度销售目标达成率: 95%
- 下月预计流失客户: 15-20位

[查看预测报告] ← 点击跳转到投标预测AI
```

点击任意推荐即可执行。

## ❓ 常见问题

### Q1: AI模式和普通模式有什么区别?
**A**: 
- **普通模式**: 快速指令 + NLP关键词解析,适合快速查询
- **AI模式**: ChatGPT式对话,支持复杂分析和建议

### Q2: 历史记录为什么只保存10条?
**A**: 保留最近10条最有价值,避免记录过多影响查找效率。

### Q3: NLP解析识别不准确怎么办?
**A**: 
1. 尝试使用更标准的关键词
2. 参考成功的历史查询模式
3. 切换到AI模式,用自然语言描述

### Q4: AI回复的数据是实时的吗?
**A**: 当前是模拟数据,后续版本将接入真实数据源。

### Q5: 可以保存AI对话记录吗?
**A**: 当前对话记录在会话中保留,刷新页面会清空。后续版本将支持持久化保存。
