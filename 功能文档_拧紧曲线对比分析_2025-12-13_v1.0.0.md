# 📊 拧紧曲线对比与分析功能文档 v2.0

## 🎯 功能概述

拧紧曲线对比与分析是工艺优化模块中的核心功能，通过导入标准拧紧曲线和现场采集的拧紧曲线进行自动对比分析，智能诊断各类装配问题。参考业界专业软件（BoltRotation、Lixing DIS、ToolsNet 8等），提供全面的曲线分析能力。

### ✨ 核心特性

- 📈 **最多支持100条曲线同时对比**
- 📁 **标准CSV格式导入**（角度,扭矩）
- 🤖 **AI智能诊断**（7种典型异常模式全覆盖）
- 📊 **实时曲线对比可视化**（基于ECharts）
- 🔍 **智能材质识别**（钢、铝合金、塑料、铸铁、复合材料）
- 🎛️ **工艺参数对比**（扭矩控制 vs 转角控制）
- 📉 **批次一致性分析**（工艺稳定性评估）
- 📝 **专业分析报告导出**

#### 支持的异常模式识别（全面增强版）

##### 1. **内螺纹滑牙（严重）**
**特征**：
- 扭矩未达目标值时突然下降或波动
- 曲线尾部呈波浪形
- 上升斜率明显降低
- 扭矩下降但不快速归零

**可能原因**：
- 内螺纹材料硬度不足（如铝合金、塑料）
- 螺纹啮合长度不够
- 螺纹已有损伤、磕碰或交叉螺纹
- 拧紧扭矩超过材料承载能力
- 螺纹加工质量差（脱碳、尺寸偏差、粗糙度不良）

**解决方案**：
- 增加螺纹孔材料硬度（热处理、表面渗碳）
- 增加螺纹啮合长度（至少1.5倍螺栓直径）
- 加强来料检验，避免螺纹损伤
- 重新评估工艺扭矩，降低拧紧扭矩10-20%
- 改善螺纹加工工艺，控制脱碳层深度<0.2mm
- 使用螺纹护套或自攻螺套增强连接

---

##### 2. **粘滑现象/Stick-Slip（中等）**
**特征**：
- 扭矩曲线呈锯齿状波动
- 拧紧过程伴随咔吱声响和异常振动
- 周期性抖动

**可能原因**：
- 拧紧速度过高（>80rpm）
- 螺纹副摩擦系数过大或不稳定
- 表面粗糙度不佳，局部卡滞
- 抗扭臂刚性连接不足，扭转积累释放
- 润滑不良或螺纹胶干燥、粘度过高
- 螺纹配合间隙过小

**解决方案**：
- 降低拧紧转速至40-60rpm
- 使用螺纹胶或润滑剂降低摩擦系数
- 改善螺纹表面粗糙度至Ra1.6-3.2μm
- 提高抗扭臂的刚性连接（增加夹持长度）
- 避免高扭矩启动，采用软启动策略
- 检查螺纹配合等级，适当放宽公差

---

##### 3. **扭矩上升缓慢（中等）**
**特征**：
- 扭矩-角度曲线斜率明显小于正常值
- 弹性段不明显
- 初期扭矩上升较快，某个扭矩值后突然变缓
- 最终扭矩未达到监控下限

**可能原因**：
- 连接件间隙过大，贴合阶段过长
- 螺栓或连接件刚度不足
- 垫片压缩量过大
- 被连接件支撑不足，产生弯曲变形
- 螺纹配合间隙过大
- 焊接对手件有间隙
- 开档结构尺寸不匹配
- 软连接、垫片跟转

**解决方案**：
- 减小连接件装配间隙，增加定位销
- 更换高刚度螺栓或增大螺栓直径
- 优化垫片选型，控制压缩量
- 增加被连接件底部支撑，提高刚性
- 检查螺纹配合等级，改用更紧配合
- 检查连接件匹配性
- 调整开档尺寸
- 改善焊接质量

---

##### 4. **被连接件压溃（严重）**
**特征**：
- 曲线出现双峰或平台现象
- 连接件局部塑性变形或压溃
- 曲线初期下降或平缓
- 随后再次上升，形成两个连续弹性区域

**可能原因**：
- 被连接件材料强度不足（如薄壁塑料、软质铝合金）
- 支撑面积过小，局部应力集中
- 拧紧扭矩过大
- 垫片硬度过高，局部压入
- 被连接件厚度不足

**解决方案**：
- 增加垫圈面积，分散压力
- 使用软质垫片（如尼龙、橡胶）缓冲
- 降低拧紧扭矩15-30%
- 增加被连接件局部厚度或加强筋
- 更换高强度材料或进行局部热处理

---

##### 5. **连接件开裂（严重）**
**特征**：
- 扭矩达到峰值后突然下降
- 伴随清脆断裂声
- 连接件开裂
- 达到峰值后（未到工艺扭矩）开始下降
- 未快速降至零
- 呈现陡峭斜率和周期性波动

**可能原因**：
- 塑料件或脆性材料韧性不足
- 螺纹孔壁厚不足
- 材料存在内应力或裂纹
- 拧紧扭矩严重超标
- 低温环境下材料脆性增大
- 工艺扭矩超过零件破坏扭矩
- 螺纹孔设计不合理、应力集中

**解决方案**：
- 更换韧性更好的材料（如PA66+GF改性塑料）
- 增加螺纹孔壁厚（建议≥2倍螺栓直径）
- 降低拧紧扭矩20-40%
- 增加预埋螺母或金属嵌件
- 进行退火处理消除内应力
- 避免在低温环境（<-10℃）下拧紧
- 合理设计螺纹孔孔径、壁厚
- 避免应力集中

---

##### 6. **浮钉现象/Floating Bolt（严重）**
**特征**：
- 扭矩快速上升后停滞
- 螺钉未有效拧入连接件，处于浮动状态
- 曲线扭矩快速上升或波动缓慢上升
- 与正常拧紧曲线存在显著差异

**可能原因**：
- 螺纹孔深度不足，螺钉触底
- 螺纹孔内有切屑、焊渣等异物
- 盲孔底部存在残留切削液或油污
- 螺纹孔已损坏或螺纹不完整
- 螺钉长度过长

**解决方案**：
- 增加螺纹孔深度（至少螺钉长度+2个螺距）
- 清理螺纹孔内异物，使用压缩空气吹净
- 清理盲孔底部，确保排气通畅
- 修复或重新加工螺纹孔
- 更换合适长度的螺钉
- 增加螺纹孔检验工序

---

##### 7. **螺栓断裂（严重）**
**特征**：
- 扭矩上升至峰值后急剧下降至零
- 螺栓杆部或螺纹根部断裂
- 曲线突然中断
- 扭矩归零

**可能原因**：
- 螺栓强度等级不足
- 螺栓存在材料缺陷（夹杂、裂纹）
- 拧紧扭矩严重超过螺栓屈服极限
- 螺纹根部应力集中
- 螺栓疲劳损伤（重复使用）
- 螺栓过屈服
- 设备下压力度过大

**解决方案**：
- 更换高强度等级螺栓（如10.9级或12.9级）
- 加强螺栓进货检验，进行无损检测
- 重新计算拧紧扭矩，严格控制上限
- 优化螺纹根部圆角半径，降低应力集中
- 避免重复使用螺栓，特别是高强度螺栓
- 检查拧紧设备精度，避免过载
- 禁止重复使用已拧紧螺栓
- 验证螺栓等级（8.8、10.9、12.9）
- 调整设备压力

### 4️⃣ **参数对比表（增强版）**

| 参数 | 说明 | 单位 |
|------|------|------|
| 峰值扭矩 | 曲线达到的最大扭矩值 | N·m |
| 峰值角度 | 达到峰值扭矩时的角度 | 度(°) |
| 平均斜率 | 线性段的扭矩上升速率 | - |
| **控制策略** | 扭矩控制 / 转角控制 | - |
| 弹性区间 | 弹性变形的角度范围 | 度(°) |
| 塑性区间 | 塑性变形的角度范围 | 度(°) |
| 偏差率 | 与标准曲线的偏差百分比 | % |
| **质量评级** | 优秀 / 良好 / 需改进 | - |

**控制策略识别**：
- 🔵 **转角控制**：角度偏差 < 20°，适用于屈服点拧紧
- 🟢 **扭矩控制**：角度可能波动，扭矩相对稳定

**质量评级标准**：
- 🟢 优秀：偏差率 < 5%
- 🔵 良好：偏差率 5-10%
- 🟡 需改进：偏差率 > 10%

### 5️⃣ **智能材质分析（增强版）**

系统根据曲线形态特征（斜率、峰值扭矩、塑性段长度），智能识别连接件材质。

#### 钢质连接件
- **识别条件**：平均斜率 > 0.1 且峰值扭矩 > 20 N·m
- **曲线特征**：高强度、高刚度、线性弹性段明显、塑性变形区域较长、抗疲劳性能好
- **推荐扭矩**：25-35 N·m（M8-M10）
- **推荐转速**：40-60 rpm
- **控制策略**：屈服点拧紧控制（Yield Point Control）
- **注意事项**：防止过拧导致螺栓延伸超限，确保预紧力稳定

#### 铝合金连接件
- **识别条件**：平均斜率 0.05-0.1 且峰值扭矩 10-20 N·m
- **曲线特征**：中等强度、较软、易产生压痕、热膨胀系数大、螺纹易损伤
- **推荐扭矩**：10-18 N·m（M8-M10）
- **推荐转速**：30-50 rpm
- **控制策略**：扭矩控制
- **注意事项**：铝合金螺纹易滑牙，建议降低拧紧扭矩20-30%，使用螺纹护套增强连接，注意温度变化导致的预紧力松弛

#### 塑料连接件
- **识别条件**：平均斜率 < 0.05 或峰值扭矩 < 10 N·m
- **曲线特征**：低刚度、非线性明显、蠕变性大、对温度敏感、易产生应力开裂
- **推荐扭矩**：3-8 N·m（M6-M8）
- **推荐转速**：20-40 rpm
- **控制策略**：转角控制 + 扭矩监控
- **注意事项**：严格控制扭矩上限避免开裂，监控扭矩上升斜率，使用自攻螺钉或预埋螺母可提高连接可靠性，注意蠕变导致的预紧力衰减

#### 材质特性对比参考表

| 材质类型 | 典型斜率 | 扭矩范围 | 推荐转速 | 主要风险 |
|---------|---------|---------|---------|---------|
| 钢质 | >0.10 | 25-35 N·m | 40-60 rpm | 过拧导致螺栓延伸 |
| 铝合金 | 0.05-0.10 | 10-18 N·m | 30-50 rpm | 螺纹滑牙、压痕 |
| 塑料 | <0.05 | 3-8 N·m | 20-40 rpm | 开裂、蠕变松弛 |
| 铸铁 | 0.08-0.12 | 20-30 N·m | 30-50 rpm | 脆性断裂 |
| 复合材料 | 0.03-0.08 | 5-15 N·m | 25-45 rpm | 分层、纤维断裂 |

### 6️⃣ **批次一致性分析（新增）**

#### 功能说明
对比同一工位不同批次的拧紧曲线，评估工艺稳定性和一致性。通过统计分析方法，计算曲线参数的标准差和变异系数，量化工艺波动情况。

#### 分析指标

| 指标 | 说明 | 单位 |
|------|------|------|
| 扭矩标准差 | 所有曲线峰值扭矩的标准差 | N·m |
| 角度标准差 | 所有曲线峰值角度的标准差 | 度(°) |
| 稳定性评分 | 综合评估工艺稳定性（0-100分） | 分 |

#### 评估标准

**工艺稳定（评分≥90）**
- 所有批次的拧紧曲线高度一致
- 工艺参数稳定，质量控制良好
- 扭矩和角度的标准差均在合理范围内
- 建议：继续保持当前工艺

**基本稳定（评分70-90）**
- 大部分批次的拧紧曲线较为一致，但存在一定波动
- 建议措施：
  - 定期校准拧紧设备，检查扭矩传感器精度
  - 加强原材料进货检验，关注材料批次差异
  - 统一操作规范，减少人为因素影响

**波动较大（评分<70）**
- 不同批次的拧紧曲线差异明显，工艺稳定性不足
- 需要立即排查原因
- 紧急措施：
  - 紧急检查拧紧设备状态，进行全面维护保养
  - 暂停生产，对异常批次进行全检
  - 排查原材料供应商变更或材料批次问题
  - 重新培训操作人员，统一拧紧标准
  - 增加过程监控点，实时跟踪曲线变化

#### 典型应用
- **新产品验证**：评估新工艺的可重复性
- **量产监控**：每小时/每班次监控工艺稳定性
- **设备维护**：设备保养前后对比曲线一致性
- **供应商审核**：评估不同供应商的材料一致性

## 📁 CSV文件格式

### 文件格式要求

```csv
角度（度）,扭矩（N·m）
0,0
30,5
60,10
90,15
120,20
150,25
...
```

### 格式说明
- **编码**：UTF-8
- **分隔符**：逗号(,)
- **第一行**：标题行（可选，会自动跳过）
- **数据行**：角度值,扭矩值
- **数据类型**：数值型，支持小数

### 数据要求
- 角度范围：0-3600°（支持多圈拧紧）
- 扭矩范围：0-500 N·m
- 数据点数：建议100-1000点
- 采样频率：建议1°/点或更密集

## 🎨 使用流程

### 步骤1：导入标准曲线
1. 点击右上角"导入曲线"按钮
2. 选择"导入标准曲线"标签
3. 拖拽或选择CSV文件
4. 点击"确认导入"

### 步骤2：批量导入采集曲线
1. 切换到"批量导入采集曲线"标签
2. 一次性拖拽多个CSV文件（最多100个）
3. 点击"确认导入"
4. 系统自动解析并添加到曲线列表

### 步骤3：选择对比曲线
- 在左侧曲线列表中勾选要对比的曲线
- 取消勾选不需要对比的曲线
- 图表实时更新显示

### 步骤4：查看分析结果
- **综合评估**：查看合格率和整体状态
- **问题诊断**：查看详细的异常分析（7种异常类型全覆盖）
- **参数对比**：对比各曲线的关键参数（包含控制策略识别）
- **材质分析**：查看智能材质识别结果和工艺建议
- **批次对比**：评估工艺稳定性和一致性（新增）

### 步骤5：导出报告
- 点击"导出报告"按钮
- 自动生成TXT格式分析报告
- 包含完整的诊断结果和建议措施

## 🎯 典型应用场景（增强版）

### 场景1：新产品工艺验证
**目标**：验证新产品的拧紧工艺是否合理

**操作**：
1. 导入理论标准曲线
2. 进行首批10-20件试装
3. 导入10-20条采集曲线
4. 对比分析，查看批次一致性评分
5. 根据材质识别结果调整工艺参数

**成功标准**：
- 合格率 ≥ 95%
- 批次稳定性评分 ≥ 85
- 无严重异常（滑牙、断裂、开裂）

---

### 场景2：量产过程监控
**目标**：监控量产过程中的拧紧质量

**操作**：
1. 建立标准曲线库
2. 每小时采集5条曲线
3. 批量导入进行对比
4. 监控批次一致性评分趋势
5. 识别异常曲线并追溯

**预警机制**：
- 批次稳定性评分 < 70：立即排查
- 出现严重异常：停线检查

## 📞 技术支持

### 常见问题

**Q1：为什么导入的曲线无法显示？**
A：检查CSV文件格式是否正确，确保有角度和扭矩两列数据。

**Q2：分析结果不准确怎么办？**
A：确保标准曲线和采集曲线在相同工况下，检查数据质量。

**Q3：可以导入其他格式的文件吗？**
A：目前仅支持CSV格式，可使用Excel转换为CSV。

**Q4：100条曲线限制如何突破？**
A：可分批次分析，或联系我们升级到企业版。

**Q5：批次稳定性评分如何计算？**
A：基于扭矩和角度的变异系数，综合评估工艺波动情况。评分越高表示工艺越稳定。

**Q6：材质识别不准确怎么办？**
A：材质识别基于曲线特征统计，建议至少导入10条以上同材质曲线以提高准确性。特殊材质可人工标注。

**Q7：如何选择扭矩控制还是转角控制？**
A：系统会根据曲线特征自动识别。一般建议：钢质连接件用转角控制（屈服点法），塑料件用扭矩控制+角度监控。

## 📎 附录

### A. 专业术语解释

- **屈服点拧紧**：螺栓达到屈服点后的拧紧控制方法，预紧力更稳定
- **变异系数(CV)**：标准差与平均值的比值，衡量数据离散程度
- **粘滑效应**：摩擦力在静摩擦和动摩擦之间切换导致的振动
- **蠕变**：材料在恒定应力下产生的缓慢塑性变形
- **螺纹啮合长度**：螺栓与螺纹孔有效接触的长度

- GB/T 16823.3-2010《螺纹紧固件拧紧试验方法》
