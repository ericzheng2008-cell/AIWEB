#### 访问方式
```
URL: http://localhost:3000/admin/cms
需要登录: 是
```

#### 功能模块

##### 1️⃣ Logo和基本设置
**功能**:
- 上传/更新网站Logo
- 设置公司名称（中英文）
- 设置公司标语（中英文）

**使用步骤**:
1. 登录后台管理系统
2. 进入"CMS管理" → "Logo和基本设置"
3. 输入Logo图片URL
4. 点击"更新Logo"按钮
5. 填写公司名称和标语
6. 点击"保存配置"

**示例**:
```
Logo URL: https://example.com/logo.png
中文名称: 明升伟业
英文名称: Mingsheng
中文标语: 专业工业工具供应商
英文标语: Professional Industrial Tools Supplier
```

##### 2️⃣ 产品系列管理
**功能**:
- 查看所有产品系列
- 添加新产品系列
- 编辑现有产品系列
- 删除产品系列

**字段说明**:
- **中文名称**: 产品系列中文名称
- **英文名称**: 产品系列英文名称
- **中文描述**: 产品系列中文描述（支持多行）
- **英文描述**: 产品系列英文描述（支持多行）
- **图片URL**: 产品系列展示图片

**操作步骤**:

**添加产品系列**:
1. 点击"添加产品系列"按钮
2. 填写完整信息（中英文）
3. 输入图片URL
4. 点击"保存"

**编辑产品系列**:
1. 在列表中找到要编辑的产品
2. 点击"编辑"按钮
3. 修改信息
4. 点击"保存"

**删除产品系列**:
1. 在列表中找到要删除的产品
2. 点击"删除"按钮
3. 确认删除

##### 3️⃣ 事业部管理
**功能**:
- 查看所有事业部
- 添加新事业部
- 编辑现有事业部
- 删除事业部

**字段说明**:
- **Key**: 事业部英文标识（如：assembly, production）
- **中文名称**: 事业部中文名称
- **英文名称**: 事业部英文名称
- **中文描述**: 事业部中文描述
- **英文描述**: 事业部英文描述

**操作步骤**:
与产品系列管理类似，支持添加、编辑、删除操作。

## 💾 数据存储

### 存储方式
所有CMS数据使用 `LocalStorage` 存储，确保数据持久化：

```javascript
// 存储的数据项
localStorage.setItem('siteLogo', logoUrl)           // Logo URL
localStorage.setItem('siteConfig', configJSON)      // 网站配置
localStorage.setItem('productSeries', seriesJSON)   // 产品系列
localStorage.setItem('divisions', divisionsJSON)    // 事业部
```

### 数据备份
建议定期备份localStorage数据：
```javascript
// 在浏览器控制台执行
console.log(localStorage.getItem('productSeries'))
console.log(localStorage.getItem('divisions'))
```

### 数据迁移
如需迁移到新环境，可以：
1. 导出当前数据（JSON格式）
2. 在新环境导入数据
3. 或直接复制localStorage内容

## 📝 注意事项

### 1. Logo图片建议
- **格式**: PNG（支持透明）或SVG
- **尺寸**: 高度60px，宽度200px以内
- **大小**: 建议小于100KB
- **托管**: 使用稳定的图片CDN

### 2. 内容编辑建议
- 产品名称：简洁明了，不超过20字
- 产品描述：详细但不冗长，50-100字
- 英文翻译：准确专业，符合行业术语
- 图片URL：确保长期有效，避免失效

### 3. 数据安全
- 定期备份localStorage数据
- 重要修改前先记录原始数据
- 测试环境先验证，再发布生产

### 4. 性能优化
- 图片使用CDN加速
- 图片压缩优化大小
- 避免过多产品系列（建议10个以内）

## 🐛 常见问题

### Q1: Logo不显示？
**A**: 检查以下几点：
1. Logo URL是否正确
2. 图片是否可访问
3. 浏览器是否阻止了外部图片
4. 清除浏览器缓存重试

### Q2: 修改后前台不更新？
**A**: 
1. 刷新浏览器页面（Ctrl+F5强制刷新）
2. 检查是否点击了"保存"按钮
3. 查看浏览器控制台是否有错误

### Q3: 英文翻译不生效？
**A**: 
1. 确保中英文都已填写
2. 切换到英文模式测试
3. 检查语言包是否正确加载

### Q4: 删除后能恢复吗？
**A**: 
- 删除操作不可逆
- 建议删除前先备份数据
- 重要内容建议先禁用而非删除
