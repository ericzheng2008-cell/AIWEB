# 🎨 聊天机器人样式检查清单

## 问题诊断

如果看不到新外观，请按以下步骤操作：

### ✅ 步骤1：确认代码已更新

检查文件 `src/components/AiChat.vue` 是否包含以下样式：

```css
/* 应该是紫色渐变，不是蓝色 */
.chat-button {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  /* 旧版是：#1890ff 0%, #0066cc 100% */
}
```

### ✅ 步骤2：清除浏览器缓存

**方法1：快捷键强制刷新**
- Windows/Linux: `Ctrl + F5` 或 `Ctrl + Shift + R`
- Mac: `Cmd + Shift + R`

**方法2：手动清除缓存**
1. 按 `F12` 打开开发者工具
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

**方法3：完全清除浏览器缓存**
1. 按 `Ctrl + Shift + Delete`
2. 选择"缓存的图像和文件"
3. 时间范围选"全部"
4. 点击"清除数据"

### ✅ 步骤3：清除Vite缓存并重启

**运行清理脚本：**
```bash
🔧_强制清除缓存并重启.bat
```

**或手动执行：**
```bash
# 停止服务器（Ctrl+C）
# 删除Vite缓存
rmdir /s /q node_modules\.vite
# 重启
npm run dev
```

## 🔍 如何在开发者工具中检查

1. 按 `F12` 打开开发者工具
2. 点击 "Elements" 标签
3. 找到 `.chat-button` 元素
4. 查看 Styles 面板中的 `background` 属性
5. 应该显示：`linear-gradient(135deg, rgb(102, 126, 234) 0%, rgb(118, 75, 162) 100%)`

如果不是紫色，说明浏览器使用了旧缓存。

## 🚨 常见问题

### Q1: 代码已更新，但浏览器还是旧样式？
**A**: 浏览器缓存问题，请按 `Ctrl+F5` 强制刷新

### Q2: 强制刷新后还是旧样式？
**A**: Vite缓存问题，运行 `🔧_强制清除缓存并重启.bat`

### Q3: 重启后部分样式新，部分旧？
**A**: CSS加载问题，完全清除浏览器缓存

### Q4: 本地看到新样式，但部署后是旧的？
**A**: 需要重新构建：`npm run build`
