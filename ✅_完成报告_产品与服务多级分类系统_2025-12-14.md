## 📊 系统架构

### 目录结构
```
src/
├── store/
│   └── productsServices.js         # 产品服务数据管理
├── views/
│   ├── ProductsServices.vue        # 前台展示页面
│   └── admin/
│       └── ProductsServicesManage.vue  # 后台管理页面
└── styles/
    └── pantone-2736c.css           # PANTONE 2736C配色
```

### 数据流转
```
用户输入 → Store → LocalStorage → 持久化
       ↓
   前台展示 ← Store ← LocalStorage
```

  - 角向电动拧紧工具

- **图标**：💨 Wind
- **默认二级分类**：
  - 气动拧紧工具
  - 气动打磨工具

### 3. 手动工具 (Manual Tools)
- **图标**：🔧 Tools
- **默认二级分类**：
  - 扭力扳手
  - 套筒工具

### 4. 测量工具 (Measurement Tools)
- **图标**：📊 DataAnalysis
- **默认二级分类**：
  - 扭力测试仪
  - 传感器

### 5. 自动化系统 (Automated Systems)
- **图标**：🖥️ Monitor
- **默认二级分类**：
  - 自动拧紧系统
  - 装配工作站

- **图标**：💡 Management
- **默认二级分类**：
