### 2️⃣ LOGO上传功能

**支持格式**: PNG、JPG、SVG  
**文件大小**: < 500KB  
**推荐尺寸**: 200x60px  
**特色功能**:
- ✅ 拖拽上传
- ✅ 实时预览
- ✅ 删除功能
- ✅ Base64存储（无需服务器）

**上传提示**:
```
- 支持 PNG、JPG、SVG
- 建议尺寸 200x60px
- 大小 < 500KB
- 建议使用透明背景
```

**使用说明**:
- LOGO将显示在网站顶部导航栏左侧
- 建议上传透明背景的PNG格式图片
- 推荐尺寸为 200x60px，高度不超过60px
- 如果不上传LOGO，将显示公司名称文字

## 📝 使用指南

### 管理员操作步骤

#### 1. 上传LOGO
```
1. 登录后台管理系统
2. 进入"内容管理"
3. 点击"站点配置"标签页
4. 在LOGO上传区拖拽或点击上传图片
5. 查看实时预览效果
6. 点击"保存配置"按钮
```

#### 2. 编辑公司名称
```
1. 切换到"🇨🇳 中文"标签页
2. 填写"公司名称（中文）"
3. 填写"公司口号（中文）"（可选）
4. 切换到"🇺🇸 English"标签页
5. 填写"Company Name"
6. 填写"Company Slogan"（可选）
7. 查看实时预览效果
8. 点击"保存配置"按钮
```

#### 3. 预览效果
```
方法1: 点击"预览效果"按钮，打开新标签页查看前台效果
方法2: 保存后，刷新前台页面查看实际效果
```

## 📞 注意事项

### ⚠️ 重要提示
1. **LOGO大小**: 必须 < 500KB，建议使用压缩工具优化
2. **LOGO尺寸**: 建议 200x60px，高度不超过60px
3. **透明背景**: 建议使用PNG透明背景，适配性更好
4. **必填项**: 中英文公司名称为必填项
5. **保存操作**: 修改后必须点击"保存配置"按钮

### 💡 最佳实践
1. 使用SVG格式LOGO（矢量图，无损缩放）
2. 准备多语言版本的公司名称
3. 公司口号简洁有力，不超过20字
4. 定期备份LOGO文件
5. 保存前先预览效果
