# 📘 外部链接管理使用指南

**功能**: 产品技术销售小课堂 - 外部链接管理  
**版本**: v1.0.0  
**日期**: 2025-12-15

---

## 🎯 功能简介

外部链接管理功能允许您在小课堂中添加和管理外部学习资源链接（如节卡学院、官方文档等），这些链接会自动展示在前台小课堂和AI聊天机器人的推荐中。

---

## 🚀 快速开始

### Step 1: 进入后台管理

1. 访问后台管理页面：`http://localhost:8080/admin/classroom-manage`
2. 点击顶部的**"外部链接管理"**标签页

![外部链接管理入口](placeholder)

---

### Step 2: 添加第一个外部链接

#### 以"节卡学院"为例：

1. 点击**"添加外部链接"**按钮

2. 填写表单：

   **基本信息**：
   - **一级分类**: 选择"协作机器人" 🤖
   - **二级分类**: 选择"协作机器人基础"（可选）
   - **链接标题**: `节卡学院 - 协作机器人技术学习平台`
   - **链接地址**: `https://www.jaka.com/zh/jakaAcademy`

   **详细描述**：
   - **链接描述**: 
     ```
     节卡机器人官方学习平台，提供完整的协作机器人技术课程、
     编程教程和应用案例，适合从入门到进阶的各阶段学习者
     ```

   **分类与属性**：
   - **链接类型**: 选择"🎓 学院平台"
   - **语言**: 选择"中文"
   - **图标**: 输入 `🎓`

   **其他设置**：
   - **需要会员**: 关闭（免费资源）
   - **排序**: `1`（数字越小越靠前）
   - **标签**: 添加 `官方课程`, `协作机器人`, `编程教程`

3. 点击**"保存"**按钮

✅ 成功！您已添加第一个外部链接

## 🎨 链接类型说明

### 四种链接类型：

#### 1. 🎓 学院平台
- **适用**: 官方学习平台、在线课程网站
- **示例**: 节卡学院、ABB在线学院
- **标签颜色**: 默认（蓝色）

#### 2. 📖 技术文档
- **适用**: API文档、技术手册、白皮书
- **示例**: PLC编程手册、通信协议文档
- **标签颜色**: 绿色

#### 3. 🎬 视频教程
- **适用**: YouTube教程、B站视频、录播课程
- **示例**: 拧紧工艺实操视频、设备维护教学
- **标签颜色**: 橙色

#### 4. 🔧 在线工具
- **适用**: 在线计算器、仿真工具、测试平台
- **示例**: 扭矩计算器、PLC仿真器
- **标签颜色**: 蓝色（info）

## 🔄 状态管理

### 激活 vs 停用

每个外部链接都有**状态开关**：
- **激活** (Active): 链接在前台可见
- **停用** (Inactive): 链接不显示，但数据保留

**使用场景**：
- 临时下线某个资源 → 停用
- 资源网站维护中 → 停用
- 测试新链接 → 先停用，确认后激活

**操作方法**：
直接点击列表中的开关按钮即可切换状态

### 展示位置

## 🎯 最佳实践

### 1. 选择优质资源

✅ **推荐添加**：
- 官方学习平台（如节卡学院、ABB学院）
- 权威技术文档
- 高质量视频教程
- 实用在线工具

❌ **不建议添加**：
- 广告性质链接
- 不安全的网站
- 内容质量低的资源
- 已失效的链接

### 2. 完善链接信息

✅ **标题要求**：
- 简洁明了，15-30字
- 包含关键词
- 说明资源特点

✅ **描述要求**：
- 50-200字
- 说明资源内容
- 强调资源优势
- 注明适用人群

✅ **标签要求**：
- 2-5个标签
- 关键词准确
- 便于搜索

### 3. 合理分类

#### 一级分类原则
选择最相关的技术领域

#### 二级分类原则
- 如果资源通用 → 不选二级分类
- 如果资源针对性强 → 选择具体二级分类

**示例**：
```
节卡学院（通用协作机器人平台）
→ 一级分类: 协作机器人
→ 二级分类: 不选（通用）

协作机器人安全规范文档
→ 一级分类: 协作机器人
→ 二级分类: 安全规范（针对性强）
```

### 4. 排序优化

建议排序规则：
1. 官方资源（1-10）
2. 优质免费资源（11-50）
3. 付费资源（51-100）
4. 其他资源（101+）

## ❓ 常见问题

### Q1: 为什么我的链接不显示在前台？

**可能原因**：
1. ✅ 链接状态是"停用" → 改为"激活"
2. ✅ 没有选择一级分类 → 必须选择
3. ✅ 链接地址格式错误 → 检查URL是否完整

### Q2: 如何让链接排在最前面？

**解决方法**：
1. 编辑链接
2. 将"排序"字段设置为 `1`
3. 保存

### Q3: AI为什么不推荐我的链接？

**可能原因**：
1. ✅ 链接状态是"停用" → 只推荐激活的链接
2. ✅ 链接类型不是"学院平台" → AI优先推荐学院类型
3. ✅ 用户问题与链接分类不匹配 → 检查分类设置

### Q4: 如何批量导入外部链接？

**当前版本**：
暂不支持批量导入，需逐个添加

**未来计划**：
v2.0 版本将支持 CSV/Excel 批量导入

### Q5: 能否限制某些链接只对特定用户可见？

**当前版本**：
所有激活的链接对所有用户可见

**未来计划**：
v2.0 版本将支持权限控制和分组管理

## 🔧 故障排除

### 问题1: 保存后链接丢失

**检查**：
1. 浏览器控制台是否有错误
2. LocalStorage 是否被清理
3. 浏览器是否支持 LocalStorage

**解决**：
刷新页面后重新添加

### 问题2: 点击链接无反应

**检查**：
1. 链接地址是否完整（包含 `https://`）
2. 浏览器是否阻止弹出窗口
3. 目标网站是否可访问

**解决**：
1. 确保URL格式正确
2. 允许浏览器弹出窗口
3. 测试目标网站可用性

### 问题3: AI不推荐外部链接

**检查**：
1. 链接状态是否"激活"
2. 关键词是否匹配
3. 链接类型是否合适

**解决**：
1. 激活链接
2. 调整关键词和分类
3. 优先使用"学院平台"类型
