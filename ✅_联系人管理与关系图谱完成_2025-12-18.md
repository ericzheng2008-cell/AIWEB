- **电话**：联系电话

  - ✅ 设备管理
  - ✅ 仓库收货
  - ✅ 产品验收
  - ✅ 质检

  - ✅ 采购报价
  - ✅ 询价
  - ✅ 签合同
  - ✅ 收发票

#### 画像分析
- **兴趣偏好**：技术创新/性价比/品牌信誉等6个选项
- **沟通偏好**：自由文本描述
- **当前态度**：非常积极/积极/中立/消极/非常消极

## 🎯 使用场景

### 场景1：销售团队拜访不同联系人

**问题**：同一客户有多个部门，销售、技术支持、售后分别对接不同的人

**解决方案**：
```bash
1. 打开商机详情
2. 在任意阶段点击"添加事件"
3. 选择事件类型："新增联系人"
4. 填写联系人信息：
   - 姓名：王工
   - 职位：技术主管
   - 部门：技术部
   - 对接事务：技术沟通、产品验收、质检
   - 决策角色：影响者
5. 保存后自动添加到客户联系人库
6. 该联系人立即出现在"联系人关系图谱"中
```

### 场景2：查看组织架构和决策链

**需求**：了解客户内部的决策流程和关键角色

**操作步骤**：
```bash
1. 在商机详情Header点击"联系人关系图谱(4)"
2. 查看三层组织架构：
   - 高层：张总（决策者）- 最终拍板
   - 中层：李经理（推动者）- 内部推动采购
   - 中层：王工（影响者）- 技术建议
   - 基层：刘主任（使用者）- 实际使用工具
3. 点击任一联系人卡片查看详情
4. 查看互动历史、兴趣偏好
5. 点击"查看客户360°"跳转完整画像
```

### 场景3：跟踪不同联系人的互动

**需求**：记录与每个联系人的沟通历史

**操作步骤**：
```bash
1. 添加事件时选择"关联联系人"
2. 从下拉列表选择具体联系人
3. 事件自动记录到该联系人的互动历史
4. 在联系人详情中可查看所有相关事件
```
