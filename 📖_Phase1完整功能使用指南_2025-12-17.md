## 📝 功能3：国际化询盘表单

| **公司** | ✅ 必填 | 公司名称 |
| **电话** | ✅ 必填 | 联系电话（含国际区号） |

### 填写示例

#### 中文示例
```
姓名：张明
邮箱：zhang.ming@example.com
公司：上海汽车制造有限公司
电话：+86-21-12345678
国家：🇨🇳 中国

```

Product: Torque Wrench
Quantity: 100 units

         Torque range 60-200Nm with data collection capability.

### 提交流程
```
1. 填写所有必填字段
2. 点击"提交询盘"按钮
3. 系统验证表单（实时显示错误）
4. 提交成功后显示确认消息
5. 表单自动重置
6. 24小时内收到报价
```

### 表单验证规则
```
✅ 姓名：不能为空
✅ 邮箱：必须是有效的邮箱格式（如：xxx@example.com）
✅ 公司：不能为空
✅ 电话：不能为空
✅ 国家：必须选择一个国家
✅ 产品：不能为空
✅ 详细需求：不能为空，最多500字
```

## 💡 常见问题

### Q1: 切换语言后部分内容仍是中文？
**A:** 这是正常的，部分动态内容（如产品名称）可能保留原始语言。刷新页面可解决。

### Q2: AI助手识别不了国际买家？
**A:** 请确保使用英文关键词，如"international"、"export"、"overseas"等。

### Q3: 询盘表单提交后无反应？
**A:** 当前为模拟提交，实际部署后会连接真实API。测试环境会显示成功消息。

### Q4: 移动端按钮太小点不到？
**A:** 已优化移动端按钮大小和间距，如仍有问题，请反馈具体机型。

### Q5: 如何添加更多语言？
**A:** 在`src/i18n/locales/`目录下创建新的语言文件，并在`src/i18n/index.js`中注册。
