### 2. 年包智能推荐（3分钟锁定方案）

#### 功能位置
- 访问：`http://localhost:5173/intelligent-calculator` → Tab 2
- 代码：`src/store/equipmentLifecycle.js` → `recommendAnnualPackage()`

#### 3个标准SKU
```
A. 基础保障包（AP-BASIC）
   • 费率：10%
   • 适合：预算紧张、工具量大
   • ROI：75-125%
   • 服务：检测+维修+48h响应

B. 标准运营包（AP-STANDARD）⭐推荐
   • 费率：12%
   • 适合：主线产线、追求稳产
   • ROI：125-167%
   • 服务：基础包 + 备机 + 月度巡检
   • 续签率：90%

C. 战略托管包（AP-STRATEGIC）
   • 费率：15%
   • 适合：关键产线、集团客户
   • ROI：139-278%
   • 服务：标准包 + 全托管 + TCO对赌
```

#### 智能推荐逻辑
```javascript
评分规则（示例）：
  if (预算紧张) → 基础包 +30分
  if (产线重要性 >= 8) → 战略包 +30分
  if (产线重要性 5-7) → 标准包 +25分
  if (痛点包含"频繁停线") → 标准包/战略包 +20分

自动排序，推荐最高分方案
生成推荐理由，提升专业性
```

```

    • 回本：5-8个月
    • 适合：预算紧、快速见效

L2. 原厂翻新
    • 成本：10-20万
    • ROI：150%
    • 回本：6-9个月
    • 适合：恢复出厂性能

L3. 部件替换
    • 成本：20-40万
    • ROI：167%
    • 回本：4.5-7个月
    • 适合：故障频繁、消除隐患

L4. 年包服务
    • 成本：年费12-18%
    • ROI：125-167%
    • 回本：持续
    • 适合：追求确定性、风险转移

    • 成本：100万+
    • ROI：150-200%
    • 回本：6-8个月
```
