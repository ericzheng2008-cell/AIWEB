# 📖 首页LOGO和公司名称编辑功能使用指南

> **版本**: v2.6.4  
> **日期**: 2025-12-21  
> **目标用户**: 网站管理员

---

## 🎯 功能概述

本功能允许管理员通过后台管理系统编辑网站顶部导航栏的LOGO和公司信息，支持多语言配置和实时预览。

---

## 📍 访问路径

### 后台入口
```
URL: http://localhost:5173/admin
导航: 左侧菜单 → 内容管理 → 站点配置（第一个标签页）
```

---

## 🖼️ 功能模块详解

### 1️⃣ 网站LOGO上传

#### 上传方式
- **方式A**: 拖拽图片文件到上传区域
- **方式B**: 点击上传区域，选择本地图片文件

#### 格式要求
| 项目 | 要求 |
|------|------|
| **支持格式** | PNG、JPG、JPEG、SVG |
| **文件大小** | < 500KB |
| **推荐尺寸** | 200x60px |
| **推荐格式** | PNG透明背景 |

#### 操作步骤
```
步骤1: 准备符合要求的LOGO图片
步骤2: 拖拽或点击上传
步骤3: 系统自动验证格式和大小
步骤4: 上传成功后显示预览
步骤5: 点击"保存配置"按钮保存
```

#### 删除LOGO
```
步骤1: 点击LOGO下方的"删除LOGO"按钮
步骤2: LOGO消失，自动显示公司名称文字
步骤3: 点击"保存配置"按钮保存
```

#### 💡 使用建议
- ✅ **最佳格式**: PNG透明背景
- ✅ **最佳尺寸**: 200x60px（高度不超过60px）
- ✅ **文件优化**: 使用TinyPNG等工具压缩图片
- ✅ **设计建议**: 简洁清晰，适配白色背景

### 2️⃣ 公司名称编辑

#### 中文版本
```
标签页: 🇨🇳 中文

字段1: 公司名称（中文）
- 必填项 ✅
- 字数限制: 50字
- 示例: 明升伟业工业设备有限公司

字段2: 公司口号（中文）
- 可选项
- 字数限制: 100字
- 示例: 智能制造 · 精益求精
```

#### 英文版本
```
标签页: 🇺🇸 English

字段1: Company Name
- 必填项 ✅
- 字数限制: 50字符
- 示例: Mingsheng Industrial Equipment Co., Ltd.

字段2: Company Slogan
- 可选项
- 字数限制: 100字符
- 示例: Smart Manufacturing · Excellence Pursuit
```

#### 操作步骤
```
步骤1: 点击"🇨🇳 中文"标签页
步骤2: 填写"公司名称（中文）"（必填）
步骤3: 填写"公司口号（中文）"（可选）
步骤4: 点击"🇺🇸 English"标签页
步骤5: 填写"Company Name"（必填）
步骤6: 填写"Company Slogan"（可选）
步骤7: 查看实时预览效果
步骤8: 点击"保存配置"按钮
```

#### 💡 填写建议
- ✅ **公司名称**: 使用完整的公司全称
- ✅ **公司口号**: 简洁有力，突出核心价值
- ✅ **英文翻译**: 确保准确专业
- ✅ **品牌一致**: 与公司VI系统保持一致

## ⚠️ 常见问题

### Q1: 上传LOGO提示"格式不支持"
**原因**: 文件格式不是PNG/JPG/SVG  
**解决**:
1. 检查文件扩展名（.png、.jpg、.svg）
2. 使用图片编辑软件另存为PNG格式
3. 确保文件没有损坏

### Q2: 上传LOGO提示"文件过大"
**原因**: 文件大小超过500KB  
**解决**:
1. 使用TinyPNG压缩图片（https://tinypng.com/）
2. 降低图片分辨率
3. 使用SVG格式（文件更小）

### Q3: LOGO上传成功但前台不显示
**原因**: 未点击"保存配置"按钮  
**解决**:
1. 点击页面右上角"保存配置"按钮
2. 等待提示"站点配置保存成功！"
3. 刷新前台页面查看效果

### Q4: LOGO显示模糊
**原因**: 图片分辨率太低  
**解决**:
1. 使用至少200x60px的高清图片
2. 对于高分辨率屏幕，使用400x120px
3. 优先使用SVG矢量格式（无损缩放）

### Q5: 保存提示"请填写公司名称"
**原因**: 中文或英文公司名称未填写  
**解决**:
1. 检查"🇨🇳 中文"标签页，填写公司名称（中文）
2. 检查"🇺🇸 English"标签页，填写Company Name
3. 确保两者都已填写

### Q6: 公司口号不显示
**原因**: Header组件需要更新  
**检查**:
1. 公司口号是可选项，可以留空
2. 如果填写了但不显示，检查Header.vue组件
3. 确认slogan字段在Header中有正确绑定

## 💾 备份与恢复

### 数据备份
```
位置: localStorage
键名: cmsAdvanced

备份方法:
1. 打开浏览器开发者工具（F12）
2. 进入Application → LocalStorage
3. 复制cmsAdvanced的值
4. 保存到文本文件
```

### 数据恢复
```
恢复方法:

2. 进入Application → LocalStorage
```
