## 🎯 功能概述

在CMS中新增首页专属板块编辑功能，支持可视化编辑以下三个板块：
1. **关于明升伟业** - 公司简介、资质认证展示
2. **我们的优势** - 企业优势项展示（图标+标题+描述）
3. **服务网络** - 办事处网点展示

所有内容支持**中英文双语**编辑，支持**图片上传**和**排版调整**。

---

## 📊 功能详解

### 1️⃣ 关于明升伟业板块

#### 可编辑内容：
- ✅ **板块标题**（中英文）
- ✅ **公司简介**（中英文，多行文本）
- ✅ **配图**（点击上传或输入URL）
- ✅ **资质认证列表**（可添加/删除多个认证项）
  - 每个认证项包含中英文名称
- ✅ **按钮文字**（中英文，如"了解更多"）

#### 编辑界面：
```

├─ 中文标题：关于明升伟业

└─ 配图：[图片上传组件]

├─ [添加认证] 按钮
└─ 认证列表
    ├─ ISO9000认证 | ISO9000 Certified [删除]
    ├─ QS9000认证 | QS9000 Certified [删除]
    └─ IATF16949认证 | IATF16949 Certified [删除]

├─ 中文按钮文字：了解更多
└─ 英文按钮文字：Learn More

[保存设置]
```

## 💡 使用技巧

### 1. **快速开始**
1. 进入CMS管理：`http://localhost:3004/admin/cms`
2. 点击"首页内容"标签
3. 在"首页专属板块"卡片中选择要编辑的板块

### 2. **编辑建议**

#### 关于明升伟业
- 公司简介控制在50-80字为佳
- 认证项建议不超过5个
- 配图建议尺寸：800x600px

#### 我们的优势
- 优势项建议4-6个（4个显示最佳）
- 每项描述控制在30字以内
- 图标选择与内容相匹配的

#### 服务网络
- 办事处数量建议9-12个
- 城市名简短明了
- 地区描述可加上区域标识（如"华南地区"）

### 3. **多语言填写**
- 必须填写中英文双语
- 英文翻译要准确专业
- 保持中英文内容含义一致

### 4. **图片使用**
- 支持上传本地图片或输入URL
- 建议使用高质量图片
- 图片大小控制在500KB以内

## 🔧 技术实现

### 组件结构
```
CmsManage.vue (CMS主页面)
  └─ 首页内容标签
      ├─ PageContentEditor (Banner编辑器)
      └─ HomeBlocksEditor (板块编辑器) ⭐新增
          ├─ 关于明升伟业
          ├─ 我们的优势
          └─ 服务网络
```

### 数据存储
- **Store**: `pageContent.js`
- **存储位置**: `localStorage`
- **键名**: `pageContents`
- **数据路径**: `pages.home.aboutCompany/advantages/serviceNetwork`

### 前台组件
- **文件**: `Home.vue`
- **数据来源**: `usePageContentStore()`
- **响应式更新**: 修改CMS后刷新首页即可看到更新

## 📌 注意事项

1. **数据同步**
   - CMS编辑后点击"保存设置"
   - 前台刷新页面查看效果
   - 支持实时预览（刷新即可）

2. **字段限制**
   - 所有文字字段必填中英文
   - 图标只能从预设列表中选择
   - 删除操作不可撤销

3. **性能优化**
   - 图片建议压缩后上传
   - 办事处数量不宜过多（建议≤15个）
   - 优势项不宜过多（建议≤8个）

4. **浏览器兼容**
   - 支持Chrome、Edge、Firefox、Safari
   - 建议使用最新版本浏览器
