# 📖 客户与联系人管理系统 - 快速使用指南

**版本**: V1.0  
**适用对象**: 销售人员、客户经理  
**最后更新**: 2025-12-18

---

## 🚀 快速开始（3分钟上手）

### 第1步：添加客户
1. 打开 **商机管理** 模块
2. 点击 🟢 **添加客户** 按钮
3. 填写客户名称（必填）
4. 点击 **保存客户**
5. ✅ 完成！

### 第2步：添加联系人
1. 点击 🟠 **添加联系人** 按钮
2. 选择客户（必填）
3. 填写姓名、职位（必填）
4. 填写其他信息（可选）
5. 点击 **保存联系人**
6. ✅ 完成！联系人已进入组织图谱

### 第3步：查看组织图谱
1. 打开任意商机详情
2. 点击 **联系人关系图谱** 按钮
3. 看到新增联系人
4. 点击卡片查看详情

## 📋 添加客户详细步骤

### 入口位置
```
首页 → AI营销中台 → 明升AICRM → 商机管理 → 🟢 添加客户
```

### 必填字段（1个）
- ✅ **客户名称**：如"上海明升汽车"

### 可选字段（7个）
- **行业**：选择行业类型
  - 汽车制造 / 电子制造 / 机械制造 / 航空航天 / 新能源 / 其他
  
- **客户类型**：
  - 直接客户 / 代理商 / 分销商
  
- **公司规模**：
  - 大型企业(>1000人) / 中型企业(100-1000人) / 小型企业(<100人)
  
- **年营业额**：如"5亿"
- **地址**：公司地址
- **官网**：公司网址
- **备注**：其他信息

### 保存后效果
- ✅ 提示："客户 XXX 添加成功！"
- ✅ 客户出现在联系人下拉列表中
- ✅ 可以开始添加该客户的联系人

## 👤 添加联系人详细步骤

### 入口位置
```
首页 → AI营销中台 → 明升AICRM → 商机管理 → 🟠 添加联系人
```

### 必填字段（3个）
1. ✅ **所属客户**：从下拉列表选择
2. ✅ **姓名**：联系人姓名
3. ✅ **职位**：如"采购经理"

### 区块1：基本信息（1个可选）
- **性别**：男 / 女

### 区块2：组织结构信息（重要！）
- **分厂**（可选）：
  - 总部 / 一分厂 / 二分厂 / 三分厂 / 其他
  
- **车间**（可选）：
  - 焊装车间 / 总装车间 / 涂装车间 / 冲压车间 / 质检车间 / 其他
  
- **部门**（可选）：
  - 采购部 / 技术部 / 生产部 / 质检部 / 市场部 / 销售部 / 财务部 / 管理层 / 其他
  
- **组织层级**（推荐填写）：
  - 高层 / 中层 / 基层
  
- **汇报上级**（可选）：
  - 从已有联系人中选择
  
- **工作性质**（多选，推荐填写）：
  - 管理决策 / 技术研发 / 生产制造 / 质量控制 / 采购供应 / 销售市场 / 财务核算 / 设备维护
  
- **工作关系**（多选，推荐填写）：
  - 直接负责人 / 配合协同 / 信息接收 / 决策参与 / 技术支持 / 流程审批

### 区块3：决策角色（重要！）
- **决策角色**（推荐填写）：
  - 决策者（拍板人）
  - 推动者（内部支持）
  - 影响者（建议者）
  - 反对者（阻碍者）
  - 使用者（最终用户）
  
- **决策权级别**：1-5星评分

### 区块4：联系方式
- **电话**
- **邮箱**
- **微信**

### 区块5：业务信息
- **主要对接事务**（多选，推荐填写）：
  - 技术沟通 / 生产现场 / 设备管理 / 仓库收货 / 产品验收 / 质检 / 工具管理和标定 / 采购报价 / 询价 / 签合同 / 收发票
  
- **兴趣偏好**（多选）：
  - 技术创新 / 性价比 / 品牌信誉 / 售后服务 / 交付速度 / 产品质量
  
- **沟通偏好**：如"喜欢电话沟通"
- **当前态度**：非常积极 / 积极 / 中立 / 消极
- **备注**：其他信息

### 保存后效果
- ✅ 提示："联系人 XXX 已添加到客户 YYY，已自动进入组织关系图谱"
- ✅ 联系人立即在组织图谱中显示
- ✅ 可以在图谱中看到完整的组织信息

### 入口位置
```
商机详情 → 联系人关系图谱按钮（右上角）
```

```
┌────────────────────────────────────┐
│        高层管理                     │
│  （决策者居多，决策权5星）           │
└────────────────────────────────────┘
              ↓
┌────────────────────────────────────┐
│        中层管理                     │
│  （推动者/影响者，决策权3-4星）      │
│  显示汇报上级关系                   │
└────────────────────────────────────┘
              ↓
┌────────────────────────────────────┐
│        基层执行                     │
│  （使用者居多，决策权1-2星）         │
│  显示汇报上级关系                   │
└────────────────────────────────────┘
```

### 联系人卡片信息
每个联系人卡片显示：
- 👤 头像（姓名首字母）
- 📛 姓名
- 💼 职位
- 🏢 组织信息
  - 分厂（绿色标签）
  - 车间（粉色标签）
  - 部门（蓝色标签）
- 🎯 决策角色标签
- ⭐ 决策权星级
- 🔧 工作性质标签（灰色）
- 🤝 工作关系标签（橙色）

### 查看联系人详情
1. 点击任意联系人卡片
2. 查看完整的20个字段信息
3. 查看互动历史（最近5次）
4. 可点击"编辑"按钮修改信息

## 💡 实用技巧

### 技巧1：快速建立客户档案
```
1. 先添加客户（只填名称即可）
2. 立即添加关键联系人（决策者优先）
3. 后续逐步完善其他联系人
```

### 技巧2：识别关键人物
```
重点填写：
- 决策角色：选择"决策者"或"推动者"
- 决策权级别：4星以上
- 工作关系：包含"决策参与"或"流程审批"
```

### 技巧3：理清组织关系
```
按层级添加：
1. 先添加高层（总经理、副总等）
2. 再添加中层（部门经理）
3. 最后添加基层（主管、工程师）
4. 设置汇报上级关系
```

### 技巧4：区分不同分厂/车间
```
对于多分厂客户：
1. 每个联系人选择对应分厂
2. 选择对应车间
3. 在组织图谱中会自动按组织单位区分
```

### 技巧5：标注工作重点
```
通过"工作性质"和"工作关系"：
- 技术人员：选"技术研发"+"技术支持"
- 采购人员：选"采购供应"+"流程审批"
- 生产人员：选"生产制造"+"信息接收"
```

## ❓ 常见问题

### Q1：没有商机也能添加客户和联系人吗？
**A1**：可以！"添加客户"和"添加联系人"功能完全独立，任何时候都能使用。

### Q2：一个客户可以有多少个联系人？
**A2**：没有限制，一个客户可以添加无限个联系人。

### Q3：必须填写所有20个字段吗？
**A3**：不需要。只有3个字段必填（客户、姓名、职位），其他都是可选的。但建议填写完整以获得更好的管理效果。

### Q4：如何删除联系人？
**A4**：在联系人详情对话框中，点击"编辑"后可以删除。

### Q5：工作性质和工作关系有什么区别？
**A5**：
- **工作性质**：描述该联系人的工作内容（如技术研发、生产制造）
- **工作关系**：描述与我们的协作关系（如直接负责人、技术支持）

### Q6：组织图谱中找不到新增的联系人？
**A6**：请检查：
1. 是否保存成功（查看成功提示）
2. 是否在正确的层级查找（高层/中层/基层）
3. 点击"刷新图谱"按钮

## 🎯 最佳实践

### 初次使用建议
1. **第1天**：添加主要客户（5-10家）
2. **第2天**：为每个客户添加1-2个关键联系人（决策者优先）
3. **第1周**：逐步补充中层和基层联系人
4. **持续**：每次拜访后及时更新联系人信息

### 信息完整度建议
- **最低标准**（必填）：客户、姓名、职位
- **基础标准**（推荐）：+ 决策角色 + 决策权级别 + 联系方式
- **完整标准**（最佳）：+ 组织结构 + 工作性质/关系 + 对接事务

### 数据维护建议
- ✅ 每月更新一次联系人态度
- ✅ 每季度检查汇报关系是否变化
- ✅ 及时添加新接触的联系人
- ✅ 记录每次互动（在互动历史中）
