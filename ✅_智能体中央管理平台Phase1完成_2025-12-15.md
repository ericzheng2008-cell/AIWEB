### 3. 🎯 预注册智能体(8个)

已预注册当前系统的8个核心智能体:

| # | 智能体ID | 名称 | 类型 | 主要能力 |
|---|---------|------|------|---------|
| 1 | `tool-selection-agent` | 拧紧工具智能选型 | 推荐系统 | 工具推荐、品牌匹配、规格查询 |
| 2 | `tightening-process-agent` | 拧紧工艺改进与验证 | 工艺优化 | 曲线分析、参数优化、故障诊断、工艺验证 |
| 3 | `device-management-agent` | 设备全生命周期管理 | 资产管理 | 设备档案、故障追踪、维保管理、备件管理 |
| 4 | `ticketing-agent` | 工单管理系统 | 流程管理 | 工单创建、分配、跟踪、SLA管理 |
| 5 | `curve-analysis-agent` | 拧紧曲线对比分析 | 数据分析 | 曲线对比、异常检测、趋势分析、报告生成 |
| 6 | `brand-matching-agent` | 品牌型号智能匹配 | 推荐系统 | 品牌匹配、型号识别、替代推荐、价格比较 |
| 7 | `data-collection-agent` | 拧紧数据采集分析 | 数据采集 | 数据采集、协议解析、实时监控、数据存储 |
| 8 | `antong-learning-agent` | 安彤自主学习智能体 | 对话AI | NLU、知识问答、主动学习、个性化推荐 |

## 🎯 核心功能演示

### 示例1: 调用智能体

```javascript
import { useAgentRegistryStore } from '@/store/agentRegistry'

const agentStore = useAgentRegistryStore()

// 调用拧紧工具选型智能体
const result = await agentStore.invokeAgent(
  'tool-selection-agent',
  'recommend',
  {
    torqueRange: "80-120Nm",
    industry: "汽车",
    precision: "高"
  }
)

console.log(result)
// {
//   interactionId: "int-1702618800000-abc123",
//   result: {
//     recommendations: [
//       { tool: "ETV ST61-100", score: 0.95, reason: "扭矩范围匹配" },
//       { tool: "Atlas Copco QST15", score: 0.88, reason: "精度要求匹配" }
//     ]
//   },
//   duration: 245
// }
```

### 示例2: 提交反馈

```javascript
// 提交用户反馈
agentStore.submitFeedback(
  'int-1702618800000-abc123',
  {
    feedbackType: 'thumbsUp',
    comment: '推荐很准确',
    correction: null
  }
)

// 自动创建学习记录
// learningRecords中新增一条记录
```

### 示例3: 健康检查

```javascript
// 单个智能体健康检查
const isHealthy = await agentStore.performHealthCheck('tool-selection-agent')

// 所有智能体定期检查(自动执行)
// 每60秒自动检查所有活跃智能体
```
