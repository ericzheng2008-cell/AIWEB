### 方法4：Pandoc 命令行（高级）

**安装 Pandoc**：
```bash
# Windows (需先安装 Chocolatey)
choco install pandoc

# Mac
brew install pandoc

# Linux
sudo apt-get install pandoc
```

**转换命令**：
```bash
cd c:/Users/EricZ/CodeBuddy/AIWEB1

pandoc "📊_AIWEB网站完整汇报文档_2025-12-17_v1.0.md" \
  -o "AIWEB网站完整汇报.pdf" \
  --pdf-engine=xelatex \
  -V CJKmainfont="SimSun" \
  -V geometry:margin=2cm \
  --toc \
  --toc-depth=2 \
  -V documentclass=report \
  -V fontsize=11pt
```

**优点**：
- ✅ 自动化批量处理
- ✅ 高度可定制
- ✅ 命令行集成
