## 🔗 访问路径

### 方法1: 智能体页面
1. 访问 `http://localhost:3002/ai-agents`
2. 滚动到智能体列表
3. 找到**拧紧数据采集分析**卡片（带"新功能"标签）
4. 点击卡片或"立即使用"按钮

### 方法2: 分类筛选
1. 访问智能体页面
2. 点击顶部分类标签：**"数据分析"**
3. 查看拧紧数据采集智能体

### 方法3: 搜索功能
1. 在智能体页面搜索框输入：**"拧紧数据"**
2. 快速定位到该智能体

### 方法4: 直接访问
```
http://localhost:3002/admin/tightening-data
```

## 🎬 使用流程

### Step 1: 进入智能体
```
智能体页面 → 拧紧数据采集分析卡片 → 点击
```

### Step 2: 连接控制器
```
选择控制器类型（PF4000/PF8000）
→ 输入IP地址和端口
→ 点击"连接控制器"
```

### Step 3: 生成模拟数据（测试）
```
点击"使用模拟数据"按钮
→ 自动生成100条测试数据
```

### Step 4: 查看分析结果
- **实时监控Tab**: 查看拧紧记录
- **数据分析Tab**: 统计分析、OK率、Cpk
- **异常报警Tab**: 异常检测结果
- **拧紧曲线Tab**: 可视化曲线

### Step 5: 导出数据
```
点击"导出数据"按钮
→ 下载CSV文件
```
