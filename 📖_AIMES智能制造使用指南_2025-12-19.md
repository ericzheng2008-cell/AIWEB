# 📖 AIMES 智能制造执行系统使用指南

## 🏭 产品概述

**AIMES** (AI Manufacturing Execution System) 是一套面向中小制造业的智能制造执行系统,通过AI技术实现:
- 生产现场实时感知与预警
- 智能排产与资源优化
- 质量预测与全程溯源  
- 设备预测性维护
- 运营洞察与决策支持
- 执行闭环自动化

---

## ⚙️ 六大核心功能模块

### 1️⃣ 生产现场感知
**功能**: 实时监控、异常预警、自动诊断

**主要能力**:
- 实时读取PLC/传感器数据(温度、电流、扭矩、振动)
- 工位节拍自动统计
- 15秒内识别节拍偏差
- 自动生成班组日报

**使用场景**:
```
案例: 机加工产线1号节拍突然下降
系统: 自动识别→分析原因(刀具磨损)→推送建议
```

---

### 2️⃣ 智能排产与调度
**功能**: 自动排产、插单重排、瓶颈识别

**主要能力**:
- 根据工艺路线自动排产
- 订单变化时自动重排并说明影响
- 识别瓶颈工序
- 提前1-3天预测延迟风险

**使用场景**:
```
案例: 紧急插单20件产品
系统: 自动重算→显示影响(需加班2小时)→一键确认
```

---

### 3️⃣ 质量预测与溯源
**功能**: 工艺监控、不良预警、原因分析

**主要能力**:
- 工艺窗口实时监控
- 提前数小时预测质量偏移
- 自动定位导致不良的工位
- 自动归因(人/机/料/法/环)
- 生成8D报告草稿

**使用场景**:
```
案例: 批次B-2025-015出现尺寸偏差
系统: 自动溯源→定位工位2→识别原因(刀具磨损)→隔离批次
```

---

### 4️⃣ 设备预测维护
**功能**: AI PM、故障预测、备件建议

**主要能力**:
- 实时监控振动/温度/电流
- 预测未来X天故障概率
- 自动推荐PM计划
- 自动生成维保工单
- 备件库存预警

**使用场景**:
```
案例: CNC-B02振动异常上升20%
系统: 预测7天后故障→推荐润滑保养→创建PM工单
```

---

### 5️⃣ 运营洞察分析
**功能**: OEE分析、产能预测、成本优化

**主要能力**:
- 自动生成OEE诊断
- 自动发现产能浪费点
- 识别最耗成本工序
- 每周AI运营报告
- 自动推送"最值得改善的3件事"

**指标说明**:
```
本周OEE: 85% (优秀: ≥85%, 良好: 70-85%, 需改善: <70%)
产能浪费: 12% (停机8% + 换线4%)
成本节约: 8.5万/月
```

---

### 6️⃣ 执行闭环管理
**功能**: 任务下发、进度跟踪、闭环确认

**主要能力**:
- 异常自动创建任务
- 自动匹配责任人
- 提供标准SOP
- 延误自动升级
- 任务完成率统计

**闭环流程**:
```
异常发现 → 创建任务 → 推送责任人 → 提供SOP → 执行确认 → 自动归档
```

## 🎯 典型应用场景

### 场景1: 机加工刀具寿命预测
**痛点**: 刀具突然失效导致停机,影响交期

**AIMES解决方案**:
1. 实时监控电流/扭矩/振动
2. AI预测刀具剩余寿命
3. 提前推送换刀建议
4. 延长刀具寿命10-20%

**效果**: 减少意外停机,降低成本

---

### 场景2: 插单重排产
**痛点**: 靠经验+Excel排产,耗时且容易出错

**AIMES解决方案**:
1. 输入紧急订单
2. 自动重算所有工单
3. 显示交期影响
4. 一键确认执行

**效果**: 排产时间从2小时→5分钟

---

### 场景3: 质检不良溯源
**痛点**: 批次不良难以快速定位原因

**AIMES解决方案**:
1. 自动记录全程工艺参数
2. AI关联分析(材料/设备/人员/工艺)
3. 定位最可能原因
4. 自动生成8D报告

**效果**: 返修率降低10-30%

## 💡 快速操作指南

### 如何启动智能体?
1. 进入AIMES视图
2. 找到"AI智能体助手矩阵"
3. 点击对应智能体卡片
4. 点击"启动助手"按钮

### 如何查看设备健康?
1. 查看"设备健康指数(EHI)"图表
2. 关注红色区域(健康指数<70)
3. 点击"排PM"按钮创建维护工单

### 如何处理异常预警?
1. 查看"今日异常与预警"时间轴
2. 点击异常项查看详情
3. 系统自动创建任务并分派责任人
4. 责任人执行并确认闭环
