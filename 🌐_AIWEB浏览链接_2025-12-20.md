## 🔍 故障排除

### 1. 端口被占用
```bash
# 查看5173端口占用
netstat -ano | findstr "5173"

# 强制结束进程
taskkill /PID <进程ID> /F
```

### 2. 服务器未启动
```bash
# 检查Node.js是否安装
node --version
npm --version

# 重新安装依赖
npm install
```

### 3. 页面空白
- 清除浏览器缓存（Ctrl+Shift+Delete）
- 检查控制台错误（F12）
- 重启服务器
