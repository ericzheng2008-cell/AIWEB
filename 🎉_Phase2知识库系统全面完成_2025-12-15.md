## 📖 使用场景示例

### 场景1: 智能体使用知识

```javascript
// 拧紧工具选型智能体查询知识
const kb = useKnowledgeBase()

// 搜索相关知识
const knowledge = kb.searchKnowledge('电动扳手选型', {
  category: 3,  // 品牌型号
  type: 'faq'
})

// 使用知识回答用户
const answer = knowledge[0].content

// 记录使用
kb.recordAgentUsage(knowledge[0].id, 'agent-1')
```

### 场景2: 技术文档管理

```javascript
// 添加新的技术规范
kb.addKnowledge({
  title: 'ISO 5393拧紧标准',
  content: '# ISO 5393...',
  type: 'specification',
  category: 4,  // 技术文档
  tags: ['ISO标准', '拧紧规范'],
  status: 'active',
  author: '技术部'
})
```

### 场景3: 案例库建设

```javascript
// 添加成功案例
kb.addKnowledge({
  title: '某汽车厂拧紧改进案例',
  content: '## 项目背景\n...\n## 解决方案\n...',
  type: 'case',
  category: 5,  // 案例库
  tags: ['工艺优化', '汽车行业'],
  status: 'active',
  author: '项目经理'
})
```
