# 📖 AICRM三项新功能使用指南

**版本**: V1.0  
**发布日期**: 2025-12-19  
**适用对象**: AICRM系统用户

---

## 🎯 功能概述

本次更新为AICRM系统增加了3个核心功能的完整实现：
1. **联系人管理** - 添加客户联系人及生产信息
2. **投标预测** - 创建和管理投标项目
3. **客户360** - 添加客户时录入详细的生产信息

---

## 1️⃣ 联系人管理功能

### 📍 功能位置
**路径**: 主导航栏 → 联系人与关系 → 添加联系人

### 🎯 使用场景
- 新增客户关键联系人
- 记录联系人的生产岗位信息
- 评估联系人的影响力等级
- 建立客户关系网络图谱

### 📝 操作步骤

#### 步骤1：打开添加联系人对话框
1. 点击顶部导航栏的"**联系人与关系**"标签
2. 在页面右上角点击"**添加联系人**"按钮
3. 弹出"添加联系人"对话框

#### 步骤2：填写基础信息（必填）
| 字段 | 说明 | 示例 |
|------|------|------|
| 姓名 | 联系人姓名 | 张工 |
| 职位 | 联系人职位 | 工艺工程师 |
| 所属公司 | 联系人所在公司 | 广汽本田 |

#### 步骤3：填写生产信息（可选）

**分厂名称**（8个预设选项 + 自定义）
```
传祺一厂、传祺二厂
埃安一厂、埃安二厂
丰田三厂、丰田五厂
本田3厂、本田采购总部
```

**车间名称**（4个预设选项 + 自定义）
```
总装1部、焊装1部
总装2部、焊装2部
```

**线体名称**（5个预设选项 + 自定义）
```
总装1线、总装2线
焊装1线、焊装2线
涂装1线
```

**部门名称**（6个预设选项 + 自定义）
```
工艺科、质量科
工具房、库房
生产管理室、设备科
```

**工位名称**（手动输入）
```
示例：OP10、OP20、OP30、焊接工位1号等
```

#### 步骤4：填写联系方式（可选）
| 字段 | 说明 |
|------|------|
| 联系电话 | 手机或座机 |
| 电子邮箱 | 工作邮箱 |
| 微信号 | 微信账号 |

#### 步骤5：评估影响力
- 使用星级评分（1-5星）
- 默认为3星（中等影响力）
- 可根据联系人在决策中的作用调整

#### 步骤6：添加备注
- 记录特殊说明或重要信息
- 支持多行文本输入

#### 步骤7：提交表单
1. 检查必填项是否填写完整
2. 点击"**确定添加**"按钮
3. 系统提示"联系人添加成功！"
4. 对话框自动关闭
5. 关系图谱自动刷新

### 💡 使用技巧

#### 技巧1：快速选择
- 所有下拉框支持**可搜索**功能
- 输入关键字快速筛选选项
- 例如：输入"总装"快速找到总装相关选项

#### 技巧2：自定义输入
- 如果预设选项不满足需求
- 可以直接在下拉框中**输入新值**
- 系统会自动创建新选项

#### 技巧3：批量录入
- 建议先准备好联系人信息表格
- 逐个录入时可复制粘贴信息
- 提高录入效率

### ⚠️ 注意事项
- ❗ 姓名、职位、公司为**必填项**
- ❗ 影响力评分会影响关系图谱展示
- ❗ 备注信息支持关键字搜索

## 2️⃣ 投标预测功能

### 📍 功能位置
**路径**: 主导航栏 → 投标预测 → 新建投标项目

### 🎯 使用场景
- 创建新的投标项目
- 记录竞争对手信息
- 预测投标成功率
- 管理投标进度

### 📝 操作步骤

#### 步骤1：打开新建投标项目对话框
1. 点击顶部导航栏的"**投标预测**"标签
2. 在页面中点击"**新建投标项目**"按钮
3. 弹出"新建投标项目"对话框

#### 步骤2：填写项目基础信息（必填）
| 字段 | 说明 | 示例 |
|------|------|------|
| 项目名称 | 投标项目名称 | XX工厂拧紧系统升级项目 |
| 客户名称 | 目标客户 | 广汽本田 |

#### 步骤3：填写项目详细信息（可选）

**项目金额**
- 单位：万元
- 示例：500（表示500万元）
- 可留空，后续补充

**投标日期**
- 使用日期选择器
- 选择正式投标的日期
- 可留空，后续补充

**竞争对手**
- 支持多选
- 预设选项：竞争对手A、竞争对手B、竞争对手C
- 支持手动输入新竞争对手名称
- 示例：阿特拉斯、英格索兰、马头

**项目描述**
- 多行文本输入
- 描述项目背景、需求、特点等
- 建议详细填写，便于后续分析

#### 步骤4：提交项目
1. 检查必填项是否填写完整
2. 点击"**确定**"按钮
3. 系统提示"投标项目创建成功！"
4. 对话框自动关闭
5. 项目列表自动刷新

### 💡 使用技巧

#### 技巧1：竞争对手管理
- 第一次输入竞争对手时可手动创建
- 后续项目可直接选择已有竞争对手
- 支持同时选择多个竞争对手

#### 技巧2：项目金额规范
- 统一使用"万元"为单位
- 例如：500万元输入"500"
- 避免输入"5000000"等大数字

#### 技巧3：投标日期提醒
- 设置投标日期后系统可自动提醒
- 建议提前规划投标准备时间

### ⚠️ 注意事项
- ❗ 项目名称、客户名称为**必填项**
- ❗ 竞争对手信息影响AI预测准确性
- ❗ 项目创建后状态默认为"进行中"

## 3️⃣ 客户360功能增强

### 📍 功能位置
**路径**: 主导航栏 → 客户360° → 新增客户

### 🎯 使用场景
- 新增客户档案
- 记录客户的生产组织架构
- 管理客户联系信息
- 为客户打标签分类

### 📝 操作步骤

#### 步骤1：打开新增客户对话框
1. 点击顶部导航栏的"**客户360°**"标签

#### 步骤2：填写基础信息（必填）

```
丰田三厂、丰田五厂
本田3厂、本田采购总部
```

**车间名称**（4个预设选项 + 自定义）
```
总装1部、焊装1部
总装2部、焊装2部
```

**线体名称**（5个预设选项 + 自定义）
```
总装1线、总装2线
焊装1线、焊装2线
涂装1线
```

**部门名称**（6个预设选项 + 自定义）
```
工艺科、质量科
工具房、库房
生产管理室、设备科
```

**工位名称**（手动输入）

```
示例：OP10、OP20、OP30等
```

|------|------|

### 💡 使用技巧

#### 技巧1：生产信息分层录入
```
推荐录入顺序：
分厂 → 车间 → 线体 → 部门 → 工位
```
- 先填写上级单位，再填写下级单位
- 有助于建立完整的组织架构

#### 技巧2：快速选择与自定义结合
- 优先选择预设选项（提高数据一致性）
- 遇到特殊情况可手动输入
- 例如："丰田六厂"可直接输入

#### 技巧3：标签分类管理
- 同一客户可打多个标签
- 便于后续按标签筛选客户
- 建议根据企业实际情况定义标签体系

### ⚠️ 注意事项
- ❗ 客户名称、联系人为**必填项**
- ❗ 生产信息虽为可选，但建议尽量填写完整
- ❗ 所有下拉框支持**可搜索**和**允许创建**
- ❗ 对话框宽度已扩展至700px，请使用较大屏幕操作

## 🔧 通用功能说明

### 下拉框特性

#### 1. 可搜索（Filterable）
- 点击下拉框后可直接输入关键字
- 系统自动筛选匹配的选项
- 提高选择效率

**示例**：
```
输入"总装" → 显示：总装1部、总装2部、总装1线、总装2线
输入"焊" → 显示：焊装1部、焊装2部、焊装1线、焊装2线
```

#### 2. 允许创建（Allow Create）
- 如果预设选项中没有需要的值
- 可以直接在输入框中输入新值
- 按Enter键或失焦后自动创建

**示例**：
```
预设选项：传祺一厂、传祺二厂
自定义输入：传祺三厂 → 自动添加为新选项
```

### 表单验证

#### 必填项验证
- 必填项标记为"红色星号 ✱"
- 未填写必填项时点击提交
- 系统弹出友好提示："请填写必填项：XXX"

#### 实时验证
- 部分字段支持实时格式验证
- 例如：邮箱格式、电话号码格式
- 输入错误时显示提示信息

### 数据保存

#### 自动保存功能
- 提交成功后数据立即保存
- 无需手动刷新页面
- 列表和图表自动更新

#### 数据完整性
- 所有数据包含创建时间
- 自动生成唯一ID
- 支持后续编辑和删除

## 🎓 最佳实践

### 1. 数据录入规范

#### 命名规范
```
✅ 推荐：广汽本田汽车有限公司
❌ 避免：gqbt、本田

✅ 推荐：总装1部
❌ 避免：总装部1

✅ 推荐：张工、李经理
❌ 避免：小张、老李
```

#### 信息完整性
- 尽量填写所有可选字段
- 完整信息有助于AI分析
- 备注字段记录特殊情况

### 2. 工作流程建议

#### 新客户录入流程
```
1. 新增客户（客户360）
2. 添加客户联系人（联系人管理）
3. 如有投标机会，创建投标项目（投标预测）
4. 定期更新客户信息和联系人影响力评分
```

#### 投标项目管理流程
```
1. 创建投标项目
2. 记录竞争对手信息
3. 运行AI投标预测分析
4. 根据AI建议调整投标策略
5. 更新项目进度状态
```

### 3. 数据质量保证

#### 数据一致性
- 同一客户的多个联系人应使用相同的公司名称
- 分厂、车间等名称应保持一致
- 避免"本田3厂"和"本田三厂"混用

#### 定期维护
- 每月检查客户信息准确性
- 更新联系人职位变动
- 清理无效或重复数据

## ❓ 常见问题解答

### Q1：为什么有些字段是必填，有些是可选？
**A**: 必填项是保证系统正常运行的最小信息集。可选项可以根据实际情况填写，填写越完整，AI分析越准确。

### Q2：下拉框中的选项是否可以自定义？
**A**: 可以！所有带有"允许创建"功能的下拉框都支持自定义输入。只需在输入框中输入新值即可。

### Q3：如何修改已添加的联系人或客户信息？
**A**: 目前支持在列表中查看详情，后续版本将增加编辑功能。

### Q4：联系人的影响力评分有什么作用？
**A**: 影响力评分用于：
- 关系图谱中节点大小展示
- AI分析客户决策链
- 优先级排序和提醒

### Q5：投标项目的竞争对手信息会如何使用？
**A**: 竞争对手信息将用于：
- AI投标成功率预测
- 竞争态势分析
- 历史胜率统计

### Q6：生产信息（分厂、车间等）必须按层级填写吗？
**A**: 不是必须的。您可以只填写其中几项，系统会保存您填写的所有信息。但建议尽量完整填写，便于后续分析。
