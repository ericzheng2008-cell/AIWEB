# 快速入门：产品与服务页面编辑

## 🚀 5分钟上手指南

### 第一步：登录后台
1. 访问：http://localhost:3003/admin/login
2. 输入账号：`admin`
3. 输入密码：`admin123`
4. 点击登录

### 第二步：进入管理界面
1. 在左侧菜单找到 **"产品与服务管理"**
2. 点击进入

### 第三步：编辑横幅（可选）
1. 点击 **"📸 页面横幅"** 标签
2. 修改标题和副标题
3. 上传背景图片或选择背景颜色
4. 点击 **"保存横幅"**

### 第四步：添加产品分类
1. 点击 **"📦 产品分类卡片"** 标签
2. 点击 **"添加分类"** 按钮
3. 填写信息：
   ```
   中文名称：电动工具
   中文描述：高性能电动拧紧工具
   
   英文名称：Electric Tools
   英文描述：High-performance electric tools
   ```
4. 上传分类图片（推荐尺寸：600x450px）
5. 点击 **"保存"**

### 第五步：查看前台效果
1. 访问：http://localhost:3003/products-services
2. 查看更新后的页面效果
3. 如果没有更新，按 F5 刷新页面

---

## 💡 常用操作

### 修改横幅颜色
```
1. 进入"页面横幅"标签
2. 点击"背景颜色"色块
3. 选择颜色或输入：
   linear-gradient(135deg, #667eea 0%, #764ba2 100%)
4. 保存
```

### 编辑现有分类
```
1. 鼠标悬停在分类卡片上
2. 点击"编辑"按钮
3. 修改信息
4. 保存
```

### 调整页面布局
```
1. 进入"⚙️ 页面设置"标签
2. 选择卡片列数（2/3/4列）
3. 调整圆角和阴影
4. 保存设置
```

## ❓ 常见问题

**Q: 修改后前台没有更新？**  
A: 按 F5 刷新页面，或按 Ctrl+F5 强制刷新

**Q: 上传的图片不显示？**  
A: 检查图片格式（JPG/PNG），大小不超过5MB

**Q: 忘记密码怎么办？**  
A: 默认密码是 `admin123`
