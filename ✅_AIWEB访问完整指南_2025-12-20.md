## 🔍 故障排除

### 1. 服务器启动失败
```bash
# 检查Node.js版本
node --version  # 需要 v14+
npm --version   # 需要 v6+

# 重新安装依赖
npm install
```

### 2. 端口被占用
```bash
# 查看占用进程
netstat -ano | findstr "5173"

# 结束进程
taskkill /PID <进程ID> /F
```

### 3. 页面空白或报错
- 清除浏览器缓存：`Ctrl+Shift+Delete`
- 打开开发者工具：`F12` 查看错误
- 重启服务器

### 4. 样式丢失
- 强制刷新：`Ctrl+F5`
- 检查控制台CSS错误
