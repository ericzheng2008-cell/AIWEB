## 🎯 AI赢率系统 (核心亮点)

### Salesforce Einstein级公式

```javascript
WinRate = BaseProb(stage) 
        + RelationshipScore × 0.25    // 客户关系强度
        + SentimentScore × 0.18       // 客户情绪
        + CompetitorImpact × -0.15    // 竞争对手压力
        + ProcessScore × 0.12         // 内部流程
        + ChampionPower × 0.30        // 黑马推动者/决策者
        + ProductFit × 0.05           // 产品契合度
        + PriceSensitivity × -0.05    // 价格敏感度
```

### 8大影响维度权重

| 维度 | 权重 | 说明 |
|------|------|------|
| 客户关系强度 | 25% | 拜访次数、互动频率、信任度 |
| 客户情绪 | 18% | 正面/负面情绪分析 |
| 竞争对手压力 | -15% | 竞品活跃度、客户态度 |
| 内部流程 | 12% | 方案完成度、审批进度 |
| 黑马推动者/决策者 | 30% | 决策链影响力 |
| 产品契合度 | 5% | 需求匹配度 |
| 价格敏感度 | -5% | 价格谈判难度 |
| 流程推进 | - | 阶段转化速度 |
