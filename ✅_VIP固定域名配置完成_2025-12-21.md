## 📝 快速配置步骤

### 步骤1：创建配置文件

在 `%USERPROFILE%\.cpolar\cpolar.yml` 文件中配置：

```yaml
version: "2"
authtoken: YOUR_AUTHTOKEN_HERE

tunnels:
  # 前端服务 - 使用已保留的二级子域名
  aiweb-frontend:
    proto: http
    addr: 5173
    region: cn_vip
    subdomain: msaiweb
    inspect: true
    bind_tls: true
  
  # 后端服务 - 需要先保留 msaiweb-api 子域名
  aiweb-backend:
    proto: http
    addr: 5000
    region: cn_vip
    subdomain: msaiweb-api
    inspect: true
    bind_tls: true
```

### 步骤2：保留后端子域名

1. 访问：https://dashboard.cpolar.com/reserved
2. 点击「保留二级子域名」
3. 填写信息：
   - 地区：`China VIP`
   - 二级域名：`msaiweb-api`
   - 描述：`AIWEB后端API服务`
4. 点击保留

### 步骤3：设置authtoken

```bash
cpolar authtoken YOUR_COPIED_TOKEN
```

获取token：https://dashboard.cpolar.com/get-started

### 步骤4：启动服务

```bash
# 方式1：启动所有隧道
cpolar start-all

# 方式2：单独启动
cpolar start aiweb-frontend
cpolar start aiweb-backend
```

## ⚠️ 重要提示

### 1. 保留域名规则
- ✅ 即使隧道离线，域名也不会分配给其他用户
- ✅ 启动隧道时必须使用 `-subdomain=msaiweb` 参数
- ❌ 不使用参数将分配临时随机域名

### 2. 自定义域名CNAME
- ✅ DNS配置正确：CNAME → `1c716cc.cname.vip.cpolar.cn`
- ✅ 启动隧道使用：`-hostname=www.eqtcf.cn`
- ⏱️ DNS生效时间：5-30分钟

### 3. TCP地址使用
```bash
# 用于SSH、RDP等TCP服务
cpolar tcp 22 -remote-addr=3.tcp.vip.cpolar.cn:12423
```

### 4. authtoken安全
- ⚠️ 不要分享您的authtoken
- ⚠️ authtoken相当于账号密码
- ✅ 定期在控制台重置token
