git push origin main

## 📞 获取帮助

### GitHub仓库相关问题

**无法访问仓库**：
- 检查网络连接
- 确认GitHub未被防火墙阻止
- 尝试使用VPN或代理

**克隆失败**：
```bash
# 检查Git是否安装
git --version

# 检查网络连接
ping github.com

# 使用HTTPS代替SSH
git clone https://github.com/ericzheng2008-cell/AIWEB.git
```

**推送失败**：
```bash
# 检查远程仓库配置
git remote -v

# 重新添加远程仓库
git remote add origin https://github.com/ericzheng2008-cell/AIWEB.git

# 强制推送（谨慎使用）
git push -f origin main
```
