## 🎓 使用场景示例

### 场景1：客户问"你们和传统维修商有什么区别？"
```
操作：打开 Tab 4「成本冰山」
话术：
  "您看，传统维修商只关注水上的30%（采购成本），
   我们帮您优化水下的70%（隐性成本）。
   通过年包服务，可以将隐性成本降低40-60%。"

效果：客户理解差异化价值
```

### 场景2：客户问"能帮我省多少钱？"
```
操作：打开 Tab 1「必中算账法」
步骤：
  1. 问客户："您现在有多少把工具？"（输入100）
  2. "单价大概多少？"（输入20000）
  3. "现在一般用几年？"（输入3）
  4. 点击「立即算账」
  5. 指着屏幕："您看，每年直接省10.5万，这还不包括..."
  6. 指向额外收益："停线规避20-30万，库存释放50-80万"

效果：客户看到具体数字，容易做决策
```

### 场景3：客户问"哪个方案适合我？"
```
操作：打开 Tab 2「年包智能推荐」
步骤：
  1. "您预算充裕吗？"（选择预算水平）
  2. "这条产线重要吗？"（拖动滑块1-10分）
  3. "现在最痛的是什么？"（勾选痛点）
  4. 点击「智能推荐」
  5. 展示推荐产品："根据您的情况，推荐标准运营包"
  6. 说明推荐理由："90%客户选择，续签率90%"

效果：客户感受专业性，信任度提升
```

### 场景4：客户问"分几步实施？"
```
操作：打开 `/equipment-lifecycle`，下滑到实施路线图
展示：
  阶段1（1-2月）：资产摸底 → 产出风险清单
  阶段2（3-6月）：快速止血 → 停机率下降30-50%
  阶段3（6-12月）：结构降本 → CAPEX下降52%

话术："第一阶段只需配合3件事（指向P18内容），
       其余全部我们负责"

效果：客户放心，降低决策门槛
```
