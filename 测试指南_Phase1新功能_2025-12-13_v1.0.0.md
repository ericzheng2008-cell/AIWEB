## 🎯 Phase 1 功能测试清单

### ✅ **功能1：拧紧策略优化**

#### 访问路径
- **直接访问**：`http://localhost:3003/tightening-strategy`
- **智能体入口**：首页 → AI智能体 → 拧紧工具选型与工艺改进 → 拧紧策略优化

#### 测试场景
1. **场景一：查看预置策略案例**
   - 左侧应显示5个典型策略案例卡片
   - 检查每个案例的信息完整性（工位、螺栓数量、推荐策略等）

2. **场景二：筛选策略**
   - 在右侧筛选面板输入工位名称："发动机"
   - 点击"查询策略"按钮
   - 应高亮显示"发动机缸盖"案例

3. **场景三：智能推荐**
   - 在筛选条件中选择：
     - 螺栓数量：8个
     - 扭矩范围：60Nm
     - 重要程度：A级（关键部件）
   - 点击"智能推荐"
   - 应显示推荐的策略（对称拧紧法）

4. **场景四：可视化查看**
   - 点击任意案例的"查看拧紧顺序"按钮
   - 应弹出模态框，显示拧紧顺序的可视化图
   - 检查是否有螺栓编号和箭头指示

5. **场景五：下载策略**
   - 点击任意案例的"下载策略"按钮
   - 应自动下载txt格式的策略文档
   - 打开文件检查内容完整性

#### ✓ 测试通过标准
- [ ] 5个案例正确显示
- [ ] 筛选功能正常工作
- [ ] 智能推荐算法有效
- [ ] 可视化图正确渲染
- [ ] 文档下载成功

### ✅ **功能2：成本优化分析**

#### 访问路径
- **直接访问**：`http://localhost:3003/cost-optimization`
- **智能体入口**：首页 → AI智能体 → 拧紧工具选型与工艺改进 → 成本优化分析

#### 测试场景
1. **场景一：查看优化案例**
   - 左侧应显示3个优化案例卡片
   - 检查案例的关键指标：年节省额、优化点数等

2. **场景二：成本计算器**
   - 在右侧计算器中输入：
     - 工具数量：10台
     - 单价：5000元
     - 年使用小时：2000h
     - 能耗：0.5kW
     - 电价：0.6元/度
     - 维护成本：500元/年
     - 寿命：5年
   - 点击"计算成本"
   - 应显示TCO、年运营成本等指标

3. **场景三：优化策略查看**
   - 向下滚动查看"优化策略库"
   - 应显示5个优化策略卡片
   - 检查每个策略的描述和潜在收益

4. **场景四：智能建议生成**
   - 点击"生成优化建议"按钮
   - 应在右侧显示个性化的优化建议
   - 建议应包含具体的成本节省预估

5. **场景五：报告下载**
   - 点击"下载分析报告"按钮
   - 应下载包含完整分析的txt文件
   - 检查报告内容是否包含计算结果和建议

#### ✓ 测试通过标准
- [ ] 3个案例正确显示
- [ ] 成本计算器计算准确
- [ ] 5个策略正确展示
- [ ] 智能建议生成合理
- [ ] 报告下载成功

### ✅ **功能3：在线设备查询**

#### 访问路径
- **直接访问**：`http://localhost:3003/device-status`
- **智能体入口**：首页 → AI智能体 → 服务状态查询 → 在线设备查询

#### 测试场景
1. **场景一：设备概览**
   - 顶部应显示4个统计卡片
   - 检查总设备数、在线数、离线数、健康率

2. **场景二：设备列表查看**
   - 应显示6台设备的详细信息
   - 检查每台设备的：
     - 设备编号、型号、位置
     - 运行状态（在线/离线/维护中）
     - 健康度评分
     - 运行时长、扭矩范围

3. **场景三：状态筛选**
   - 点击"在线"筛选按钮
   - 应只显示在线设备（5台）
   - 点击"离线"筛选按钮
   - 应只显示离线设备（1台）

4. **场景四：位置筛选**
   - 在筛选条件中选择位置："装配线A"
   - 应只显示该位置的设备
   - 点击"重置筛选"恢复全部

5. **场景五：关键词搜索**
   - 在搜索框输入："博世"
   - 应筛选出所有博世品牌的设备

6. **场景六：查看设备详情**
   - 点击任意设备的"查看详情"按钮
   - 应弹出详情对话框
   - 检查详情信息：运行状态、参数、预警信息、维护记录

7. **场景七：报告下载**
   - 点击"导出报告"按钮
   - 应下载包含所有设备信息的txt报告
   - 检查报告内容完整性

#### ✓ 测试通过标准
- [ ] 统计卡片数据正确
- [ ] 6台设备正确显示
- [ ] 状态筛选功能正常
- [ ] 位置筛选有效
- [ ] 关键词搜索准确
- [ ] 详情对话框完整
- [ ] 报告下载成功

## 🐛 常见问题排查

### 问题1：页面无法访问
**解决方案**：
```bash
# 检查服务器是否正在运行
# 查看控制台是否有错误信息
# 确认端口3003未被占用
```

### 问题2：数据不显示
**解决方案**：
- 打开浏览器开发者工具（F12）
- 查看Console标签页的错误信息
- 检查Network标签页的网络请求

### 问题3：样式错乱
**解决方案**：
- 清除浏览器缓存（Ctrl + Shift + Delete）
- 硬刷新页面（Ctrl + F5）
- 检查Element Plus组件库是否正确加载
