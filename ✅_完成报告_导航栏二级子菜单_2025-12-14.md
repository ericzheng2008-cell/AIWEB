## 🚀 使用指南

### 1. 基本使用

**悬停查看**:
```
鼠标移动到菜单项 → 自动展开下拉菜单
鼠标移出菜单区域 → 自动收起菜单
```

**点击跳转**:
```
点击子菜单项 → 跳转到对应页面
URL参数自动携带 → 页面自动筛选/定位
```

### 2. URL参数说明

**产品与服务**:
```
/products-services?category=electric     # 电动工具
/products-services?category=pneumatic    # 气动工具
/products-services?category=manual       # 手动工具
/products-services?category=measurement  # 测量工具
/products-services?category=automation   # 自动化系统
/products-services?category=smart        # 智能解决方案
```

**事业部**:
```
/divisions?id=1  # 工业智能装配事业部
/divisions?id=2  # 工业智能智造事业部
/divisions?id=3  # 工业配套事业部

/divisions?id=5  # 汽车部件事业部
```

```javascript
// 自动检测子菜单数量，超过4个切换为双列
<div 
  class="nav-item-dropdown"
  :data-has-many-children="navItem.children.length > 4">
```

### 2. 流畅动画效果
```css
transition: all 0.3s ease;
transform: translateX(4px);  /* 悬停微位移 */
```

### 3. 多语言支持
```javascript
{{ navItem.name[locale] || navItem.name['zh-CN'] }}
// 自动切换 + 降级显示
```
