# 📘 小课堂智能体功能使用演示

**更新日期:** 2025-12-15  
**适用版本:** v1.0.0

---

## 🎯 功能概览

产品技术销售小课堂是一个完整的在线学习平台,支持:
- 📚 **多级分类体系** - 一级分类(9个技术领域) + 二级分类(40+专题)
- 📝 **图文课程** - 支持封面、标签、难度等级、统计数据
- 🎬 **视频教程** - 支持上传视频、缩略图、时长显示
- 🔗 **外部链接** - 集成官方学院、技术文档等外部资源
- 📊 **数据统计** - 浏览量、点赞数实时统计

## 🎬 前台用户操作演示

### 场景1: 浏览技术课程

#### 步骤1: 进入小课堂
1. 访问: `http://localhost:5173/#/tech-classroom`
2. 看到首页Banner和9个一级分类卡片
3. 每个卡片显示:
   - 🤖 大号图标
   - 分类名称
   - 描述文字
   - 主题数量、课程数量

#### 步骤2: 选择技术领域
1. 点击"🤖 协作机器人"卡片
2. 弹出二级分类对话框
3. 显示4个二级分类:
   - 协作机器人基础
   - 编程与调试
   - 安全规范
   - 应用案例

#### 步骤3: 选择具体主题
1. 点击"协作机器人基础"
2. 弹出课程列表对话框
3. 看到三个区域:
   - 🎬 **视频教程** (如有)
   - 🔗 **推荐学习资源** (外部链接)
   - 📖 **图文课程**

#### 步骤4: 观看视频
1. 点击任意视频卡片
2. 弹出视频播放器对话框
3. 自动播放视频
4. 显示视频信息:
   - 标题、描述
   - 浏览量、点赞数
   - 时长、上传时间
5. 点击"点赞"按钮,点赞数+1
6. 点击X按钮关闭播放器

#### 步骤5: 查看图文课程
1. 点击课程卡片
2. 弹出课程详情对话框
3. 显示:
   - 封面图片
   - 课程标题、描述
   - 作者、时长、难度等级
   - 浏览量、点赞数
   - 完整课程内容
   - 标签列表
4. 点击"点赞"按钮
5. 关闭对话框

#### 步骤6: 访问外部资源
1. 点击外部链接卡片(如"节卡学院")
2. 新标签页打开外部网站
3. 浏览量自动+1
4. 显示成功提示

## 🛠️ 后台管理操作演示

### 场景2: 添加新的视频教程 ⭐

#### 步骤1: 进入视频管理
1. 访问: `http://localhost:5173/#/admin/classroom`
2. 点击"视频管理"标签
3. 看到按分类分组的视频列表

#### 步骤2: 上传视频
1. 找到目标分类,点击"添加视频"按钮
2. 或点击顶部"上传视频"按钮
3. 弹出视频上传对话框

#### 步骤3: 填写视频信息
```
选择分类: [一级分类] → [二级分类]
例如: 协作机器人 → 协作机器人基础

视频标题: "协作机器人安装与调试教程"
视频描述: "详细讲解JAKA Zu系列协作机器人的安装步骤..."

视频文件: 
  方式1: 点击上传 → 选择本地MP4文件 (≤50MB推荐)
  方式2: 直接输入URL → https://cdn.example.com/videos/jaka-install.mp4

视频时长: "15:30"

缩略图:
  方式1: 点击上传 → 选择图片
  方式2: 输入URL → https://cdn.example.com/thumbnails/jaka-install.jpg
```

#### 步骤4: 保存视频
1. 点击"保存"按钮
2. 看到成功提示
3. 视频出现在列表中
4. 显示缩略图、标题、描述、时长、统计数据

#### 步骤5: 验证前台显示
1. 切换到前台页面
2. 进入对应分类
3. 看到新上传的视频
4. 点击播放测试

### 场景3: 添加外部学习资源

#### 步骤1: 进入外部链接管理
1. 点击"外部链接管理"标签
2. 看到现有链接列表

#### 步骤2: 添加节卡学院链接
1. 点击"添加外部链接"按钮
2. 填写表单:

```
一级分类: 协作机器人
二级分类: 协作机器人基础 (可选)

链接标题: "节卡学院 - 协作机器人技术学习平台"
链接地址: https://www.jaka.com/zh/jakaAcademy

链接描述: "节卡机器人官方学习平台,提供完整的协作机器人技术课程、编程教程和应用案例"

链接类型: 🎓 学院平台
语言: 中文
图标: 🎓

需要会员: 关闭 (免费资源)
排序: 1

标签: [官方课程] [协作机器人] [编程教程]
```

#### 步骤3: 保存并验证
1. 点击链接地址旁的"预览链接"测试
2. 点击"保存"
3. 在列表中看到新添加的链接
4. 切换状态开关测试激活/停用

#### 步骤4: 前台验证
1. 前台进入对应分类
2. 在"推荐学习资源"区域看到节卡学院卡片
3. 点击卡片跳转到官网

### 场景5: 删除测试数据

#### 步骤1: 删除视频
1. 在视频列表找到测试视频
2. 点击"删除"按钮
3. 弹出确认对话框: "确定要删除这个视频吗?"
4. 点击"确定"
5. 视频从列表消失

#### 步骤2: 验证级联删除
1. 删除一级分类时:
   - 确认框提示: "删除分类将同时删除其下的所有子分类和课程"
   - 确认后,该分类下所有内容全部删除
2. 删除二级分类时:
   - 提示: "删除二级分类将同时删除其下的所有课程"
   - 该子分类下的课程、视频全部删除

## 🔍 调试技巧

### 查看LocalStorage数据
1. 打开浏览器开发者工具(F12)
2. 切换到"Application"或"存储"标签
3. 左侧展开"Local Storage"
4. 点击当前域名
5. 查看键值对:
   - `classroom_categories` - 一级分类数据
   - `classroom_subcategories` - 二级分类数据(含视频)
   - `classroom_lessons` - 课程数据
   - `classroom_external_links` - 外部链接数据

### 查看控制台日志
1. Console标签
2. 检查红色错误信息
3. 检查警告信息(黄色)
4. 查看网络请求(Network标签)

### 数据格式检查
```javascript
// 在Console中运行
const store = JSON.parse(localStorage.getItem('classroom_subcategories'))
console.log(store.find(s => s.id === 101)?.videos)
// 查看某个二级分类的视频列表
```

## 🎓 最佳实践

### 视频上传建议
```
✅ 推荐做法:
- 使用CDN链接而非Base64
- 视频文件≤50MB
- 格式: MP4 (H.264编码)
- 分辨率: 1920x1080或1280x720
- 缩略图: 1280x720, JPEG格式,压缩后<500KB

❌ 避免做法:
- 直接上传>100MB的视频(超LocalStorage限制)
- 使用非标准格式(如AVI、FLV)
- 不设置缩略图
- 不填写时长
```

### 内容组织建议
```
✅ 推荐做法:
- 一级分类: 技术大类(如协作机器人)
- 二级分类: 具体主题(如编程与调试)
- 课程: 某个知识点(如"Lua脚本编程入门")
- 标签: 辅助分类(如"入门"、"实战"、"案例")

❌ 避免做法:
- 分类层级过深
- 分类名称过长
- 课程重复
- 标签混乱
```

### 外部链接管理
```
✅ 推荐做法:
- 定期检查链接有效性
- 标注是否需要会员
- 明确标注语言(中文/英文)
- 使用合适的图标和类型

❌ 避免做法:
- 添加失效链接
- 描述不清晰
- 类型选择错误
```
