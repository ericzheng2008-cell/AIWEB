## 🔧 问题排查流程

```
遇到问题
   ↓
1. 确认聊天窗口已打开？
   ├─ 否 → 点击右下角 AI 助手按钮
   └─ 是 → 继续
   ↓
2. 能看到输入框？
   ├─ 否 → 运行诊断工具修复
   └─ 是 → 继续
   ↓
3. 输入框是灰色/禁用状态？
   ├─ 是 → 等待AI输入完成或停止录音
   └─ 否 → 继续
   ↓
4. 能输入文字但不能发送？
   ├─ 是 → 检查Enter键或发送按钮
   └─ 否 → 继续
   ↓
5. 仍有问题？
   └─ 查看完整文档或重启服务器
```
