## 🎯 使用步骤

### 1. 准备 Excel 文件
- 打开 Excel 或 WPS
- 第一行必须是**表头**(完全按照上表列名)
- 从第二行开始填写数据

### 2. 必填字段
- **品牌**
- **工具类型**
- **四方尺寸**
- **套筒型号**
- **套筒名称**

### 3. 导入到系统
1. 进入后台管理 → 套筒数据库
2. 点击 **"Excel批量导入"** 按钮
3. 选择准备好的 Excel 文件
4. 系统自动解析并导入

### 4. 导入结果
- 系统会显示成功导入的条数
- 如果某行数据格式错误会跳过,并显示失败条数
- 导入后可在表格中查看数据

---

## ⚠️ 注意事项

1. **表头必须完全一致**,包括中文字符和顺序
2. **是否抗振** 和 **配密封圈销子** 只能填 "是" 或 "否"
3. **应用场景** 如果有多个,用**逗号分隔**(不要用分号或其他符号)
4. **价格** 和 **库存** 必须是数字
5. 如果某列不需要,可以留空,但**不要删除该列**
6. 保存为 `.xlsx` 或 `.xls` 格式
