# ✅ Vue 模板引号问题已修复

**修复时间**: 2025-12-15  
**问题类型**: Vue 模板语法错误  
**影响文件**: `src/views/admin/LearningEngine.vue`

## 问题描述

在 Vue 模板中使用了中文引号（""），导致编译错误：

```
Attribute name cannot contain U+0022 ("), U+0027 ('), and U+003C (<).
```

**错误位置**: 第328行
```vue
<el-empty v-else description="点击"评估所有知识质量"按钮开始评估" />
```

## 问题原因

Vue 模板的属性值必须使用英文引号（`"` 或 `'`），不能使用中文引号（`""`）。

在属性值内部如果需要引号，应该：
- 外层用双引号，内层用单引号
- 或者外层用单引号，内层用双引号

## 解决方案

将中文引号改为英文单引号：

```vue
<!-- 修复前 -->
<el-empty v-else description="点击"评估所有知识质量"按钮开始评估" />

<!-- 修复后 -->
<el-empty v-else description="点击'评估所有知识质量'按钮开始评估" />
```

## 修复内容

### 文件: `src/views/admin/LearningEngine.vue`

- **第328行**: 将 `description` 属性中的中文引号替换为英文单引号

## 最佳实践

### 在 Vue 模板中使用引号的规范：

1. **属性值使用英文引号**
   ```vue
   ✅ <el-button type="primary">按钮</el-button>
   ❌ <el-button type="primary">按钮</el-button>
   ```

2. **属性值内需要引号时使用单引号**
   ```vue
   ✅ <el-empty description="点击'开始'按钮" />
   ❌ <el-empty description="点击"开始"按钮" />
   ```

3. **或使用单引号包裹，内部用双引号**
   ```vue
   ✅ <el-empty description='点击"开始"按钮' />
   ```

4. **使用模板字符串时注意转义**
   ```vue
   ✅ <el-empty :description="`点击「开始」按钮`" />
   ```
