# 📚 产品技术销售小课堂

一个专业的工业自动化技术知识分享平台，支持多级分类管理和课程发布。

## 🚀 快速开始

### 启动项目
```bash
# 方式1: 使用测试脚本
双击运行: 测试产品技术销售小课堂.bat

# 方式2: 命令行启动
npm run dev
```

### 访问页面
- **前台页面**: http://localhost:5173/tech-classroom
- **后台管理**: http://localhost:5173/admin/classroom

## 📋 功能特点

### ✨ 核心功能
- 🎯 **2级分类体系** - 一级分类（技术领域） + 二级分类（专题）
- 📝 **课程管理** - 完整的CRUD功能，支持图片、标签、难度等级
- 🎨 **精美界面** - 渐变配色、动画效果、响应式布局
- 💾 **数据持久化** - LocalStorage自动保存

### 🗂️ 预设内容
- 9个技术领域（协作机器人、AGV、PLC、拧紧工艺等）
- 36个二级主题
- 完整的分类结构

## 📖 使用指南

### 后台管理
1. 进入管理后台 → 产品技术销售小课堂
2. 管理一级分类、二级分类
3. 创建和发布课程
4. 查看数据统计

### 前台展示
1. 浏览技术分类
2. 选择感兴趣的主题
3. 查看课程列表
4. 学习课程内容，点赞支持

## 🛠️ 技术栈

- Vue 3 (Composition API)
- Pinia (状态管理)
- Element Plus (UI组件)
- Vue Router (路由)
- AOS (滚动动画)
- LocalStorage (数据持久化)

## 📊 项目结构

```
src/
├── store/
│   └── classroom.js              # Pinia Store
├── views/
│   ├── TechClassroom.vue        # 前台页面
│   └── admin/
│       └── ClassroomManage.vue  # 后台管理
└── router/
    └── index.js                  # 路由配置
```
