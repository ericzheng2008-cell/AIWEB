## 🔧 技术特性

### CSS 3D Transform
```scss
perspective: 1000px;              // 父容器透视
transform-style: preserve-3d;     // 保持3D空间
transform: translateZ(20px);      // Z轴移动
transform: rotateX(5deg);         // X轴旋转
```

### GPU硬件加速
```scss
transform: translateZ(0);         // 强制GPU加速
will-change: transform, opacity;  // 性能提示
```

### 弹性缓动曲线
```scss
transition: all 0.8s cubic-bezier(0.34, 1.56, 0.64, 1);
                     ↑ 超出范围的弹性效果
```

### 毛玻璃效果
```scss
backdrop-filter: blur(20px);      // 背景模糊
background: rgba(255, 255, 255, 0.08);  // 半透明
```

## 🐛 常见问题

### Q1: 页面白屏或加载缓慢
**原因**: 浏览器性能不足或缓存问题

**解决**:
1. 按Ctrl+Shift+R强制刷新
2. 清除浏览器缓存
3. 使用Chrome或Edge最新版本
4. 关闭其他占用资源的程序

### Q2: 动画卡顿不流畅
**原因**: GPU性能不足

**解决**:
1. 关闭其他占用GPU的程序
2. 更新显卡驱动
3. 降低屏幕分辨率
4. 减少粒子数量(修改代码)

### Q3: 3D效果不明显
**原因**: 显示器或浏览器设置

**解决**:
1. 放大浏览器窗口(>1200px宽度)
2. 启用浏览器硬件加速
3. 检查显卡是否启用

### Q4: 控制台报错
**原因**: 数据store未初始化

**解决**:
1. 刷新页面
2. 检查localStorage是否启用
3. 查看浏览器控制台具体错误

### Q5: 端口连接被拒绝
**原因**: 访问了错误端口或服务器未启动

**解决**:
1. 使用快速访问脚本: `⚡_快速访问3D驾驶舱.bat`
2. 确认服务器运行在3002端口
3. 手动访问: http://localhost:3002/tightening-dashboard
