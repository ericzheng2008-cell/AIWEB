# 快速入门 - 品牌型号匹配系统

**日期**: 2025-12-13  
**版本**: v1.0.0

## 🎯 系统简介

品牌型号匹配系统帮助您快速找到最合适的工具品牌和型号。无论是精密装配还是重型装配，系统都能为您推荐最佳工具。

## 🚀 快速开始（用户端）

### 第一步：访问匹配页面

**方式1 - 通过AI智能体**：
1. 访问首页
2. 点击"AI智能体"
3. 选择"拧紧工具选型与工艺改进"
4. 点击"品牌型号匹配" → "立即使用"

**方式2 - 直接访问**：
```
http://localhost:3003/tool-brand-match
```

### 第二步：配置匹配条件

#### 基础配置
```
工具品牌：EQTCF 或 博世
工具类型：锂电池 或 有线电动
控制类型：离合器、油压脉冲、电脉冲、直驱
扭矩范围：选择合适的范围
```

#### 精确配置（可选）
```
需求扭矩：输入具体值（如：10Nm）
便携性要求：不限 / 便携 / 固定
数字化要求：不限 / 数字化 / 传统型
```

### 第三步：查看匹配结果

1. 点击 **"智能匹配"** 按钮
2. 系统显示所有匹配的工具
3. 第一个是 **"最佳推荐"**（绿色高亮）
4. 每个工具显示：
   - 型号和名称
   - 品牌和类型
   - 控制方式
   - 扭矩范围
   - 规格参数
   - 特性标签
   - 应用场景
   - 价格和库存

### 第四步：选择工具

- 查看"最佳推荐"工具（评分最高）
- 对比其他匹配工具
- 点击 **"咨询采购"** 联系销售
- 点击 **"生成匹配报告"** 下载详细报告

## 🎓 使用技巧

### 技巧1：使用"需求扭矩"精确匹配
```
不填写：显示所有该范围的工具
填写具体值：只显示能满足该扭矩的工具
```

### 技巧2：利用便携性筛选
```
不限：显示所有工具
便携：只显示锂电池工具
固定：只显示有线电动工具
```

### 技巧3：数字化筛选
```
不限：显示所有工具
数字化：只显示带数字显示的工具
传统型：只显示传统控制工具
```

### 技巧4：查看详细规格
每个工具卡片显示：
- 供电方式和电压
- 重量和转速
- 精确扭矩值
- 特性标签
- 应用场景

### 技巧5：生成报告
点击"生成匹配报告"自动下载包含：
- 完整匹配条件
- 所有推荐工具
- 详细规格参数
- 应用场景说明

## 🔧 管理员操作

### 登录管理后台

1. 访问：`http://localhost:3003/admin/login`
2. 输入管理员账号密码
3. 点击左侧菜单 **"品牌型号数据库"**

### 添加新工具

1. 点击右上角 **"添加工具"** 按钮
2. 填写基本信息：
   ```
   品牌：EQTCF / 博世
   工具型号：如 ETBP10-42
   工具名称：完整名称
   描述说明：详细功能
   ```

3. 填写工具参数：
   ```
   工具类型：锂电池 / 有线电动
   控制类型：离合器 / 油压脉冲 / 电脉冲 / 直驱
   扭矩范围：选择标准范围
   ```

4. 填写详细规格：
   ```
   供电方式：锂电池 / 有线电动
   最小扭矩：数值
   最大扭矩：数值
   重量：如 1.2kg
   转速：如 0-800 rpm
   电压：如 220V（有线电动需填）
   ```

5. 设置工具特性：
   ```
   ☑ 便携性
   ☑ 可调节
   ☑ 数字化
   ```

6. 选择应用场景（多选）
7. 填写价格和库存
8. 点击 **"确定"** 保存

### 编辑工具

1. 在列表中找到要编辑的工具
2. 点击 **"编辑"** 按钮
3. 修改相应字段
4. 点击 **"确定"** 保存

### 删除工具

1. 点击 **"删除"** 按钮
2. 确认删除操作

### 筛选查询

使用顶部筛选栏：
```
品牌：EQTCF / 博世
工具类型：锂电池 / 有线电动
控制类型：离合器 / 油压脉冲 / 电脉冲 / 直驱
扭矩范围：选择标准范围

```

## ❓ 常见问题

### Q1：如何快速找到合适的工具？
**A**：点击"示例场景"按钮，然后根据实际需求微调参数。

### Q2：扭矩范围和需求扭矩有什么区别？
**A**：
- 扭矩范围：预设的标准范围，用于快速筛选
- 需求扭矩：精确的扭矩值，用于精确匹配

### Q3：为什么没有匹配结果？
**A**：条件可能过于严格，建议：
1. 不填写"需求扭矩"
2. 便携性和数字化选择"不限"
3. 放宽扭矩范围

### Q4：如何选择控制类型？
**A**：
- **离合器**：精密装配（1-5Nm）
- **油压脉冲**：重型装配（5-30Nm）
- **电脉冲**：智能装配（6-35Nm）
- **直驱**：快速装配（5-45Nm）

### Q5：便携式和固定式如何选择？
**A**：
- **便携式（锂电池）**：移动作业、多工位
- **固定式（有线电动）**：固定工位、长时间作业
