## 🎯 AI赢率计算模型(Salesforce Einstein级)

### 公式
```
WinRate = BaseProb(stage) 
        + Σ(RelationshipEvents × 0.25)
        + Σ(SentimentEvents × 0.18)
        + Σ(CompetitorEvents × -0.15)
        + Σ(ProcessEvents × 0.12)
        + Σ(ChampionPower × 0.15)
        + Σ(DecisionMakerEvents × 0.15)
```

### 8大影响维度

1. **客户关系强度** (权重25%)
   - 互动频率
   - 主动度
   - 响应速度

2. **客户情绪** (权重18%)
   - NLP情感分析
   - 正向/负向反馈

3. **竞争对手压力** (权重-15%)
   - 提及次数
   - 竞品活动

4. **内部流程** (权重12%)
   - 审批进度
   - 流程复杂度

5. **黑马推动者** (权重15%)
   - Champion支持度
   - 内部推动力

6. **决策者参与** (权重15%)
   - 高层关注度
   - 决策链完整度

7. **产品契合度** (权重5%)
   - 需求匹配度
   - 定制需求

8. **价格敏感度** (权重-5%)
   - 价格谈判
   - 预算压力

### AI模型(轻量级)
```javascript
// 赢率预测引擎
const predictWinRate = (opportunity) => {
  const baseProb = getStageBaseProb(opportunity.stage)
  const events = opportunity.events
  
  const relationshipScore = calcRelationshipScore(events)
  const sentimentScore = calcSentimentScore(events)
  const competitorImpact = calcCompetitorImpact(events)
  const processScore = calcProcessScore(events)
  const championScore = calcChampionScore(events)
  
  return Math.min(95, Math.max(5,
    baseProb +
    relationshipScore * 0.25 +
    sentimentScore * 0.18 +
    competitorImpact * -0.15 +
    processScore * 0.12 +
    championScore * 0.30
  ))
}
```
