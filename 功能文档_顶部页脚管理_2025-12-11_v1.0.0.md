# 功能文档：顶部和页脚内容管理

## 📋 功能概述

实现网站顶部联系信息和页脚内容的完整CMS管理功能，管理员可以通过后台修改所有显示内容。

## ✨ 功能特性

### 1. 顶部联系信息管理
- **服务热线**：可修改显示在网页最顶部的服务电话
- **邮箱地址**：可修改显示在网页最顶部的联系邮箱
- **实时生效**：修改后立即在Header显示

### 2. 页脚公司信息管理
- **公司名称**：支持中英文分别设置
- **公司描述**：支持中英文长文本描述
- **资质认证**：可添加/删除认证标签（如ISO9000、QS9000、IATF16949）

### 3. 页脚联系方式管理
- **联系电话**：页脚显示的联系电话
- **联系邮箱**：页脚显示的联系邮箱
- **公司地址**：支持中英文分别设置

### 4. 服务网络城市管理
- **中文城市列表**：用 · 分隔的城市名称
- **英文城市列表**：英文版本的城市列表
- **示例**：广州 · 长沙 · 株洲 · 柳州 · 武汉 · 杭州 · 上海

### 5. 版权信息管理
- **中文版权**：中文版权声明和备案号
- **英文版权**：英文版权声明

## 🎯 使用指南

### 访问编辑器
1. 登录CMS管理后台
2. 点击 **"顶部和页脚"** 标签页

### 修改顶部联系信息
1. 在"顶部联系信息"卡片中修改
   - 服务热线：如 `400-123-4567`
   - 邮箱地址：如 `sales@mingsheng.com`
2. 点击"保存顶部联系信息"按钮

### 修改页脚内容
1. **公司信息**：
   - 中文公司名：明升伟业
   - 英文公司名：Mingsheng
   - 中英文公司描述
   - 资质认证：输入认证名称后按回车或点击"添加"

2. **联系方式**：
   - 联系电话、邮箱、地址（中英文）

3. **服务网络**：
   - 输入城市名称，用 · 分隔

4. **版权信息**：
   - 中英文版权声明

5. 点击"保存所有页脚配置"按钮

## 🔧 技术实现

### 涉及文件
1. **Store**: `src/store/cmsAdvanced.js` - 数据管理
2. **Header**: `src/components/Header.vue` - 顶部显示
3. **Footer**: `src/components/Footer.vue` - 页脚显示
4. **编辑器**: `src/components/HeaderFooterEditor.vue` - CMS编辑界面
5. **CMS**: `src/views/admin/CmsManage.vue` - 后台管理页面

### 数据持久化
- 使用 `localStorage` 存储
- 键名：`siteConfig`（顶部）、`footerConfig`（页脚）

### 国际化支持
- Footer自动根据当前语言切换内容
- 支持中文/英文双语显示

## 📝 注意事项

1. **服务网络城市**：使用 `·` (中文圆点) 分隔，不是英文句点
2. **备案号**：确保填写正确的ICP备案号
3. **邮箱格式**：确保邮箱格式正确
4. **双语一致性**：中英文内容应保持语义一致
