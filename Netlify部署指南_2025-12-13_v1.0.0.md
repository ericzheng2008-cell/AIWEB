## 📋 部署流程

### 步骤 1/5：检查 Netlify CLI
```
检查是否已安装 netlify-cli
如未安装，自动安装
```

### 步骤 2/5：清理旧构建
```
删除 dist 目录
清理 Vite 缓存
```

### 步骤 3/5：构建项目
```
npm run build
需要 1-2 分钟
```

### 步骤 4/5：登录 Netlify
```
浏览器会自动打开
使用 GitHub/Email 登录
完全免费，无需信用卡
```

### 步骤 5/5：部署
```
上传 dist 文件到 Netlify
自动生成域名
```

## 🔄 更新网站

每次修改代码后，重新部署：

### 方法1：使用脚本
```bash
deploy-to-netlify.bat
```

### 方法2：手动命令
```bash
npm run build
netlify deploy --prod --dir=dist
```

### 方法3：连接 GitHub（推荐）
1. 登录 Netlify 控制台
2. 连接 GitHub 仓库
3. 每次 push 自动部署

## ⚠️ 常见问题

### Q1: 浏览器自动打开但无法登录
**A:** 
1. 手动访问：https://app.netlify.com
2. 使用 GitHub 账号登录
3. 返回命令行继续

### Q2: 部署成功但网站打不开
**A:**
1. 等待 1-2 分钟（DNS 生效）
2. 清除浏览器缓存
3. 使用无痕模式访问

### Q3: 构建失败
**A:**
1. 检查是否已安装依赖：`npm install`
2. 本地测试：`npm run build`
3. 查看错误日志

### Q4: 想要更改域名
**A:**
1. 登录 Netlify 控制台
2. Site settings → Site details → Change site name
3. 输入新名称（如：my-website）
4. 新地址：`https://my-website.netlify.app`
