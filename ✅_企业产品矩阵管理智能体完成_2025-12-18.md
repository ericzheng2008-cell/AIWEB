## 二、功能特性

### 🎯 核心功能

#### 1. 产品收益-现金流战略矩阵
- **四象限分析**: 核心战略、优化提升、低投入维持、潜力产品
- **气泡可视化**: 
  - X轴: 毛利率 (0-50%)
  - Y轴: 现金流贡献率 (0-30%)
  - 气泡大小: 销售额
  - 气泡颜色: 战略分类渐变色
- **交互功能**:
  - 点击气泡查看详情
  - 悬停显示快速信息
  - 点击象限过滤产品

#### 2. AI智能推荐系统

**🎯 战略建议**
- 强化核心产品
- 优化现金流
- 优化产品组合

**⚡ 运营优化**
- 库存优化（针对高库存周转产品）
- 回款加速（针对长回款周期产品）

**⚠️ 异常预警**
- 毛利率过低预警（<15%）
- 现金流贡献低预警（<10%）
- 库存积压预警（>90天）
- 回款超期预警（>60天）

#### 3. 大类对比分析
- **3种对比维度**: 销售额、毛利、现金流
- **可视化展示**: 水平条形图 + 渐变色
- **数据指标**: 百分比、产品数量、趋势变化
- **钻取功能**: 点击大类查看详细产品

#### 4. 单产品详情分析
- **基础信息**: 8个关键指标
- **销售趋势**: 6个月趋势柱状图
- **AI推荐**: 基于象限的优化动作
- **报告导出**: TXT格式产品报告

### 🔑 核心算法

#### 1. 象限自动计算
```javascript
determineQuadrant(grossMargin, cashFlowRate) {
  const marginThreshold = 25  // 毛利率分界线
  const cashThreshold = 15    // 现金流分界线
  
  if (grossMargin >= marginThreshold && cashFlowRate >= cashThreshold) {
    return 'core'        // 核心战略
  } else if (grossMargin >= marginThreshold) {
    return 'optimize'    // 优化提升
  } else if (cashFlowRate >= cashThreshold) {
    return 'potential'   // 潜力产品
  } else {
    return 'maintain'    // 低投入维持
  }
}
```

#### 2. 气泡位置计算
```javascript
// X轴位置 (基于毛利率)
```
