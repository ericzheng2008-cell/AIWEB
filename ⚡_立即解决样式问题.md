## 🔧 开发者检查（可选）

想确认样式是否真的加载了？

1. 按 `F12` 打开开发者工具
2. 切换到 **Elements** 标签
3. 找到 `.chat-button` 元素
4. 查看 **Computed** 面板中的 `background-image`

**应该看到：**
```css
linear-gradient(135deg, rgb(102, 126, 234) 0%, rgb(118, 75, 162) 100%)
```

**如果看到的是：**
```css
linear-gradient(135deg, rgb(24, 144, 255) 0%, rgb(0, 102, 204) 100%)
```
说明还在使用旧缓存！
