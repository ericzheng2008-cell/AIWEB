## 📝 详细操作指南

### 一、编辑页面横幅

#### 1.1 编辑横幅文字

**步骤：**
1. 点击 **"📸 页面横幅"** 标签
2. 在 **"中文内容"** 区域编辑：
   - **标题**：例如 "产品与服务"
   - **副标题**：例如 "专业的工业自动化设备与智能制造解决方案"

3. 在 **"English Content"** 区域编辑英文版本

4. 点击右上角 **"保存横幅"** 按钮

**示例：**
```
中文标题：工业工具 · 智能制造
中文副标题：28年专注高端工业工具供应与应用解决方案

英文标题：Industrial Tools · Smart Manufacturing
英文副标题：28 Years Focus on High-end Industrial Tool Supply and Application Solutions
```

#### 1.2 设置横幅背景

**上传背景图片：**
1. 在 **"背景图片"** 区域点击上传按钮
2. 选择图片文件（推荐尺寸：1920x600px）
3. 或者输入图片URL

**设置背景颜色：**
1. 点击 **"背景颜色"** 色块
2. 选择颜色或输入色值
3. 支持渐变色，例如：`linear-gradient(135deg, #003366 0%, #0066cc 100%)`

**设置文字颜色：**
1. 点击 **"文字颜色"** 色块
2. 选择颜色（通常用白色 #ffffff）

#### 1.3 预览效果

- 编辑完成后，页面下方会实时显示预览效果
- 确认无误后点击 **"保存横幅"**

---

### 二、管理产品分类卡片

#### 2.1 添加新分类

**步骤：**
1. 点击 **"📦 产品分类卡片"** 标签
2. 点击 **"添加分类"** 按钮
3. 填写分类信息：

**中文内容：**
- **分类名称**：例如 "电动工具"
- **分类描述**：例如 "高性能电动拧紧工具，适用于汽车装配、机械制造等领域"

**英文内容：**
- **Category Name**: "Electric Tools"
- **Description**: "High-performance electric tightening tools for automotive assembly and machinery manufacturing"

**上传图片：**
- 推荐尺寸：600x450px
- 支持 JPG/PNG 格式

**设置排序：**
- 输入排序数字（数字越小越靠前）
- 例如：1, 2, 3, 4...

4. 点击 **"保存"** 按钮

#### 2.2 编辑现有分类

**步骤：**
1. 在分类卡片列表中找到要编辑的分类
2. 鼠标悬停在卡片上，点击 **"编辑"** 按钮
3. 修改信息
4. 点击 **"保存"** 按钮

#### 2.3 删除分类

**步骤：**
1. 鼠标悬停在要删除的分类卡片上
2. 点击 **"删除"** 按钮
3. 确认删除操作

**⚠️ 注意：** 删除分类后，该分类下的所有子分类和产品也将被删除，请谨慎操作！

---

### 三、页面布局设置

#### 3.1 调整卡片列数

**步骤：**
1. 点击 **"⚙️ 页面设置"** 标签
2. 在 **"每行显示卡片数"** 选择：
   - **2列**：适合大尺寸卡片展示
   - **3列**：默认推荐（平衡美观与信息量）
   - **4列**：适合展示更多分类

#### 3.2 自定义卡片样式

**卡片圆角：**
- 拖动滑块调整圆角大小（0-20px）
- 0px = 方形，20px = 圆角很大

**卡片阴影：**
- 开启：卡片有阴影效果，更有层次感
- 关闭：卡片无阴影，扁平风格

**悬停动画：**
- 开启：鼠标悬停时卡片上浮效果
- 关闭：无动画效果

**查看详情链接：**
- 开启：卡片底部显示"查看详情"链接
- 关闭：不显示链接

#### 3.3 保存设置

修改完成后，点击右上角 **"保存设置"** 按钮。

## 💡 常见问题

### Q1: 修改后前台页面没有更新？
**解决方法：**
1. 刷新前台页面（按 F5）
2. 如果还不行，清除浏览器缓存（Ctrl+F5）

### Q2: 上传的图片不显示？
**检查项：**
- 确认图片格式为 JPG 或 PNG
- 确认图片大小不超过 5MB
- 确认图片URL可访问（如果使用URL方式）

### Q3: 如何恢复默认设置？
**方法：**
1. 打开浏览器开发者工具（F12）
2. 在 Console 中执行：
   ```javascript
   localStorage.removeItem('productsServicesPageSettings')
   ```
3. 刷新页面

### Q4: 分类排序不生效？
**解决方法：**
- 确保每个分类都设置了不同的排序数字
- 保存后刷新前台页面查看效果

## 📊 数据存储说明

### 存储位置

所有编辑的内容存储在浏览器的 `localStorage` 中：

```javascript
// 页面设置
localStorage.getItem('productsServicesPageSettings')

// 产品分类（通过 CMS Store）
cmsStore.productCategories
```

### 备份数据

**导出当前设置：**
```javascript
// 在浏览器控制台执行
const settings = localStorage.getItem('productsServicesPageSettings')
console.log(settings)
// 复制输出的内容保存到文件
```

**恢复设置：**
```javascript
// 将之前导出的内容粘贴进去
const settings = '...'  // 粘贴之前导出的内容
localStorage.setItem('productsServicesPageSettings', settings)
location.reload()
```
