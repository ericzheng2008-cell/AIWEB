## Phase 1: 智能体注册中心

### 功能概述

智能体注册中心是所有智能体的统一管理入口，提供：

- ✅ 智能体注册与配置
- ✅ 状态管理（启用/禁用）
- ✅ 能力定义与管理
- ✅ 版本控制
- ✅ 搜索与筛选

### 界面说明

#### 1. 统计卡片区

位于页面顶部，显示4个关键指标：

| 指标 | 说明 |
|------|------|
| 总智能体数 | 系统中注册的所有智能体总数 |
| 已启用 | 当前处于启用状态的智能体数量 |
| 已禁用 | 当前处于禁用状态的智能体数量 |
| 平均能力数 | 每个智能体的平均能力数量 |

#### 2. 搜索筛选区

提供以下筛选条件：

- **搜索框**: 按名称或描述搜索智能体
- **状态筛选**: 全部 / 已启用 / 已禁用
- **分类筛选**: 按智能体类型筛选

#### 3. 操作按钮

| 按钮 | 功能 |
|------|------|
| + 注册新智能体 | 打开注册对话框，添加新智能体 |
| 刷新 | 重新加载智能体列表 |

#### 4. 智能体列表表格

显示所有智能体，包含以下列：

| 列名 | 说明 |
|------|------|
| ID | 智能体唯一标识 |
| 名称 | 智能体显示名称 |
| 描述 | 功能描述 |
| 类型 | 智能体分类 |
| 版本 | 当前版本号 |
| 状态 | 启用/禁用标签 |
| 能力数 | 该智能体拥有的能力数量 |
| 创建时间 | 注册时间 |
| 操作 | 查看详情/编辑/删除按钮 |

### 编辑功能

#### 注册新智能体

**步骤**：
1. 点击"+ 注册新智能体"按钮
2. 填写表单：
   - **智能体ID**: 唯一标识（如 `agent-001`）
   - **名称**: 显示名称（如 "拧紧工艺智能体"）
   - **类型**: 选择分类
   - **描述**: 功能说明
   - **版本号**: 遵循语义化版本（如 `1.0.0`）
   - **能力列表**: 添加智能体的能力项
3. 点击"确定"保存

#### 编辑智能体

**步骤**：
1. 在表格中找到目标智能体
2. 点击"编辑"按钮
3. 修改相关字段
4. 点击"确定"保存

#### 查看详情

**步骤**：
1. 点击"查看详情"按钮
2. 在对话框中查看完整信息：
   - 基本信息
   - 能力列表
   - 版本历史
   - 使用统计

#### 启用/禁用

**方式1**: 点击状态标签切换
**方式2**: 在编辑对话框中修改状态

#### 删除智能体

**步骤**：
1. 点击"删除"按钮
2. 确认删除操作
3. 系统从注册中心移除该智能体

## Phase 2: 知识库管理系统

### 功能概述

知识库管理系统提供：

- ✅ 知识条目管理
- ✅ Markdown 编辑与预览
- ✅ 分类与标签管理
- ✅ 知识关联
- ✅ 版本历史
- ✅ 知识图谱可视化

### 界面说明

#### 1. 统计卡片区

| 指标 | 说明 |
|------|------|
| 知识总量 | 知识库中的条目总数 |
| 分类数量 | 知识分类的总数 |
| 标签数量 | 使用的标签总数 |
| 平均质量分 | 知识条目的平均质量得分 |

#### 2. 搜索筛选区

- **搜索框**: 按标题或内容搜索
- **分类筛选**: 按分类过滤
- **标签筛选**: 按标签过滤
- **质量分筛选**: 按质量得分范围过滤

#### 3. 标签页

| 标签页 | 功能 |
|--------|------|
| 知识列表 | 显示所有知识条目 |
| 分类管理 | 管理知识分类 |
| 标签管理 | 管理标签 |
| 知识图谱 | 可视化知识关联 |

### 编辑功能

#### 添加知识

**步骤**：
1. 点击"+ 添加知识"按钮
2. 填写表单：
   - **标题**: 知识条目标题
   - **分类**: 选择或新建分类
   - **标签**: 添加标签（支持多选）
   - **内容**: 使用 Markdown 编辑器编写
   - **相关智能体**: 关联到智能体
   - **质量分**: 0-100 的评分
3. 实时预览 Markdown 渲染效果
4. 点击"确定"保存

#### 编辑知识

**步骤**：
1. 在知识列表中找到目标条目
2. 点击"编辑"按钮
3. 修改内容
4. 保存后自动创建新版本

#### 查看版本历史

**步骤**：
1. 点击"版本历史"按钮
2. 查看所有历史版本
3. 可以对比不同版本
4. 可以回滚到历史版本

#### 管理分类

**在"分类管理"标签页**：
1. 查看所有分类
2. 添加新分类
3. 编辑分类名称
4. 删除分类（需确认无知识使用）
5. 查看每个分类的知识数量

#### 管理标签

**在"标签管理"标签页**：
1. 查看所有标签及使用次数
2. 添加新标签
3. 合并重复标签
4. 删除未使用的标签

#### 知识图谱

**在"知识图谱"标签页**：
1. 可视化展示知识之间的关联
2. 点击节点查看详情
3. 拖拽调整布局
4. 搜索定位特定知识

2. 点击"评估所有知识质量"按钮
3. 系统自动评估所有知识
4. 查看评估报告
5. 针对低质量知识采取优化措施

#### 配置学习参数

**在"学习配置"标签页**：
1. 设置反馈阈值
2. 配置学习频率

## 常见问题

### Q1: 页面显示空白怎么办？

**解决方案**：
1. 按 `F12` 打开控制台
2. 执行：
   ```javascript
   localStorage.clear();
   sessionStorage.clear();
   location.reload(true);
   ```
3. 或按 `Ctrl + Shift + R` 硬刷新

### Q2: 数据没有保存怎么办？

**检查**：
1. 确认点击了"确定"或"保存"按钮
2. 检查浏览器控制台是否有错误
3. 确认 LocalStorage 未被禁用
4. 尝试清除缓存后重试

### Q3: 按钮点击没有反应？

**排查步骤**：
1. 检查浏览器控制台错误信息
2. 确认开发服务器正在运行
3. 尝试刷新页面
4. 检查网络连接

### Q4: 如何重置演示数据？

**步骤**：
1. 按 `F12` 打开控制台
2. 执行：
   ```javascript
   localStorage.clear();
   location.reload();
   ```
3. 页面会自动重新初始化演示数据

### Q5: 编辑对话框不显示？

**解决方案**：
1. 检查浏览器缩放比例（建议100%）
2. 检查浏览器兼容性（推荐 Chrome/Edge）
3. 清除浏览器缓存
4. 查看控制台错误信息
