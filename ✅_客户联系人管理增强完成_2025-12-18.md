## 🚀 使用指南

### 前台访问路径
```
首页 → 营销板块 → 明升AICRM → 客户360°
```

### 后台管理路径
```
http://localhost:3004/admin/customer360
```

### 功能操作流程

#### 1. 查看客户360画像
1. 进入AICRM → 客户360°
2. 选择要查看的客户
3. 查看8大维度数据
4. 查看健康分和风险预警
5. 切换到沙盘分析标签

#### 2. 使用沙盘分析
1. 在客户360页面点击"沙盘分析"标签
2. 选择预测时间范围
3. 选择要模拟的场景
4. 查看营收预测和风险分析
5. 查看AI推荐的应对策略

#### 3. 后台配置
1. 进入后台管理 → 客户360画像
2. 配置数据源同步选项
3. 设置AI分析参数
4. 配置沙盘分析参数
5. 点击"同步所有数据"更新数据
