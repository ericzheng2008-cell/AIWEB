## 📞 如果遇到问题

### ERR_CONNECTION_REFUSED
**原因：** 服务器未运行  
**解决：** 运行 `npm run dev`

### 端口被占用
```bash
# 查看占用端口的进程
netstat -ano | findstr :3002

# 杀死进程
taskkill /F /PID <进程ID>
```

### 页面空白
**解决：** Ctrl+Shift+Delete 清除缓存，然后 Ctrl+F5
