# 快速入门 - 拧紧工艺改进与验证智能体

**版本**: v1.0.0

---

## 🚀 5分钟快速上手

### 步骤1: 访问智能体 (30秒)

1. 打开浏览器访问系统首页
2. 点击顶部导航栏"AI智能体"
3. 找到"拧紧工艺改进与验证智能体"卡片(带"新功能"标签)
4. 点击"立即使用"按钮

**提示**: 也可以直接访问 `/process-verification` 路径

---

### 步骤2: 填写工艺条件 (2分钟)

#### 必填项

**1. 工位名称** (多选)
```
示例: 选择"轮胎工位"
可同时选择多个工位进行批量分析
```

**2. 工位扭矩** (Nm)
```
示例: 输入 45
范围: 1-500Nm，步进0.5Nm
```

**3. 工具类型**
```
示例: 选择"锂电手持油压脉冲定扭扳手"
根据扭矩范围选择合适工具
```

#### 选填项

**4. 工具型号** (推荐填写)
```
示例: ETBP45-10 (45Nm)
选择工具类型后可选
```

**5. 螺栓类型** (推荐填写)
```
示例: "标准机螺栓"
常用: 标准机螺栓、长螺栓
```

**6. 连接件弹性** (默认:中性)
```
选项: 偏软 | 中性 | 偏硬
有橡胶垫片选"偏软"，刚性连接选"偏硬"
```

**7. 工件材料** (推荐填写)
```
示例: "钢对钢"
常用: 钢对钢、钢铝、铝合金
```

**8. 特殊工况** (如有)
```
可选: 大扭矩拧紧、软连接、长螺栓拧紧等
```

**9. 测试类型** (默认:动态扭矩测试)
```
动态扭矩测试: 拧紧过程中测量
静态扭矩测试: 拧紧完成后测量
```

---

### 步骤3: 智能分析 (1分钟)

1. 检查必填项是否已填写
2. 点击右上角蓝色"智能分析"按钮
3. 等待1-2秒，系统自动生成推荐结果
4. 页面自动滚动到"智能分析结果"区域

**成功提示**: 
```
✅ 分析完成！已为您生成专业工艺建议
```

## 💡 典型应用示例

### 示例1: 标准刚性连接

**场景**: 轮胎工位，M12螺栓，钢对钢连接

**输入**:
```
工位名称: 轮胎工位
工位扭矩: 45 Nm
工具类型: 锂电手持油压脉冲定扭扳手
工具型号: ETBP45-10
螺栓类型: 标准机螺栓
连接件弹性: 偏硬
工件材料: 钢对钢
```

**推荐结果**:
- **最优策略**: 扭矩控制 (匹配度95%)
- **关键参数**: 
  - P113: 45 Nm (目标扭矩)
  - P112: 40.5 Nm (最小值)
  - P114: 49.5 Nm (最大值)

**建议**: 
- ✅ 刚性连接适合扭矩控制
- ✅ 参数窗口±10%，确保质量

---

### 示例2: 软连接工位

**场景**: 减震器工位，有橡胶垫片

**输入**:
```
工位名称: 减震器工位
工位扭矩: 55 Nm
工具类型: 锂电离合器定扭扳手
工具型号: ETBC65-12
螺栓类型: 标准机螺栓
连接件弹性: 偏软
工件材料: 双层钢板中间橡胶
特殊工况: 软连接
```

**推荐结果**:
- **最优策略**: 角度控制 (匹配度90%)
- **关键参数**:
  - P120: 38.5 Nm (开始角度)
  - P123: 100° (目标角度)

**特别提醒**:
```
⚠️ 软连接特殊注意 (高优先级)
1. 橡胶压缩会导致扭矩衰减5-10%
2. 建议24小时后复检扭矩
3. 或设置自动补拧流程
```

---

### 示例3: 长螺栓拧紧

**场景**: 飞轮工位，M14×80长螺栓

**输入**:
```
工位名称: 飞轮工位
工位扭矩: 120 Nm
工具类型: 锂电手持油压脉冲定扭扳手
工具型号: ETBP180-20
螺栓类型: 长螺栓
连接件弹性: 中性
工件材料: 钢对钢
特殊工况: 长螺栓拧紧
```

**推荐结果**:
- **最优策略**: DS控制 (匹配度88%)
- **关键参数**:
  - P113: 96 Nm (第一阶段，提高至80%)
  - P123: 120° (角度增加，补偿损失)

**特别提醒**:
```
⚠️ 长螺栓拧紧注意 (高优先级)
1. 长螺栓扭矩损失可达15-20%
2. 采用DS控制，第一阶段扭矩提高
3. 监控角度变化率，防止打滑
```

## ❓ 常见问题

### Q1: 分析按钮点击后没有反应?

**检查项**:
- ✅ 工位名称是否已选择
- ✅ 工位扭矩是否已输入
- ✅ 工具类型是否已选择
- ✅ 网络连接是否正常

**解决方法**:
填写所有必填项后重新点击"智能分析"

---

### Q2: 如何理解匹配度评分?

**评分含义**:
- 90-100%: 非常适合，强烈推荐
- 75-89%: 适合，可以使用
- 60-74%: 可用，但非最优
- <60%: 不推荐

**影响因素**:
- 连接件类型(刚性/软连接)
- 螺栓类型(标准/长螺栓)
- 扭矩大小
- 材料特性

---

### Q3: 推荐参数可以直接使用吗?

**建议流程**:
1. **初次使用**: 按推荐值设置PSE程序
2. **试验验证**: 采集30组数据
3. **Cpk分析**: 计算过程能力指数
4. **参数微调**: 根据Cpk结果微调窗口
5. **正式生产**: 参数固化并监控

**注意事项**:
- 推荐参数是基于理论和经验的起点
- 实际生产中需根据具体情况微调
- 不同批次螺栓可能需要小幅调整

---

### Q4: 角度控制的目标角度如何确定?

**影响因素**:
1. **螺栓直径**: 直径越大，角度越大
2. **螺栓长度**: 长螺栓需要增加角度
3. **连接类型**: 软连接需要增加角度
4. **螺距**: 粗牙螺纹角度较大

**快速参考**:
- M6-M8: 60-80°
- M10-M12: 70-100°
- M14-M16: 80-120°
- 长螺栓: 在标准值基础上+20-30°
- 软连接: 在标准值基础上+10-20°

---

### Q5: DS控制比单一控制好在哪里?

**优势对比**:

| 特性 | 扭矩控制 | 角度控制 | DS控制 |
|-----|---------|---------|--------|
| 预紧力一致性 | 中等 | 好 | 最好 |
| 适应性 | 刚性连接 | 软连接 | 通用 |
| 防护能力 | 扭矩窗口 | 角度窗口 | 双重窗口 |
| Cpk值 | 1.33-1.67 | 1.67-2.0 | 2.0+ |
| 工具要求 | 标准 | 标准 | 需支持双控 |

**推荐场景**:
- 高价值零部件: 发动机、变速箱
- 安全关键件: 底盘、转向
- 高一致性要求: Cpk≥2.0
