## 💡 技术细节

### 组件传递
```vue
<!-- 在MingShengAICRM_V3.vue中 -->
<OpportunityFunnel 
  :opportunity-id="currentOpportunity.id" 
  @update="handleOpportunityUpdate"
/>
```

### 数据更新处理
```javascript
const handleOpportunityUpdate = (updatedData) => {
  // 更新当前商机
  Object.assign(currentOpportunity.value, updatedData)
  
  // 更新列表中的商机
  const index = opportunities.value.findIndex(opp => opp.id === updatedData.id)
  if (index !== -1) {
    opportunities.value[index] = { ...opportunities.value[index], ...updatedData }
  }
}
```
