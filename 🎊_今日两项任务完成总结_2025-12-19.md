### 2️⃣ 设备页面卡片尺寸优化

**优化目标**:
所有卡片缩小15%，改善页面浏览体验

**实现方案**:
```css
/* 12种卡片类型统一缩小 */
.handle-card,
.arch-layer .layer-content,
.widget-card,
.enhanced-pain-card,
.case-card,
.loop-step-card,
.retrofit-card,
.intelligence-card,
.pricing-card,
.hexagon,
.equipment-card,
.el-card {
  transform: scale(0.85);  /* 缩小至85% */
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

/* Hover恢复100% */
.xxx-card:hover {
  transform: scale(1.0);
  z-index: 10;
}
```

**修改文件**:
- `src/views/EquipmentLifecycle.vue` (+50行CSS)

**优化效果**:
| 指标 | 优化前 | 优化后 | 改善 |
|------|--------|--------|------|
| 卡片尺寸 | 100% | 85% | ✅ -15% |
| 单屏显示 | 2-3个 | 3-4个 | ✅ +33% |
| 滚动频率 | 频繁 | 减少 | ✅ 体验提升 |
