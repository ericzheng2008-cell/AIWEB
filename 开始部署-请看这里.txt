================================================================================
                    🔧 配置演示数据（可选）
================================================================================

如果需要让所有访客看到相同的内容（产品、课程等）：

1. 本地运行网站
   打开命令行，运行：npm run dev

2. 在后台配置所有内容
   访问：http://localhost:3002/admin
   添加产品、Banner、课程等

3. 导出数据
   访问：http://localhost:3002/import-demo-data.html
   点击"导出当前数据"，复制代码

4. 更新演示数据文件
   编辑：src/utils/initDemoData.js
   替换 demoData 对象的内容

5. 重新构建和部署
   运行：npm run build
   重新上传 dist 文件夹到 Netlify Drop

⏱️ 总耗时：10-20分钟
