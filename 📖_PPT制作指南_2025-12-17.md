## 📊 关键页面设计要点

### P4：成本冰山（核心视觉）
```
设计要素：
• 使用渐变色区分水上/水下
• 水上30%：浅蓝色背景
• 水下70%：深蓝色背景
• 添加动画：逐层展示隐性成本
```

### P9：五级价值路径（阶梯图）
```
设计要素：
• 使用阶梯式布局
• 每级不同颜色（从浅到深）
• 添加箭头指示进阶方向
• 高亮推荐组合方案
```

### P16：必中算账法（对比表）
```
设计要素：
• 左右分栏对比
• 传统模式：红色系
• 年包模式：绿色系
• 中间大号显示节省金额
```

### P25-P26：成功案例（数据表）
```
设计要素：
• 使用表格清晰展示数据
• 改善指标用绿色箭头
• ROI数字特别放大
• 客户评价使用引用框
```

## 🎬 动画建议

### 入场动画
- **标题**：淡入（0.5秒）
- **要点**：逐条飞入（每条延迟0.3秒）
- **图表**：擦除/缩放（1秒）

### 强调动画
- **关键数字**：脉冲/放大
- **对比数据**：颜色变化
- **结论**：闪烁/高亮

### 切换动画
- **建议**：推进/淡化（统一使用）
- **速度**：中速（0.5-0.7秒）

## 💡 常见问题预案

### Q1："太贵了"
```
翻到P16「必中算账法」
• 指着节省金额："您看，第一年就省回10.5万"
• 指着额外收益："这还不包括停线规避20-30万"
• 结论："实际是赚的，不是花的"
```

### Q2："我们自己能修"
```
翻到P4「成本冰山」
• "自己修只是解决了水上30%"
• "水下70%的隐性成本才是黑洞"
• "我们帮您优化的是看不见的部分"
```

### Q3："和XX公司有什么区别？"
```
翻到P7「角色定义」+ P22「风险划分」
• "他们是项目制，修一次收一次钱"
• "我们是年包制，成本锁定，风险我们承担"
• "您买的不是服务，是确定性"
```

### Q4："我再考虑考虑"
```
翻到P38「下一步行动」
• "理解，这是大决策"
• "我们先做免费诊断，您无需付费"
• "拿到报告再决定，不着急"
• 现场演示Tab 1，把算账结果发微信
```
