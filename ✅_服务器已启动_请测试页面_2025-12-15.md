## 🔍 如果页面空白怎么办

### 方法 1：清除浏览器缓存（推荐）

1. 在页面上按 `F12` 打开开发者工具
2. 在 Console 控制台中执行：

```javascript
localStorage.clear();
sessionStorage.clear();
location.reload(true);
```

### 方法 2：使用隐私模式

1. 按 `Ctrl + Shift + N` (Chrome) 或 `Ctrl + Shift + P` (Firefox)
2. 在隐私窗口中访问页面
3. 这样可以避免缓存问题

### 方法 3：硬刷新

- 按 `Ctrl + Shift + R` 或 `Ctrl + F5`
- 强制浏览器重新加载所有资源

## 🔧 故障排查

### 问题 1：页面显示 404 Not Found

**原因**: 路由不存在  
**解决**: 检查 URL 是否正确，确保访问的是 `/admin/knowledge` 而不是其他路径

### 问题 2：页面空白无内容

**原因**: 数据未初始化或有 JS 错误  
**解决**: 
1. 打开控制台查看错误
2. 清除 localStorage
3. 刷新页面

### 问题 3：页面样式错乱

**原因**: CSS 未加载或缓存问题  
**解决**: 硬刷新浏览器 (Ctrl+Shift+R)

### 问题 4：按钮点击无响应

**原因**: JS 错误或事件未绑定  
**解决**: 查看控制台错误信息
