## 🐛 常见问题排查

### 问题1: 布局错乱

**症状**: 移动端显示多列，按钮重叠

**排查**:
```css
/* 检查是否有强制宽度 */
.el-col {
  width: 100% !important;  /* 应该有这个 */
}
```

**解决**: 清除浏览器缓存，强制刷新 (Ctrl+Shift+R)

### 问题2: 图表不显示

**症状**: 移动端图表区域空白

**排查**:
```javascript
// 检查图表容器高度
console.log(document.getElementById('chart').offsetHeight)
```

**解决**: 确保容器有明确高度，不是0

### 问题3: 按钮太小无法点击

**症状**: 移动端按钮点击困难

**排查**:
```css
/* 检查按钮最小尺寸 */
.el-button {
  min-height: 44px;  /* 应该有这个 */
  min-width: 44px;
}
```

**解决**: 添加响应式CSS规则

### 问题4: 文字太小看不清

**症状**: 移动端文字模糊或太小

**排查**:
```css
/* 检查媒体查询 */
@media (max-width: 767px) {
  .title { font-size: 16px; }  /* 应该有这个 */
}
```

**解决**: 添加移动端字体大小规则

## 🎓 最佳实践建议

### 测试技巧

1. **从小到大测试**
   - 先测小屏手机
   - 再测平板
   - 最后测桌面

2. **横竖屏都要测**
   - 尤其是平板设备
   - 验证布局切换流畅

3. **真机测试必不可少**
   - DevTools只是模拟
   - 真机体验才是真实

4. **不同浏览器都要测**
   - Chrome/Safari/Firefox
   - iOS Safari和Android Chrome
