# 页面内容管理系统使用指南

## 概述

CMS系统已经包含了完整的页面内容管理功能，可以编辑首页、关于我们、服务与支持、联系我们等页面的所有内容。

## 访问入口

### 方式一：通过管理后台菜单
1. 登录管理后台：`http://localhost:3004/admin/login`
2. 点击左侧菜单 **"页面内容管理"**
3. 或直接访问：`http://localhost:3004/admin/page-content`

### 方式二：通过CMS管理页面
1. 访问：`http://localhost:3004/admin/cms`
2. 顶部有导航栏管理和其他功能

## 图片上传功能

### 支持两种方式：

#### 1. 本地上传
- 点击上传区域
- 选择本地图片文件
- 支持格式：JPG, PNG, GIF, WebP
- 自动上传并显示

#### 2. 网络图片URL
- 在输入框中粘贴图片URL
- 按回车确认
- 推荐使用：
  - Unsplash（免费高质量图片）
  - 自己的图床服务

#### 3. 图片替换
- **直接点击图片**即可替换
- 无需删除后重新上传
- 更加便捷快速

### 推荐图片尺寸

| 位置 | 推荐尺寸 | 说明 |
|------|---------|------|
| Banner背景图 | 1920×600px | 首页、内页顶部大图 |
| 分类封面图 | 800×600px | 产品分类、案例分类 |
| 内容配图 | 600×400px | 内容模块配图 |
| Logo | 200×60px | 网站Logo |

## 数据存储

### 存储方式
所有页面内容保存在浏览器的 **localStorage** 中：

| 数据类型 | LocalStorage Key |
|---------|------------------|
| 导航栏配置 | `navItems` |
| 页面内容 | `pageContents` |
| 应用案例分类 | `solutionCategories` |
| 2级分类 | `solutionSubCategories` |
| 3级分类 | `solutionThirdCategories` |
| 具体案例 | `solutionCases` |

### 数据备份
**重要提醒：** 清除浏览器缓存会丢失所有数据！

**备份方法：**
1. 打开浏览器开发者工具（F12）
2. 进入 Console 控制台
3. 运行以下代码导出数据：
   ```javascript
   // 导出所有页面内容
   const backup = {
     navItems: localStorage.getItem('navItems'),
     pageContents: localStorage.getItem('pageContents'),
     solutionCategories: localStorage.getItem('solutionCategories'),
     solutionSubCategories: localStorage.getItem('solutionSubCategories'),
     solutionThirdCategories: localStorage.getItem('solutionThirdCategories'),
     solutionCases: localStorage.getItem('solutionCases')
   }
   console.log(JSON.stringify(backup))
   // 复制控制台输出的内容保存为JSON文件
   ```

**恢复数据：**
```javascript
// 将备份的数据粘贴到这里
const backup = { /* 备份数据 */ }
Object.keys(backup).forEach(key => {
  localStorage.setItem(key, backup[key])
})
location.reload() // 刷新页面
```

## 常见问题

### Q1：为什么我的修改没有生效？
**A：** 
1. 检查是否点击了"保存"按钮
2. 刷新前台页面（Ctrl+F5 强制刷新）
3. 检查浏览器控制台是否有错误

### Q2：图片上传后显示不出来？
**A：**
1. 检查图片URL是否正确
2. 确认图片格式是否支持
3. 网络图片需要支持HTTPS
4. 尝试使用本地上传方式

### Q3：如何修改多个模块的顺序？
**A：**
当前版本需要：
1. 删除需要移动的模块
2. 在目标位置重新添加
3. 后续版本会增加拖拽排序功能

### Q4：可以添加视频吗？
**A：**
当前版本只支持图片，可以：
1. 使用视频封面图
2. 在内容中添加视频链接
3. 后续版本会支持视频嵌入

### Q5：内容能实时预览吗？
**A：**
1. 修改后需要保存
2. 返回前台页面查看
3. 建议在新标签页打开前台，方便对比

## 最佳实践

### 1. 内容组织
- **模块化设计**：每个模块专注一个主题
- **层次分明**：使用标题区分不同部分
- **图文结合**：适当配图增加视觉效果

### 2. 图片优化
- **压缩图片**：使用TinyPNG等工具压缩
- **统一风格**：保持图片色调、风格一致
- **合适尺寸**：不要上传过大的图片

### 3. 文案编写
- **简洁明了**：突出重点信息
- **中英对照**：保持翻译准确
- **定期更新**：及时更新过时内容

### 4. 数据管理
- **定期备份**：每次重大修改后备份
- **版本记录**：记录修改时间和内容
- **测试验证**：修改后检查前台显示
