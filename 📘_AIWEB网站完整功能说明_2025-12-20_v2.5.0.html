网站概述

网站定位

AIWEB是一个集成了智能制造、AI营销、设备全生命周期管理等多功能的企业级Web平台。

核心技术栈

  • 前端: Vue 3 + Element Plus + ECharts
  • 后端: Node.js + Express + SQLite
  • 部署: 支持云端/内网双模式

访问方式

  • 开发环境: http://localhost:3002
  • 生产环境: http://localhost:3000

📋 使用说明

如何使用本文档截图

  1. 启动开发服务器:

    # 前端
npm run dev

# 后端
cd server
node index.js

  2. 访问对应页面: 根据上表”访问路径”打开浏览器

  3. 截图工具推荐:

    • Windows: Snipping Tool / Snipaste
    • Mac: Command+Shift+4
    • 浏览器扩展: Awesome Screenshot

  4. 截图规范:

    • 分辨率: 1920x1080
    • 格式: PNG
    • 命名: 功能模块_页面名称_日期.png
    • 标注: 使用红框标注关键功能

  5. 生成PDF:

    • 使用Word/Google Docs插入截图
    • 按本文档章节结构排版
    • 导出为PDF格式
