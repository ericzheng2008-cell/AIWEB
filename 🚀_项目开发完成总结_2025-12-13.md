## 📊 项目概览

本项目是一个集**营销获客**、**产品展示**、**设备管理**、**工单系统**、**AI智能体**于一体的企业级管理平台。

### 核心特性
- 🌐 中英文双语支持
- 🤖 AI客服智能理解
- 📱 响应式设计
- 🔐 完整的权限管理
- 💾 数据持久化存储
- 📊 可视化数据分析

## 🔧 核心功能亮点

### 1. 设备全生命周期管理 ⭐⭐⭐
**最大亮点**: 完整的设备管理闭环

#### 功能特性
- **设备档案管理**: 6大类设备分类管理
- **ROI投资回报率计算**: 8个计算因子，智能分析投资回报
- **AI保养预测**: 基于运行时长、故障历史智能预测保养时间
- **服务成本动态采集**: 备件成本、人工成本、停机损失自动统计
- **数据可视化**: ECharts图表展示趋势

#### 数据流
```
设备创建 → ROI计算 → 运行监控 → 故障预警 → 创建工单 → 维修处理 → 
成本记录 → 设备档案更新 → AI分析 → 保养建议 → 成本优化
```

**核心价值**: 故障全流程追踪

#### 功能特性
- **智能工单创建**: 自动更新设备状态
- **多维度追踪**: 故障类型、优先级、处理状态
- **处理记录时间线**: 完整的处理过程记录
- **成本自动计算**: 备件+人工+停机损失
- **数据联动**: 工单关闭自动更新设备健康度

#### 优先级管理
- 🔴 紧急 (Urgent)
- 🟠 高 (High)
- 🟡 中 (Medium)
- 🟢 低 (Low)

#### 状态流转
```
待处理 → 已分配 → 处理中 → 已解决 → 已关闭
```
