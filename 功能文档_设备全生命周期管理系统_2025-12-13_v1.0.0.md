## 📋 系统概述

设备全生命周期管理系统是一套完整的关键设备资产管理解决方案，涵盖设备档案、ROI计算、维护保养预测、服务成本分析等核心功能，帮助企业实现设备资产的精细化管理和降本增效。

### 管理设备类别

基于维修降本清单，系统支持以下**6大类关键设备**：

| 类别 | 包含设备/部件 | 部件数量 |
|------|---------------|----------|
| **机器人** | 电路板、减速机、伺服放大器、示教器、伺服电机 | 5个 |
| **焊钳** | 电极握杆、钳臂、伺服马达/电缸、焊接控制器 | 4个 |
| **电气类** | 激光扫描仪、触摸屏、变频器、伺服放大器、电机 | 5个 |
| **机械类** | 气缸、强力缸、电磁阀、包胶轮、涂胶枪、涂胶定量机、钛制旋杯 | 7个 |
| **喷涂** | 控制器、控制基板、油压脉冲工具换油 | 3个 |
| **工具** | 离合器工具维修保养、油压脉冲工具维修保养 | 2个 |

**总计**: 6大类, 26个关键部件

### 2. ROI投资回报率分析

#### 计算模型
系统采用标准的ROI计算模型，综合考虑：

**投资成本**:
- 设备购买价格
- 累计维护成本
- 备件更换成本

**产出收益**:
- 年产值 = 实际运行小时 × 每小时产值
- 实际运行小时 = 年度总小时 × 使用率 - 计划停机 - 非计划停机

**ROI计算公式**:
```
ROI = (总回报 - 总投资) / 总投资 × 100%
回本期 = 总投资 / 年产值
```

#### 数据展示
- **ROI百分比**: 绿色(≥20%)、黄色(10-20%)、红色(<10%)
- **回本期**: 多少年可收回投资
- **当前价值**: 考虑折旧后的设备价值
- **年折旧**: 直线折旧法计算
- **总维护成本**: 累计维护支出
- **年产值**: 预计年度产出价值

#### 决策支持
- 识别高ROI设备，优化投资策略
- 发现低ROI设备，考虑更换或优化
- 评估设备性价比，指导采购决策

### 3. AI保养预测

#### 预测算法
基于多因素加权模型进行智能预测：

**因素权重**:
- 运行时间：40%
- 故障历史：30%
- 部件年龄：20%
- 环境因素：10%

**风险评分**:
```
风险评分 = 使用率因子 × 0.6 + 部件状况因子 × 0.4
```

#### 预警等级
- 🔴 **紧急** (≤30天): 需立即安排保养
- 🟡 **警告** (31-60天): 需尽快安排保养
- 🟢 **正常** (>60天): 按计划保养

#### 预测内容
- 下次保养日期
- 剩余天数
- 风险评分(0-100)
- 预估成本
- 推荐操作

#### 智能功能
- 自动识别高风险部件
- 提前预警保养到期
- 生成保养计划
- 一键创建保养工单

### 4. 服务成本动态采集

#### 成本类型
系统自动追踪4类服务成本：

1. **备件成本** (Parts)
   - 更换部件的采购成本
   - 与工单关联记录
   - 支持多个备件累加

2. **人工成本** (Labor)
   - 维修人员工时成本
   - 默认80元/小时
   - 基于实际维修时长计算

3. **保养成本** (Maintenance)
   - 定期保养费用
   - 润滑油、清洁用品等
   - 按保养计划记录

4. **维修成本** (Repair)
   - 非计划维修支出
   - 应急处理费用
   - 外包服务费用

#### 数据采集
- **自动采集**: 工单关闭时自动记录成本
- **手动录入**: 支持补充历史成本数据
- **关联工单**: 每笔成本关联到具体工单
- **实时统计**: 动态计算累计成本

#### 成本分析
- **月度成本**: 按月统计服务成本
- **成本占比**: 备件/人工/保养/维修占比
- **成本趋势**: 12个月成本变化趋势
- **月均成本**: 平均每月服务支出

#### 降本建议
系统基于成本数据提供优化建议：
- 识别高成本设备
- 分析成本结构
- 提供降本方案
- 预测未来支出

## 🔗 与工单管理系统集成

### 双向数据同步

#### 工单创建时
```
工单系统 → 设备系统
- 设备状态变更为 'fault'
- 记录故障时间
- 暂停运行小时计数
```

#### 工单关闭时
```
工单系统 → 设备系统
1. 设备状态恢复为 'running'
2. 记录维护记录到设备档案
   - 维护类型（计划/非计划）
   - 维护人员
   - 维护时长
   - 维护成本
3. 记录服务成本
   - 备件成本（逐项记录）
   - 人工成本（基于工时）
4. 更新部件状态
   - 提升部件状况评分
   - 更新下次保养日期
```

### 保养工单创建
从设备生命周期系统可一键创建保养工单：

**自动填充字段**:
- 设备ID和设备名称
- 部件ID和部件名称
- 故障类型：设为"保养"
- 优先级：根据紧急程度设定
- 描述：自动生成保养说明

**流程示例**:
```
1. AI预测：部件需保养，显示预警
2. 点击"创建工单"按钮
3. 系统自动跳转到工单系统
4. 工单表单已预填设备信息
5. 补充详细信息后提交
6. 工单进入正常流转流程
```

## 💡 应用场景

### 场景1：新设备采购决策
**需求**: 采购一台新的焊接机器人，预算28万元

**使用流程**:
1. 查看同类设备的ROI数据
2. 对比不同品牌/型号的投资回报
3. 分析历史维护成本
4. 预测未来5年的总拥有成本
5. 做出最优采购决策

**预期收益**: 选择ROI最高的设备，避免低效投资

---

### 场景2：保养计划优化
**需求**: 降低非计划停机，优化保养计划

**使用流程**:
1. AI系统分析所有设备的保养预测
2. 识别高风险部件（紧急、警告）
3. 按优先级生成保养计划
4. 自动创建保养工单
5. 技术人员按计划执行保养

**预期收益**: 
- 减少非计划停机50%
- 延长设备寿命20%
- 降低维修成本30%

---

### 场景3：成本降本分析
**需求**: 某条生产线设备维护成本过高，需要分析原因

**使用流程**:
1. 查看该生产线所有设备的成本分析
2. 识别高成本设备TOP3
3. 分析成本结构（备件vs人工vs保养vs维修）
4. 查看成本趋势，发现异常增长
5. 根据数据提出优化建议：
   - 备件成本高→考虑国产替代
   - 人工成本高→优化维修流程
   - 维修成本高→加强预防性保养

**预期收益**: 年度维护成本降低15-25%

---

### 场景4：设备健康监控
**需求**: 实时掌握关键设备的健康状况
